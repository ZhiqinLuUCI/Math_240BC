Section 5.1 Eigenvalue

We first make different notations of the Laplace operator.

We assume that \(M\) is a Riemannian manifold with the Riemannian metric

\begin{equation*} ds^2 = \sum g_{ij} d x _i d x _j\text{.} \end{equation*}

Let \((g^{ij}\)) be the inverse matrix of \((g_{ij}\)) and let \(g = { det} (g_{ij})\text{.}\) Then under the local coordinates \((x _1, \ldots, x _n )\text{,}\) we define

\begin{equation*} \Delta = \frac{1}{\sqrt{g}} \frac{\partial }{\partial x_i} \left( g ^{ij} \sqrt{g} \frac{\partial }{\partial x_j} \right)\text{.} \end{equation*}

Apparently, we can write

\begin{equation*} \Delta = g^{ij}\frac{\partial^2 }{\partial x_i \partial x _j} +\frac{1}{\sqrt{g} } \frac{\partial }{\partial x_i} \left( g ^{ij} \sqrt{g} \right)\frac{\partial }{\partial x_j}\text{.} \end{equation*}

As before, we have the following computation

\begin{align*} \lefteqn{\frac{1}{\sqrt{g} } \frac{\partial }{\partial x_i} ( g^{ij} \sqrt{g} )}\\ \amp \amp = \frac{1}{2} g^{ij} \frac{\partial}{\partial x_i} \log g + \frac{\partial }{\partial x_i} g ^{ij}\\ \amp \amp = \frac{1}{2} g^{ij}g^{kl}\frac{\partial g _{kl} }{\partial x_i} - g^{in} g^{jm} \frac{\partial g _{mn} }{\partial x_i}\\ \amp \amp = - g^{kl} \Gamma ^j _{kl} \text{.} \end{align*}

Thus we also have the following formula

\begin{equation*} \Delta = g^{ij} \frac{\partial^2 }{\partial x_i \partial x _j} - g ^{kl} \Gamma ^j_{kl} \frac{\partial }{\partial x_j}\text{.} \end{equation*}

From the above representation, we have the following third formula for the Laplace operator.

Let \(ds^2 = \sum^n _{j=1} \omega ^2_j\text{.}\) With respect to the frame, the connection \((\omega _{ij} )\) is well-defined. We are above the define the covariant derivatives of a function like the following

\begin{align*} df \amp = \amp f _i \omega _i\\ f_{ij} \omega _j \amp = \amp d f _i - f _s \omega _{si}\text{.} \end{align*}

Using the above notations, we can define

\begin{equation*} \Delta f = \sum^n_{i=1} f _{ii} = \sum _i \nabla _i \nabla _i f\text{.} \end{equation*}

Finally, we can define the Laplace operator on \(p\)-forms as follows:

Let \(d : \wedge ^p (M) \rightarrow \wedge^{p+1} (M)\) be the ordinary differential operator, where \(\wedge ^p(M)\) be the space of smooth \(p\)-forms. With respect to the Riemannian metric, \(\wedge^p(M)\) becomes an infinite dimensional inner product space.

Let \(\delta\) be the formal dual operator with respect to \(d\text{.}\) Then

\begin{equation*} \delta: \wedge ^p(M) \rightarrow \wedge ^{p-1} (M)\text{.} \end{equation*}

We can prove that \(\delta\) is also a differential operator of first order. The Laplace operator can be defined as

\begin{equation*} \Delta = d \delta + \delta d\text{.} \end{equation*}

In particular, on the space of functions, or \(0\)-forms,

\begin{equation*} \Delta = \delta d\text{.} \end{equation*}

We have the following:

Theorem 5.1.1. Weitzenböck formula.

For function \(f\text{,}\) we have

\begin{equation*} \Delta f = \delta d f = - \sum \nabla _i\nabla_i f\text{.} \end{equation*}

In general, we have the following

\begin{equation*} \Delta = - \sum \nabla _i \nabla _i + \ \mbox{curvature terms} \end{equation*}

which is also called Weizenböck formula.

We go back to the Laplacian on functions. We know that \(\Delta = \sum \nabla _i \nabla _ i\) is well-defined on \(C^\infty (M)\text{.}\)

Unfortunately, with the following \(L^2\)-inner product

\begin{equation*} \langle f \cdot g \rangle = \int _M fg d V \end{equation*}

\(C^\infty\) (M) is not a complete metric space. The complete metric space is \(L^2(M)\text{.}\) However, there is no way that we can extend \(\Delta\) on \(L^2(M)\text{.}\)

Proof.

The key point is that any differential operator is a closed-graph operator. Thus if \(\Delta\) is extendable, then by ***, \(\Delta\) has to be a bounded operator, as by the example of Heaviside function.

Thus we can only extend the operator into a densely defined self-adjoint operator.

Recall that an operator \(\Delta\) is self-adjoint if

\begin{equation*} { Dom} (\Delta) = { Dom} ( \Delta ^\ast) \end{equation*}

and

\begin{equation*} \langle \Delta f , g \rangle = \langle f, \Delta g \rangle \end{equation*}

for any \(f, g \in { Dom} (\Delta)\text{.}\)

In functional analysis, we have the following theorem. Let

\begin{equation*} Q (\phi, \psi)= \int \nabla \phi \nabla \psi\text{.} \end{equation*}

Then \(Q\) is a non-negative quadratic form defined on \(H^1_0 (M)\text{.}\) Then there is a unique densely defined operator \(A\) such that

\begin{equation*} Q (\phi, \psi) = - ( A \phi , \psi)\text{.} \end{equation*}

Such an operator \(A\) is in fact called the Dirichlet Laplacian operator.

As an exercise, we prove that on an only manifold \(L^2\) harmonic function must be constant. \end{proof}

Theorem 5.1.3.

Let \(A\) be the Dirichlet extension of the Laplacian \(\Delta\text{.}\) A function \(f\) is called \(A\)-harmonic, if \(f \in { Dom} (A)\) and \(A f = 0\text{.}\) If \(f \in L^2 (M)\text{,}\) then \(f\) is a constant.

Proof.

The key point is that

\begin{equation*} Q (\rho ^2 f, f) = Q \langle \rho^2 f , \Delta f \rangle = 0\text{.} \end{equation*}

Thus using the same method as before, \(f\) is a constant. If \(M\) is a compact manifold with no boundary, we still use \(\Delta\) to denote the Dirichlet extension of the Laplace operator. By the elliptic regularity, the spectrum of \(\Delta\) are discrete. That is, there is a sequence

\begin{equation*} 0 = \lambda _0 \lt \lambda _1 \leq \lambda _2 \leq \ldots \end{equation*}

such that for any \(\lambda_i\text{,}\) there \(\exists f_i , f _i \in L^2\)

\begin{equation*} \Delta f _i = - \lambda _i f_i\text{.} \end{equation*}

We have similar results for manifolds with boundary conditions. To be more precise, the Laplacians acting on functions with the following boundary conditions.

  1. (A).

    Dirichlet boundary condition: \(\Delta\) acting on \(f\) vanishing on the boundary.

  2. (B).

    Neumann boundary condition: \(\Delta\) acting on functions such that \(\frac{\partial f}{\partial n} = 0\text{.}\)

For eigenvalues, we have the following minimax principle. Assume that \(M\) is a closed manifold, then

\begin{equation*} \lambda _1 = \inf_{\int f = 0 } \frac{\int |\nabla f|^2}{\int f^2}\text{.} \end{equation*}

To prove the above result, we let \(\phi\) be any smooth function such that

\begin{equation*} \int_M \phi = 0\text{.} \end{equation*}

Then by the definition of \(\lambda _1\text{,}\) we have

\begin{equation*} \frac{\int | \nabla (f + \varepsilon \phi )|^2}{\int (f + \varepsilon \phi )^2} \geq \lambda _1\text{.} \end{equation*}

However, if we take the first order term, we get

\begin{equation*} \int (\delta f + \lambda _1 f ) \phi = 0\text{.} \end{equation*}

Note that

\begin{equation*} \int (\Delta f + \lambda _ 1 f ) = 0\text{.} \end{equation*}

Then

\begin{equation*} \Delta f + \lambda _1 f \equiv 0 \end{equation*}

and \(\lambda _ 1\) is the first eigenvalue.

By elliptic regularity, \(\lambda _ 1 > 0\text{.}\) Thus we have the following Poincaré-inequality: there exists a constant \(C\text{,}\) such that

\begin{equation*} \int | \nabla f |^2 \geq C \int f^2 \end{equation*}

for any function with

\begin{equation*} \int _M f = 0\text{.} \end{equation*}

Before going further, let's prove the following co-area formula.

Theorem 5.1.4.

Let \(M\) be a compact Riemannian manifold with boundary. Let \(f \in H^\prime (M)\text{.}\) Then

\begin{equation*} \int _M g = \int ^{+ \infty}_{-\infty} \int _{ \{ f = 0 \}} \frac{g}{| \nabla f |} d \sigma\text{.} \end{equation*}

\begin{proof} Without loss of generality, we assume that \(| \nabla f | \neq 0\text{.}\) Thus by the implicit function theorem \(\{ f = \sigma \}\) is a smooth manifold.

Using the cut-off function, we may assume that \({ supp}\, g\) is contained in a coordinate chart. Thus we may assume that the Riemannian metric is given under the global coordinates \((x_1, \ldots, x _n)\) as follows

\begin{equation*} d s^2 = \sum g _{ig} d x _i ds _j \end{equation*}

by definition

\begin{equation*} \int _M g = \int_M g \sqrt{{ det} (g _{ij} ) } \ dx_1, \ldots, dx _n\text{.} \end{equation*}

Since \(\nabla f \neq 0\text{,}\) we can solve the equation

\begin{equation*} f = \sigma \end{equation*}

by

\begin{equation*} x_1 = x_1 (\sigma , x_2, \ldots, x_n) \end{equation*}

or in other words, by the implicit function theory \((\sigma, x_2 , \ldots, x_n )\) is a local coordinate system as well. The Jacobian of the transformation is

\begin{equation*} d x_1 \wedge \ldots \wedge d x _n = \frac{\partial x _1}{\partial \sigma} d \sigma \wedge d x_2 \wedge \ldots \wedge d x_n\text{.} \end{equation*}

On the other side, restricting to \(f = \sigma\text{,}\) the Riemann metric can be written as

\begin{equation*} \left(g_{11} \frac{\partial x _1}{\partial x_k} \cdot \frac{\partial x _1}{\partial x_l} + g_{1l} \frac{\partial x _1}{\partial x_l } + g _{k1} \frac{\partial x _1}{\partial x_k} + g _{kl}\right) \, d x _k d x _l\text{.} \end{equation*}

If we choose local coordinates such that \(g_{kl} = \delta _{kl}\text{.}\) Then we have

\begin{equation*} \left( \frac{\partial x _1}{\partial x_k} \cdot \frac{\partial x _1}{\partial x_l} + \delta _{kl} \right) \, d x_k dx_l\text{.} \end{equation*}

The volume form of the above is

\begin{align*} \lefteqn{{ det} \left( \delta _{kl} + \frac{\partial x _1}{\partial x_k} \cdot \frac{\partial x _1}{\partial x_l} \right) = 1 + \sum \left|\frac{\partial x _1}{\partial x_k}\right| ^2}\\ \amp \amp = \frac{1}{| \frac{\partial f}{\partial x_1}|^2} \sum\left|\frac{\partial f}{\partial x_k}\right|^2 = \frac{|\nabla f |^2}{ | \frac{\partial f}{\partial x_1} | ^2} \text{.} \end{align*}

Thus we have

\begin{align*} \int_M g \amp = \amp \int_M g \sqrt{{ det} (g_{ij})} d x _1 \ldots d x _n\\ \amp = \amp \int_M \sqrt{{ det} (g_{ij})} \frac{\partial x_1}{\partial \sigma} d \sigma \wedge d x_2 \ldots \wedge d x _n \text{.} \end{align*}

By (\(\triangle\)), we must have

\begin{equation*} d V ds^2 = d V _{f = \sigma} \cdot \frac{ \left|\frac{{\partial f }}{{\partial x_i}}\right|}{| \nabla f |}\text{.} \end{equation*}

Thus

\begin{equation*} \int_M g = \int ^{+ \infty}_{-\infty} \left( \int _{f=\sigma} \frac{g}{|\nabla f|}\right) d \sigma\text{.} \end{equation*}

Of course, in general, there are points such that \(\nabla f = 0\text{.}\) But by a theorem of Sand, the set

\begin{equation*} \left\{ yf(y) | \nabla f (y) = 0 \right\} \end{equation*}

is of zero measure. Using the standard covering technique, we can prove the same result.

As an application of the above co-area formula, we prove the following result of Sobolev inequality.

Sobolev inequality. Let \(M\) be a compact manifold with boundary. Then there is a constant \(C> 0\) such that

\begin{equation*} C \left( \int_M |f| ^{\frac{n}{n-1}} \right)^{\frac{n-1}{n}}\leq \int_M |\nabla f | \end{equation*}

for any smooth function \(f /2M = 0\) (\(D\)-condition) or \(\int f = 0\) (Neumann condition).

Isopermetric inequality. Let \(\Omega\) be a domain in \(M, \Omega \subset \subset M\text{.}\) Then there is a constant independent to \(\Omega\) such that

\begin{equation*} C { vol} (\Omega ) ^{\frac{n-1}{n}} \leq { vol} (\partial \Omega)\text{.} \end{equation*}

We want to prove that the Isopermetric inequality is equivalent to the Sobolev inequality.

At least one-side of the implication was clear: assuming the Sobolev inequality, if we let

\begin{equation*} f _\varepsilon (x) = \left\{ \begin{array}{cl} 1 \amp x \in \Omega , \ d (x , \partial \Omega ) \geq \varepsilon \\ \frac{d(x, \partial \Omega }{\varepsilon} \amp x \in \Omega , \ d (x , \partial \Omega ) \leq \varepsilon\\ 0 \amp { otherwise} \end{array} \right\}\text{.} \end{equation*}

Then using the Sobolev inequality, the isopermetric inequality follows by letting \(\varepsilon \rightarrow 0\text{.}\)

In order to prove that the isopermetric inequality implies the Sobolev inequality, we use the co-area formula. We assume that \(f \geq 0\text{.}\) Then

\begin{equation*} \int_M | \nabla f| = \int ^\infty_0 \ \mbox{Area} \ (f = \sigma ) \, d \sigma\text{.} \end{equation*}

We also have

\begin{align*} \int_M |f| ^{\frac{n}{n-1}} \amp =\amp \int ^\infty_0 { vol} ( f ^{\frac{n}{n-1}}> \lambda ) \, d \lambda\\ \amp = \amp \frac{n}{n-1}\int^\infty_0 { vol} (f > \sigma ) \sigma ^{\frac{n}{n-1}} d \sigma \text{.} \end{align*}

Using the isopermetric inequality, we have

\begin{equation*} \int_M |\nabla f| = \int ^\infty _0 { Area} (f = \sigma ) \, d \sigma \geq C \int^\infty _0 { vol} (f > \sigma ) ^{\frac{n}{n-1}} d \sigma\text{.} \end{equation*}

Thus in order to prove the Sobolev inequality, we just need to prove that

\begin{equation*} \int ^\infty _0 { vol} (f > \sigma ) ^{\frac{n}{n-1}} d \sigma \geq C \left( \int ^\infty _0 { vol} (f > \sigma ) \sigma ^{\frac{n}{n-1}} d \sigma \right) ^{\frac{n}{n-1}}\text{.} \end{equation*}

We let

\begin{align*} F(\sigma) \amp = \amp { vol} (f > \sigma)\\ \phi (t) \amp = \amp \int^t_0 F(\sigma) ^{\frac{n}{n-1}} d \sigma\\ \psi (t) \amp = \amp \left( \int ^t _0 F(sigma ) \sigma ^{\frac{n}{n-1}} d \sigma \right) ^{\frac{n-1}{n}}\text{.} \end{align*}

Then \(\phi (0) = \psi (0)\text{.}\) Using the monotonicity of \(F(\sigma)\) we can prove that

\begin{equation*} \phi ^\prime (t) \geq \frac{n}{n-1} \psi^\prime (t)\text{.} \end{equation*}

Thus

\begin{equation*} \phi (\infty) \geq \frac{n}{n-1} \psi (\infty)\text{.} \end{equation*}

Corollary 5.1.5.

\begin{equation*} \left( \int f ^{\frac{n-p}{np}}\right) ^{\frac{np}{n-p}} \leq C\left( \int |\nabla f |^p \right) ^{\frac{1}{p}} \end{equation*}

for any \(p > 1\text{.}\)

Definition 5.1.6.

Let \(M\) be a compact Riemannian manifold if \(\partial M \neq \emptyset\text{.}\)

\begin{equation*} h_D (M) = \inf \left\{ \left.\frac{{ vol} ( \partial \Omega )}{{ vol} ( \Omega)} \right| \Omega \subset \subset M\right\} \end{equation*}

if \(\partial M = \emptyset\text{.}\)

\begin{equation*} h_N = \inf \left\{ \left.\frac{{ vol}( H )}{\min ({ vol} (M_1), { vol} (M_2) )} \right| H \ \mbox{is a hypersurface} \right\}\text{.} \end{equation*}

Theorem 5.1.7. Cheeger.

For Dirichlet condition, we have

\begin{equation*} \lambda _ 1 \geq \frac{1}{4} h^2_D\text{.} \end{equation*}

For Neumann condition

\begin{equation*} \lambda _1 \geq \frac{1}{4} h^2_N (M)\text{.} \end{equation*}

Proof.

We only prove the case for Dirichlet condition. We first observe that, if there is a constant \(\mu\) such that

\begin{equation*} \int_M |\nabla \phi | \geq \mu \int_M |\phi | \end{equation*}

for any \(\phi\) with \(\phi |_{\partial M} = 0\text{.}\) Then \(\lambda _1 \geq \frac{1}{4} \mu^2\text{.}\) To see this, we consider \(\phi = f^2\)

\begin{equation*} \mu \int _M f^2 \leq 2 \int_M |f| | \nabla f|\leq 2 \left( \int_M f^2 \right) ^{\frac{1}{2}} \left( \int |\nabla f|^2 \right) ^{\frac{1}{2}}\text{.} \end{equation*}

Thus

\begin{equation*} \frac{1}{4} \mu^2 \int_M f ^2 \leq \int_M | \nabla f|^2\text{.} \end{equation*}

Since the above is true for any function \(f\text{,}\) we must have

\begin{equation*} \lambda _1 \geq \frac{1}{4} \mu ^2\text{.} \end{equation*}

Finally, we prove a result which is well known but can't readily be found in the literature.

Theorem 5.1.8.

Let \(M\) be a compact manifold with smooth boundary. Let

\begin{equation*} M_1 = \inf_{\int f = 0} \frac{\int | \nabla f|^2}{\int f ^2}\text{.} \end{equation*}

Then we have the following result: let \(f\) be a minimizer, and let \(f\) be smooth. Then

\begin{equation*} \frac{\partial f }{\partial n} = 0\text{.} \end{equation*}

Proof.

Let \(\phi\) be a smooth function with compact support such that

\begin{equation*} \int \phi = 0\text{.} \end{equation*}

Then we have

\begin{equation*} \int |\nabla (f + \varepsilon \phi )|^2 \geq \mu_1 \int (f + \varepsilon \phi )^2\text{.} \end{equation*}

Thus we have

\begin{equation*} \int \nabla f \nabla \phi = - \mu _1 \int_M f \phi\text{.} \end{equation*}

by Green's formula

\begin{equation*} \int_M \nabla f \nabla \phi - \int _{\partial M} \phi \frac{\partial f }{\partial n} - \int_M \Delta f \phi\text{.} \end{equation*}

Thus since \(\Delta f = - \mu_1 f\text{,}\) we have

\begin{equation*} \int_{\partial M} \phi\frac{\partial f }{\partial n} = 0 \end{equation*}

and we must have \(\frac{\partial f }{\partial n} = 0\text{.}\)
