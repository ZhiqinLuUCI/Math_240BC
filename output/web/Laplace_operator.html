Section 2.1 The Laplace operator

Let \(M\) be a compact orientable Riemannian manifold and let \(\Lambda^p(M)\) be the vector space of \(p\) forms. As in the last section, we use \(\omega_1,\cdots,\omega_n\) to be the local orthonormal frame, and \(\omega_1\wedge\cdots\wedge\omega_n\) to be the volume form.

We write a \(p\)-form in the following expression

\begin{equation*} \eta=a_{i_1\cdots i_p}\,\omega_{i_1}\wedge\cdots\wedge\omega_{i_p}\text{,} \end{equation*}

where we assume the coefficients are skew-symmetric. That is, we assume that

\begin{equation*} a_{\sigma(i_1)\cdots\sigma(i_p)}=(-1)^{sgn\,\sigma}a_{i_1\cdots i_p} \end{equation*}

for any permutation \(\sigma\) of \(\{1,\cdots,p\}\text{.}\) The \(L^2\) inner product is defined as

\begin{equation} (\omega,\eta)=p!\int_M a_{i_1\cdots i_p}b_{i_1\cdots i_p}\omega_1\wedge\cdots\wedge\omega_n\text{.}\tag{2.1.1} \end{equation}

We remark that in the above definition, we only require one of the coefficients to be skew-symmetric, which is useful in the following computation.

Checkpoint 2.1.1.

Prove the following: if

\begin{equation*} \eta=b_{i_1\cdots i_p}\omega_{i_1}\wedge\cdots\wedge\omega_{i_p}\text{,} \end{equation*}

where \(\eta=b_{i_1\cdots i_p}\) are not assumed to be skew-symmetric. Then we still have

\begin{equation*} (\omega,\eta)=p!\int_M a_{i_1\cdots i_p}b_{i_1\cdots i_p}\omega_1\wedge\cdots\wedge\omega_n\text{.} \end{equation*}

Lemma 2.1.2.

We have

\begin{equation*} d\omega=(-1)^pa_{i_1\cdots i_p,k}\,\omega_{i_1}\wedge\cdots\wedge\omega_{i_p}\wedge\omega_k\text{.} \end{equation*}

Since the coefficients above are not skew symmetric, we may also rewrite the expression as

\begin{equation*} d\omega=\frac{(-1)^p}{p+1}\left( a_{i_1\cdots i_p,k}-\sum_{s=1}^p a_{i_1\cdots \underset{\underset{sth}{\uparrow}}k\cdots i_p, i_s}\right) \omega_{i_1}\wedge\cdots\wedge\omega_{i_p}\wedge\omega_k\text{.} \end{equation*}

Proof.

The proof is through a straightforward computation:

\begin{equation*} d\omega=da_{i_1\cdots i_p}\wedge\omega_{i_1}\wedge\cdots\wedge\omega_{i_p}+\sum_{s=1}^p(-1)^{s-1} a_{i_1\cdots i_p}\wedge\omega_{i_1}\wedge\cdots\wedge d\omega_{i_s}\wedge\cdots\wedge\omega_{i_p}\text{.} \end{equation*}

Using Cartan's formula \(d\omega_{i_s}=-\omega_i\wedge\omega_{ii_s}\text{,}\) we get the first formula. The second formula follows from the skew-symmetrization.

Lemma 2.1.3.

Let \(\delta\) be the dual operator of \(d\) with respect to the inner product¬†(2.1.1). We then have

\begin{equation*} \delta\omega=(-1)^pp\,a_{i_1\cdots i_p,i_p}\omega_{i_1}\wedge\cdots \wedge\omega_{i_{p-1}}\text{.} \end{equation*}

Proof.

Let

\begin{equation*} \eta=b_{i_1\cdots i_{p-1}}\omega_{i_1}\wedge\cdots\wedge\omega_{i_{p-1}} \end{equation*}

be a smooth \((p-1)\)-form with compact support. We verify that

\begin{equation*} (\delta\omega,\eta)-(\omega,d\eta)=0\text{.} \end{equation*}

We have

\begin{equation*} (\delta\omega,\eta)=(-1)^{p}p!\int_M a_{i_1\cdots i_{p-1}i_p,i_p}b_{i_1\cdots i_{p-1}}\text{,} \end{equation*}

and

\begin{equation*} (\omega,d\eta)=(-1)^{p-1}p!\int_M a_{i_1\cdots i_p}b_{i_1\cdots i_{p-1},i_p}\text{.} \end{equation*}

define an \((n-1)\) form \(\alpha\) such that

\begin{equation*} \alpha=\sum_{k=1}^n (-1)^{k-1}a_{i_1\cdots i_{p-1} k}b_{i_1\cdots i_{p-1}}\,\omega_1\wedge\cdots\hat{\omega}_k\cdots\wedge\omega_n\text{.} \end{equation*}

Then we have

\begin{equation*} d\alpha=(a_{i_1\cdots i_{p-1}i_p,i_p}b_{i_1\cdots i_{p-1}} +a_{i_1\cdots i_p}b_{i_1\cdots i_{p-1},i_p})\,\omega_1\wedge\cdots\wedge\omega_n\text{.} \end{equation*}

The lemma follows from the Stokes' theorem

\begin{equation*} \int_M d\alpha=0\text{.} \end{equation*}

Theorem 2.1.4. Weitzenb√∂ck formula.

We have

\begin{equation*} \Delta\,\omega=\left(-a_{i_1\cdots i_p,k,k}+p\sum_{s=1}^{p-1}a_{i_1\cdots \underset{\underset{sth}{\uparrow}}r\cdots i_{p-1} k}R_{ri_si_pk}+p \, a_{i_1\cdots i_{p-1} r}(Ric)_{r i_p}\right)\omega_{i_1}\wedge\cdots\wedge\omega_{i_p}\text{.} \end{equation*}

Proof.

By the above two lemmas we have

\begin{equation*} \delta d\omega=\left(-a_{i_1\cdots i_p,k,k}+\sum_{s=1}^p a_{i_1\cdots \underset{\underset{sth}{\uparrow}}k\cdots i_p, i_s,k}\right) \omega_{i_1}\wedge\cdots\wedge\omega_{i_p}\text{,} \end{equation*}

and

\begin{equation*} d\delta\omega=-p\, a_{i_1\cdots i_p,i_p,k}\omega_{i_1}\,\wedge\cdots\wedge\omega_{i_{p-1}}\wedge\omega_k\text{.} \end{equation*}

By changing the indices and using the skew-symmetry of \(a_{i_1\cdots i_p}\) in the above, we get

\begin{equation*} \delta d\omega=\left(-a_{i_1\cdots i_p,k,k}+p a_{i_1\cdots i_{p-1}k, i_p,k}\right) \omega_{i_1}\wedge\cdots\wedge\omega_{i_p}\text{,} \end{equation*}

and

\begin{equation*} d\delta\omega=-p\, a_{i_1\cdots k,k,i_p}\,\omega_{i_1}\,\wedge\cdots\wedge\omega_{i_{p-1}}\wedge\omega_{i_p}\text{.} \end{equation*}

The theorem follows by applying Theorem¬†1.3.3.

Define the raw Laplacian \(\nabla^*\nabla\) on a \(p\)-form to be

\begin{equation*} \nabla^*\nabla\omega=-\nabla_k\nabla_k\omega\text{.} \end{equation*}

Then the Weitzenb√∂ck formula can be written as

\begin{equation*} \Delta\omega=\nabla^*\nabla\omega+E(\omega)\text{,} \end{equation*}

where \(E\) is a \(0\)-th order differential operator depending on the curvature.

We would like to list the following special cases of the above theorem as exercises.

Checkpoint 2.1.5.

Using the above notations, we have

\begin{equation*} \delta=(-1)^{n(p+1)+1}*d* \end{equation*}

on \(\Lambda^p(M)\text{.}\)

Checkpoint 2.1.6.

Let \(\{e_1,\cdots, e_n\}\) be an orthonormal frame. Then

\begin{equation*} \delta=-\sum_{j=1}^n\iota(e_j)\nabla_{e_j}\text{.} \end{equation*}

Checkpoint 2.1.7.

If \(f\) is a smooth function, then

\begin{equation*} \Delta f=-f_{,k,k}=-f_{kk}\text{.} \end{equation*}

Checkpoint 2.1.8.

If \(\omega\) is a one-form, then

\begin{equation*} \Delta\omega=\nabla^*\nabla\omega+a_r(Ric)_{rk}\omega_k\text{.} \end{equation*}
