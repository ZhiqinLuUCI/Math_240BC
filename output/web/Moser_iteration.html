Section 9.1 De Giorgi-Nash-Moser Estimates

In classical Schauder estimates or \(W^{2,p}\) estimates, the continuity of the coefficients is required. Therefore, the methods are not usable in nonlinear case. In 1957, De Giorgi obtained the Hölder estimate on elliptic equations of divergence form with measurable coefficients. In 1958, Nash independently got the similar estimates on parabolic equations. In 1960, Moser obtained a simplified proof and the Harnark inequality. These methods now becomes crucial in geometric analysis.

We consider the following elliptic equation of divergence form:

\begin{equation} -D_j(a^{ij}D_iu)+b^iD_i u+cu=0\tag{9.1.1} \end{equation}

such that

\begin{align*} \amp \lambda|\xi|^2\leq a^{ij}\xi_i\xi_j\leq\Lambda|\xi|^2,\qquad \forall x\in\Omega,\,\xi\in\R^n\\ \amp \sum_{i,j}||a^{ij}||_{L^\infty}+\sum_i||b^i||_{L^\infty}+||c||_{L^\infty}\leq\Lambda\text{.} \end{align*}

For the sake of simplicity, we only consider the following equation

\begin{equation} -D_j(a^{ij}D_iu)=0\text{.}\tag{9.1.2} \end{equation}

Lemma 9.1.1.

Let \(\phi(s)\in C_{loc}^{0,1}(\R)\) be a convex function

  1. If \(u\) is (9.1.2)'s weak subsolution, \(\phi'(s)\geq 0\text{.}\) Then \(v=\phi(u)\) is also a weak subsolution;

  2. If \(u\) is (9.1.2)'s weak supersolution, \(\phi'(s)\leq 0\text{.}\) Then \(v=\phi(u)\) is also a weak subsolution.

\end{proof}

Lemma 9.1.2. Local maximum principle.

Let \(v\in H^{1,2}(B_R)\) be a weak subsolution. Then for any \(p>0\text{,}\) \(0\lt \theta\lt 1\text{,}\) we have

\begin{equation*} \underset{{B_{\theta R}}}{ess\,sup}\, v\leq C\left(\fint_{B_R}(v^+)^p\right)^{1/p}\text{,} \end{equation*}

where \(C\) only depends on \(n\text{,}\) \(\Lambda/\lambda\text{,}\) \(p\text{,}\) and \((1-\theta)^{-1}\text{.}\)

\begin{proof} The lemma is true even if \(v\) is not bounded. However, for the sake of simplicity, we assume that \(v\) is essentially bounded.

We first assume that \(p\geq 2\text{.}\) Since \(v\) is a weak subsolution, so is \(v^+\text{.}\) Thus we can assume that \(v\geq 0\text{.}\)

For any testing function \(\phi\geq 0\text{,}\) we have

\begin{equation*} \int_{B_R} a^{ij}D_ivD_j\phi \leq 0\text{.} \end{equation*}

We let \(\xi\in C_0^\infty(B_R)\) and \(\phi=\xi^2v^{p-1}\text{.}\) Then we have

\begin{equation*} (p-1)\int_{B_R}(a^{ij} D_i v D_j v)v^{p-2}\xi^2 dx \leq -2\int_{B_R}a^{ij} v^{p-1}\xi D_i v D_j\xi dx\text{.} \end{equation*}

Thus there is a constant \(C\) such that

\begin{equation*} \int_{B_R}|D(\xi v^{p/2})|^2 dx\leq C\int_{B_R}|D\xi|^2v^p dx\text{.} \end{equation*}

Using the Sobolev inequality, we have

\begin{equation*} \left(\int_{B_R}(\xi v^{p/2})^{2^*} dx\right)^{2/2^*}\leq C\int_{B_R}|D\xi|^2 v^p dx\text{,} \end{equation*}

where

\begin{equation*} 2^*=\frac{2n}{n-2}\text{.} \end{equation*}

Let

\begin{equation*} R_k=R\left(\theta+\frac{1-\theta}{2^k}\right)\text{.} \end{equation*}

Define \(\xi_k\in C_0^\infty(B_{R_k})\) such that \(\xi_k\equiv 1\) on \(B_{R_{k+1}}\text{.}\) Then

\begin{equation*} |D\xi_k|\leq\frac{2^{k+1}}{(1-\theta) R}\text{.} \end{equation*}

Using this we have

\begin{equation*} \left(\int_{B_{R_{k+1}}}v^{\frac{np}{n-2}}dx\right)^{\frac{n-2}{n}}\leq\frac{C\cdot 4^k}{(1-\theta)^2R^2}\int_{B_{R_k}}v^p dx\text{.} \end{equation*}

Thus we have

\begin{equation*} ||v||_{L^{\frac{np}{n-2}}}(B_{R_{k+1}})\leq (C\cdot 4^k)^{1/p}||v||_{L^p}(B_{R_k})\text{.} \end{equation*}

We let

\begin{equation*} p_1=\frac{np}{n-2},\cdots, p_k=\left(\frac{n}{n-2}\right)^k p\text{.} \end{equation*}

Then we have

\begin{equation*} ||v||_{L^{p_{k+1}}}(B_{R_{k+1}})\leq (C\cdot 4^k)^{1/p_k}||v||_{L^{p_k}}(B_{R_k})\text{.} \end{equation*}

Since

\begin{equation*} \log\left(\prod_{k=1}^\infty (C\cdot 4^k)^{1/p_k}\right) =\sum_{k=1}^\infty\frac{1}{p_k}(\log C+k\log 4)\lt C\text{,} \end{equation*}

there is a constant \(C\) such that

\begin{equation*} ||v||_{L^\infty}(B_{\theta R})\leq\frac{C}{((1-\theta)R)^{n/p}}\frac{1}{R^n}||v||_{L^p}(B_R)\text{.} \end{equation*}

It remains to prove that the lemma is true for \(0\lt p\lt 2\text{.}\) To see this we first use the inequality for \(p=2\text{.}\) That is

\begin{equation*} \sup_{B_{\theta R}}\, v\leq\frac{C}{(1-\theta)^{n/2}}\left(\fint_{B_R}u^2\right)^{1/2}\text{.} \end{equation*}

We have

\begin{equation*} \left(\int_{B_R}v^2\right)^{1/2}\leq\sup_{B_R}\, v^{1-p/2}\cdot\left(\int_{B_R}v^p\right)^{1/2}\text{.} \end{equation*}

Using Young's inequality, we have

\begin{equation*} \phi(\theta R)=(1-\frac p2)\phi(R)+\frac{C}{((1-\theta)R)^{n/p}}||v||_{L^p}\text{,} \end{equation*}

where

\begin{equation*} \phi(s)=\sup_{B_s}\, v\text{.} \end{equation*}

Thus we have the inequality

\begin{equation*} \phi(t)\leq (1-\frac p2)\phi(s)+\frac{A}{(t-s)^\alpha}||v||_{L^p} \end{equation*}

for any \(t\geq s\text{.}\) Let

\begin{equation*} t_0=R_0\lt R,\qquad t_{i+1}=t_i+(1-\tau)t^i(R-R_0) \end{equation*}

and \(0\lt \tau\lt 1\) to be determined later. Then we have

\begin{equation*} \phi(t_i)\leq(1-\frac p2)\phi(t_{i+1})+\frac{A}{((1-\tau)t^i(R-R_0))^\alpha}||u||_{L^2}\text{.} \end{equation*}

By iteration, we know that

\begin{equation*} \phi(R_0)\leq A||u||_{L^2}\frac{1}{((1-\tau)t^i(R-R_0))^\alpha} \left((1-\frac p2)\tau^\alpha+(1-\frac p2)^2\tau^{-2\alpha}+\cdots\right)\text{.} \end{equation*}

We can choose \(\tau\) close to \(1\) such that \((1-p/2)t^{-\alpha}\lt 1\text{.}\) Then the above sequence is convergent and the theorem is proved.

Lemma 9.1.3. Weak Harnarck inequality.

Let \(v\) be the weak supersolution of (9.1.2). Then there is a \(p_0>0\) such that

\begin{equation*} \underset{B_{\theta R}}{{ ess}\,\inf}\, v\geq\frac{1}{C}\left(\fint_{B_R}v^{p_0} dx\right)^{1/p_0} \end{equation*}

Proof.

We assume that \(v>0\) on \(B_R\text{.}\) Otherwise we can use \(v+\eps\) instead.

Since \(v\) is a supersolution, \(v^{-1}\) is a subsolution. By the above lemma, we have

\begin{equation*} \underset{B_{\theta R}}{{ ess}\,\sup}\, v^{-p}\leq C\fint_{B_R} v^{-p} dx\text{,} \end{equation*}

which is equivalent to

\begin{equation*} \underset{B_{\theta R}}{{ ess}\,\inf}\, v\geq C \left(\fint_{B_R} v^{-p} dx\right)^{-\frac 1p} \end{equation*}

We write

\begin{align*} \amp \underset{B_{\theta R}}{{ ess}\,\inf}\, v\geq C \left(\fint_{B_R} v^{-p} dx\right)^{-\frac 1p}\\ \amp =C\left(\fint_{B_R}v^{-p} dx\cdot\fint_{B_R}v^p dx\right)^{-1/p}\cdot\left(\fint_{B_R}v^p dx\right)^{1/p}\text{.} \end{align*}

We just need to prove that there exists a \(p_0>0\) such that

\begin{equation*} \fint_{B_R}v^{-p} dx\cdot\fint_{B_R}v^p dx\leq C\text{.} \end{equation*}

We let \(w=\log v-\beta\text{,}\) where \(\beta\) is a constant to be determined. We wish to prove that

\begin{equation} \int_{B_R}e^{p|w|} dx\leq C\tag{9.1.3} \end{equation}

for some \(p>0\text{.}\) If the above inequality is true then we have

\begin{equation*} \int_{B_R}e^{p\log v-p\beta} dx\leq C,\qquad \int_{B_R}e^{-p\log v+p\beta} dx\leq C\text{,} \end{equation*}

and the lemma follows.

The proof of  (9.1.3) is somewhat complicated but the basic idea is still Moser iteration. Let \(v^{-1}\xi^2\) be a test function, where \(\xi \in C_0^\infty(B_R)\text{,}\) we have

\begin{equation*} 0\geq\int a^{ij} v_i(v^{-1}\xi^2)_j=-\int \xi^2a^{ij}w_i w_j+2\int a^{ij} w_i \xi\xi_j\text{.} \end{equation*}

Without loss of generality, we assume that \(R=1\text{.}\) Then we have

\begin{equation*} \int_{B_\sigma}|Dw|^2\leq C \end{equation*}

for any \(0\lt \sigma\lt 1\text{.}\) By choosing suitable \(\beta\text{,}\) we have the Poincaré inequality

\begin{equation*} \int_{B_\sigma}|w|^2 dx\leq C\text{.} \end{equation*}

Let \(\xi\in C_0^\infty(B_1)\) and let \(\phi=\xi^2|w|^{2q}\text{.}\) Then we have

\begin{equation*} \int_{B_\sigma}\xi^2|w|^{2q}|D_i w|^2\leq 2q\int_{B_\sigma}\xi^2|w|^{2q-1} D_i w D_i|w|+2\int_{B_\sigma}\xi|w|^{2q}D_i wD_i \xi\text{.} \end{equation*}

Using the Young's inequality, we have

\begin{equation*} 2q|w|^{2q-1}\leq\frac{2q-1}{2q}|w|^{2q}+(2q)^{2q-1}\text{.} \end{equation*}

Thus we have

\begin{equation*} \frac{1}{2q}\int_{B_\sigma}\xi^2|w|^{2q}|D_iw|^2\leq (2q)^{2q-1}\int_{B_\sigma}|D_i w|^2+4q\int_{B_\sigma}|w|^{2q}|D_i\xi|^2\text{.} \end{equation*}

Finally, we get

\begin{equation*} \int_{B_\sigma}|D(\xi^2|w|^{2q})|^2 dx\leq C(2q)^{2q}+C\tau^{-2}q^2\int_{B_{\sigma+\tau}}|w|^{2q} dx\text{,} \end{equation*}

where \(C\) only depends on \(n\text{,}\) \(\Lambda/\lambda\text{,}\) and \((1-\sigma)^{-1}\text{.}\) Let \(\kappa=n/(n-1)\text{.}\) By Sobolev inequality, we have

\begin{equation*} \left(\int_{B_\sigma}|w|^{2q\kappa} dx\right)^{1/\kappa}\leq C(2q)^{2q}+C\tau^{-2}q^2\int_{B_{\sigma+\tau}}|w|^{2q} dx\text{.} \end{equation*}

We let

\begin{equation*} q_i=\kappa^{i-1},\delta_0=\sigma, \delta_i=\delta_{i-1}-\frac{\sigma-1}{2^i} \end{equation*}

Then

\begin{equation*} \left(\int_{B_{\delta_i}}|w|^{2\kappa^i} dx\right)^{1/\kappa} \leq C(\kappa)^{2(i-1)\kappa^{i-1}}+C(4\kappa)^i\int_{B_{\delta_{i-1}}}|w|^{2\kappa^{i-1}}dx \end{equation*}

Let

\begin{equation*} I_j=\left(\int_{B_{\delta_j}}|w|^{2\kappa^j}\right)^{1/2\kappa^j}\text{.} \end{equation*}

Then

\begin{equation*} I_i\leq C^{1/\kappa^{i-1}}\kappa^{i-1}+C^{1/\kappa^{i-1}}(4\kappa)^{i/\kappa^{i-1}}I_{i-1}\text{.} \end{equation*}

Running the iteration we get

\begin{equation*} I_j\leq C\kappa^j+cI_0\text{.} \end{equation*}

Thus we have

\begin{equation*} ||w||_{L^q(B_1)}\leq Cq\text{.} \end{equation*}

If \(pCe\lt 1\text{,}\) then we must have

\begin{equation*} \int_{B_1}e^{p|w|}\leq C\text{.} \end{equation*}

Theorem 9.1.4. Harnack Inequality.

Let \(u\) be a nonnegative solution of  (9.1.2). Then we have

\begin{equation*} \underset{B_{\theta R}}{{ ess}\,\sup}\,u\leq C \underset{B_{\theta R}}{{ ess}\,\inf}\,u \end{equation*}

where \(C\) only depends on \(n, \lambda/\lambda, (1-\theta)^{-1}\text{.}\)

Theorem 9.1.5. Hölder estimate.

Using the above notations, \(u\) is Hölder continuous in \(B_{\theta R}\text{.}\)

Proof.

Let

\begin{equation*} M(R)=\underset{B_{\theta R}}{{ ess}\,\sup}\,u, m(R)=\underset{B_{\theta R}}{{ ess}\,\inf}\,u\text{.} \end{equation*}

Let \(w(R)=M(R)-m(R)\text{.}\) Then

\begin{equation*} u-m(R)\geq 0 \end{equation*}

on \(B_{\theta R}\text{.}\) Thus

\begin{equation*} M(\theta R)-m(R)\leq C(m(\theta R)-m(R))\text{.} \end{equation*}

Therefore

\begin{equation*} w(\theta R)\leq \left(1-\frac 1C\right)w(R) \end{equation*}

hence

\begin{equation*} w(\theta^s R)\leq \left(1-\frac 1C\right)^sw(R)\text{.} \end{equation*}

Let \(R'\lt R\text{.}\) Let \(s\) be such that

\begin{equation*} \theta^{s+1}R\leq R'\lt \theta^s R\text{.} \end{equation*}

Then

\begin{equation*} w(R')\leq w(\theta^sR)\leq \left(1-\frac 1C\right)^sw(R)\text{.} \end{equation*}

Since

\begin{equation*} s=\left[\frac{\log (R'/R)}{\log\theta}\right]-1\text{,} \end{equation*}

We have

\begin{equation*} w(R')\leq\left(\frac{R'}{R}\right)^rw(R)\text{,} \end{equation*}

where

\begin{equation*} r=\left(1-\frac 1C\right)^{(\log\theta)^{-1}}\lt 1 \end{equation*}

Let \(x,y\in B(\theta R)\text{.}\) Then we have

\begin{equation*} |u(x)-u(y)|\leq C|x-y|^r \end{equation*}

and the Hölder continuity follows.
