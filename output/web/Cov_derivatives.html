Section 1.3 Covariant derivatives

The notation of connection on tensors field can be applied to co-tensors and yield the following

Lemma 1.3.1.

Let

\begin{equation*} \eta=a_{i_1\cdots i_p}\omega_{i_1}\otimes\cdots\otimes\omega_{i_p}\text{.} \end{equation*}

Then

\begin{equation*} \nabla_{e_k}\eta=\left(e_k(a_{i_1\cdots i_p})-\sum_s a_{i_1\cdots \underset{\underset{sth}{\uparrow}}r\cdots\cdots i_p}\omega_{r\, i_s}(e_k)\right) \omega_{i_1}\otimes\cdots\otimes\omega_{i_p}\text{.} \end{equation*}

Proof.

A straightforward computation.

Because of the above result, we make the following definition:

Definition 1.3.2.

We define

\begin{equation*} a_{i_1\cdots i_p,k}=e_k(a_{i_1\cdots,i_p})-\sum_s a_{i_1\cdots \underset{\underset{sth}{\uparrow}}r\cdots i_p}\omega_{r\, i_s}(e_k) \end{equation*}

and call it the covariant derivative of the coeficients \(a_{i_1\cdots i_p}\text{.}\)

By the above definition, we have

\begin{equation*} \nabla \eta=a_{i_1\cdots i_p,k}\,\omega_{k}\wedge \omega_{i_1}\otimes\cdots\otimes\omega_{i_p}\text{.} \end{equation*}

We have the following

Theorem 1.3.3.

We have

\begin{equation*} a_{i_1\cdots i_p,k,\ell}-a_{i_1\cdots i_p,\ell,k} = a_{i_1\cdots \underset{\underset{sth}{\uparrow}}r\cdots i_p} R_{r i_s k\ell}\text{.} \end{equation*}

Proof.

To prove the above result, we introduce some notations. We let \(I=(i_1,\cdots,i_p)\) and let \(\omega_I=\omega_{i_1}\otimes\cdots\otimes\omega_{i_p}\text{.}\) Then we have

\begin{align*} \amp \frac 12(a_{I,k,\ell}-a_{I,\ell,k})\,\omega_k\wedge\omega_\ell\otimes \omega_I\\ \amp =a_{I,k,\ell}\,\omega_k\wedge\omega_\ell\otimes\omega_I\\ \amp =d_{\nabla^2}\eta\text{.} \end{align*}

Using the above notations, we can re-write the definition of covariant derivatives as following

\begin{equation*} \nabla\eta=d a_I-a_{I_r}\,\omega_{r i_s}\text{,} \end{equation*}

where

\begin{equation*} I_r=(i_1\cdots \underset{\underset{sth}{\uparrow}}r\cdots i_p)\text{.} \end{equation*}

We compute

\begin{align} \begin{split} \amp \qquad\qquad d_{\nabla^2}\eta=d_\nabla\left((\nabla\eta)_{I,k}\omega_k\otimes\omega_I\right)\tag{1.3.1}\\ \amp =-\left(d(\nabla\eta)_{I,k}-(\nabla\eta)_{I_r,k}\omega_{r i_s}-(\nabla\eta)_{I,r}\omega_{rk}\right)\wedge \omega_k\otimes\omega_I. \end{split}\notag \end{align}

The first and the third term of the above can be consolidated because

\begin{equation*} d(\nabla\eta)_{I,k}\wedge\omega_k-(\nabla\eta)_{I,r}\omega_{rk}\wedge\omega_k =d((\nabla\eta)_{I,k}\omega_k) \end{equation*}

by the first Cartan's formula. Thus we have

\begin{equation*} d(\nabla\eta)_{I,k}\wedge\omega_k-(\nabla\eta)_{I,r}\omega_{rk}\wedge\omega_k =d(d a_{I_r}-a_{I_r}\omega_{ri_s})=-da_{I_r}\wedge\omega_{r i_s}-a_{I_r}d\omega_{r i_s}\text{.} \end{equation*}

Combining the above equation with  (1.3.1), we have

\begin{align*} \amp d_{\nabla^2}\eta=-(-da_{I_r}\wedge\omega_{r i_s}-a_{I_r}d\omega_{r i_s})\otimes \omega_I -(\nabla\eta)_{I_r,k}\omega_k\wedge\omega_{r i_s}\otimes\omega_I\\ \amp =a_{I_r}(d\omega_{r i_s}+\omega_{r\ell}\wedge\omega_{\ell i_s})\otimes\omega_I+\sum_{s\neq t}a_{i_1\cdots \underset{\underset{tth}{\uparrow}}\mu\cdots \underset{\underset{sth}{\uparrow}}r\cdots i_p}\omega_{\mu i_t}\wedge \omega_{r i_s}\otimes\omega_I\text{.} \end{align*}

For fixed \(s\neq t\text{,}\) we have

\begin{equation*} a_{i_1\cdots \underset{\underset{tth}{\uparrow}}\mu\cdots \underset{\underset{sth}{\uparrow}}r\cdots i_p}\omega_{\mu i_t}\wedge \omega_{r i_s}\otimes\omega_I+a_{i_1\cdots \underset{\underset{tth}{\uparrow}}\mu\cdots \underset{\underset{sth}{\uparrow}}r\cdots i_p}\omega_{\mu i_s}\wedge \omega_{r i_t}\otimes\omega_I=0\text{.} \end{equation*}

Thus we have

\begin{equation*} d_{\nabla^2}\eta=\frac 12 a_{I_r}R_{r i_sk\ell}\,\omega_k\wedge\omega_\ell\otimes\omega_I\text{,} \end{equation*}

and the lemma is proved.

Let \(f\) be a smooth function. The following notations are often used

\begin{align*} \amp f_i=e_i(f);\\ \amp f_{ij}=f_{i,j};\\ \amp f_{ijk}=f_{i,j,k}\text{.} \end{align*}

We have

Corollary 1.3.4. Ricci identity.

Using the above notations, we have

\begin{align*} \amp f_{ij}=f_{ji};\\ \amp f_{ijk}-f_{ikj}=f_r R_{rijk}\text{.} \end{align*}

In particular, we have the following useful Ricci identity:

\begin{equation*} \sum_i f_{iji}-\sum_i f_{iij}=(Ric)_{rj}f_r\text{.} \end{equation*}

Proof.

We only need to prove \(f_{ij}=f_{ji}\text{.}\) To see this, we compute

\begin{equation*} f_{ij}\,\omega_j\wedge\omega_i =(df_i-f_r\omega_{ri})\wedge \omega_i=df_i\wedge\omega_i-f_r\,\omega_{ri}\wedge\omega_i\text{.} \end{equation*}

By the Cartan's formula, the right hand side of the above is equal to

\begin{equation*} d(f_i\omega_i)=ddf=0\text{.} \end{equation*}
