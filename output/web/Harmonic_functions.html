Section 2.3 Harmonic functions revisited

We assume that \(u\) is a positive harmonic function defined on \(B(a) \subset M\text{,}\) where \(M\) is a complete Riemannian manifold with \({ Ric} (M) \geq - (n-1) k\text{.}\)

In this section, we re-prove the Harnack inequality using the de Giorgi-Nash-Moser estimates. Note that our result is weaker than the differential Li-Yau Harnark inequality. However, the methods we use here are useful in non-linear case, even if we only use a linear problem as the example.

We first prove the following estimate.

Theorem 2.3.1.

Under the above assumptions, then for any \(p > 0 , 0 \lt \theta \lt 1\text{,}\) there is a constant

\begin{equation*} C= C (n, p, \theta, k ) > 0 \end{equation*}

such that

\begin{equation*} \sup_{B(a\theta)} u \leq C (f _{B(a)} u^p ) ^{\frac{1}{p}}\text{.} \end{equation*}

\begin{proof} We first assume that \(p \geq 2\text{.}\) Let \(\phi \geq 0 , \phi \in C^\infty_0 (B(a))\text{.}\) Then since \(\Delta u = 0\text{,}\) we have

\begin{equation*} \int _{B(a)} \nabla u \nabla \phi = 0\text{.} \end{equation*}

Let \(\rho\) be a smooth function with compact support in \(B(a)\text{.}\) Then we have

\begin{equation*} \int _{B(a)} \nabla u \nabla (\rho^2 u ^{p-1} ) = 0\text{.} \end{equation*}

Here \(p\) is a real number to be specialized later.

Expanding the above inequality, we get

\begin{equation*} (p-1) \int _{B(a)} \rho^2 u ^{p -2} | \nabla u|^2 \leq - 2 \int _{B(a)} u ^{p-1} \rho \nabla u \nabla \rho\text{.} \end{equation*}

Using the Cauchy inequality, we get

\begin{equation*} (p-1)^2 \int _{B(a)} \rho^2 u ^{p -2} | \nabla u|^2 \leq C \int _{B(a)} |\nabla \rho | u^p\text{.} \end{equation*}

Note that

\begin{equation*} u ^{p-2} | \nabla u|^2 = \frac{4}{p^2} \left|\nabla u ^{\frac{p}{2}} \right|^2 \end{equation*}

we have

\begin{equation*} \left. \int _{B(a)} \rho^2 | \nabla u ^{\frac{p}{2}} \right|^2 \leq C \int _{B(a)} | \nabla \rho| ^2 u^p \end{equation*}

if we allow \(C\) to be a little bigger, we shall get

\begin{equation*} \left.\int _{B(a)} | \nabla (\rho u ^{\frac{p}{2}} )\right|^2 \leq C \int _{B(a)} | \nabla \rho| ^2 u^p\text{.} \end{equation*}

We let \(2^\ast = \frac{2n}{n-2} > 2\text{.}\) Using the Sobolev inequality we have

\begin{equation} \left(\int _{B(a)}( \rho u ^{\frac{p}{2}}) ^{2^{\ast}} \right) ^{\frac{1}{2^{\ast}}}\leq C \int _{B(a)} | \nabla \zeta |^2 u^p . \tag{$\star$}\tag{2.5} \end{equation}

We let

\begin{equation*} R_k = a \left(\theta + \frac{1-\theta}{2^k}\right)\text{.} \end{equation*}

Let \(\rho_k \in C^\infty_0 (B (R_k)) , 0 \leq \rho _k \leq 1, \rho _k \equiv 1\) on \(B(R_{k+1} )\text{.}\) We further assume that

\begin{equation*} | \nabla \rho_k| \leq \frac{2}{R_k - R_{k+1}} = \frac{2^{k+1}}{(1-\theta)a}\text{.} \end{equation*}

From (\(\star\)), we have

\begin{equation*} \left( \int_{B(R_{k+1})} U ^{\frac{np}{n-p}}\right)^{\frac{n-2}{n}} \leq \frac{C \cdot 4^k}{(1 - \theta )^2 a^2} \int _{B(R_k)} u^p\text{.} \end{equation*}

Specializing \(p = p_k\text{,}\) where \(p_k = p \left( \frac{n}{n-2} \right) ^k\text{,}\) we have

\begin{align*} \amp \amp \Vert u \Vert _{L^{p_{k+1}} (B (R _{k+1} ))} \leq \left ( \frac{C \cdot 4^k}{(1 - \theta )^2 a^2} \right) ^{\frac{1}{p_k}}\\ \amp \amp \times \Vert u \Vert _{L^{p} (B (R _k ))}\text{.} \end{align*}

Iterating, we have

\begin{equation*} \Vert u \Vert_{L^{p_{k+1}} (B (R _{k+1} ))} \leq \Pi \left ( \frac{C \cdot 4^k}{(1 - \theta )^2 a^2} \right) ^{\frac{1}{p_k}} \Vert u \Vert _{L^{p} (B (a))}\text{.} \end{equation*}

We need to prove that

\begin{equation} \Pi \left ( \frac{C \cdot 4^k}{(1 - \theta )^2 a^2} \right) ^{\frac{1}{p_k}} \leq \frac{C }{((1 - \theta )^2 a^2)^{\frac{n}{2p}}} \tag{{ Exercises}}\tag{2.6} \end{equation}

Since the right-hand side is independent of \(k\text{,}\) we let \(k \rightarrow \infty\) and get

\begin{equation*} \Vert u \Vert _{L^{\infty} (B (\theta a ))} \leq \frac{C }{((1 - \theta ) a) ^{\frac{n}{p}}} \Vert u \Vert _{L^{p} (B (a))}\text{.} \end{equation*}

This proves the theorem for \(p \geq 2\text{.}\)

Now we assume that \(0 \lt p \lt 2\text{.}\) Using the result for \(p = 2\text{,}\) we have

\begin{align*} \Vert u \Vert _{L^{p_\infty} (B (\theta a ))} \amp \leq \amp \frac{C }{((1 - \theta ) a)^{\frac{n}{2}}} \Vert u \Vert _{L^{\infty} (B (a))^{1 - \frac{p}{2}}}\\ \amp \times \amp \left( \int _{B(a)} u ^p \right) ^{\frac{1}{2}}\text{.} \end{align*}

Using the Young inequality, we get

\begin{align*} \lefteqn{\Vert u \Vert _{L^{\infty} (B (\theta a ))} \leq \frac{1 }{2} \Vert u \Vert _{L^{p} (B (a))}}\\ \amp \amp + \frac{C }{((1 - \theta ) a)^{\frac{n}{2}}} \Vert u \Vert _{L^{p} (B (a))}\text{.} \end{align*}

Let \(\phi (s) = \Vert u \Vert_{L^{\infty} (B (a))}\text{,}\) we get

\begin{equation*} \phi (s) \leq \frac{1}{2} \phi (t) + \frac{C}{(t-s)^{\frac{n}{p}}} \Vert u \Vert _{L^{p} (B (a))} \end{equation*}

\(\forall \ 0 \lt s \lt t \leq a\text{.}\) Iterating again, we get

\begin{equation} \phi (s) \leq \frac{C}{(1-s)a)^{\frac{n}{p}}} \Vert u \Vert _{L^{p} (B (a))} \tag{{ Ex}}\tag{2.7} \end{equation}

Theorem 2.3.2. Weak Harnack inequality.

There is a constant \(C > 0 , p_0 > 0\) such that

\begin{equation*} \inf_{B(a \theta)} u \geq \frac{1}{C} \left( f _{B(a)} u ^{p_{0}} \right) ^{\frac{1}{p_{0}}}\text{.} \end{equation*}

Here \(p_0, C\) only depends on \(a, (1 - \theta )^{-1}\) and Sobolev constants.

\begin{proof} Without loss of generality, we assume that \(u \geq \varepsilon > 0\text{.}\) Otherwise we can use \(u + \varepsilon\) in place of \(u\text{.}\) We also assume that \(a = 1\text{.}\) By a straight forward computation we get that

\begin{equation*} \Delta u^{-1} = \frac{2|\nabla u|^2}{u^3} \geq 0\text{.} \end{equation*}

Thus \(u^{-1}\) is a subsolution. Using the above lemma, for any \(p\text{,}\) we have

\begin{equation*} \sup _{B(\theta)} u^{-p} \leq C \int_{B(1)} u ^{-p}\text{.} \end{equation*}

Thus we must have

\begin{align*} \inf _{B(\theta)}u \amp \geq \amp C ^{-\frac{1}{p}} \left( \int_{B_{1}} u^{-p} d x \right) ^{-\frac{1}{p}}\\ \amp = \amp C ^{-\frac{1}{p}} \left[\int_{B_{1}} u^{-p} \cdot \int_{B_{1}} u^{p} \right] ^{\frac{1}{p}}\\ \amp \times\amp \left( \int_{B_{1}} u^{p} \right) ^{\frac{1}{p}} \text{.} \end{align*}

In order to prove the theorem, we just need to prove that for \(p > 0\) small enough

\begin{equation*} \int_{B(1)} u^{-p} \int _{B_1} u^p \leq C\text{.} \end{equation*}

We let

\begin{equation*} \omega = \log u - \beta \end{equation*}

where

\begin{equation*} \beta = f \log u\text{.} \end{equation*}

We shall establish

\begin{equation} \int_{B(1) } e ^{p |\omega |}\leq C \tag{$\star$}\tag{2.8} \end{equation}

for \(p > 0\) small enough. We first prove that

\begin{equation*} \int_{B(\sigma)} | \nabla \omega |^2 \leq C \end{equation*}

for some \(\sigma > 1\text{.}\) To see this, let \(\rho\) be the cut-off function whose support is within \(B(\bar{\sigma})\) for some \(\bar{\sigma} > \sigma\text{.}\) Since \(u\) is harmonic, we have

\begin{equation*} \int _{B(\bar{\sigma})} \nabla u \nabla ( u^{-1} \rho ^2) = 0\text{.} \end{equation*}

It follows that

\begin{align*} \int_{B(\bar{\sigma})} \rho^2 |\nabla \omega |^2 \amp \leq \amp \int_{B(\bar{\sigma})} \nabla \omega \nabla \rho^2\\ \amp \leq\amp 2 \sqrt{\int_{B(\bar{\sigma})}\rho^2 |\nabla \omega |^2 }\sqrt{\int_{B(\bar{\sigma})} |\nabla \rho |^2} \text{.} \end{align*}

Since \(\rho\) only depends on \(\sigma , \bar{\sigma}\text{,}\) we get the desired inequality.

By the Poincaré inequality, we have

\begin{equation*} \int_{B(\bar{\sigma})} | \omega |^2 \leq C\text{.} \end{equation*}

To get the estimate \((\star)\text{,}\) we still use the Moser iteration. First observe that

\begin{equation*} \Delta \omega = - | \nabla \omega |^2\text{.} \end{equation*}

Let \(\rho\) be a cut-off function to be determined later. Then we have

\begin{equation*} - \rho ^2 |\omega |^{2 q} \Delta \omega = \rho^2 | \omega |^{2q} | \nabla \omega |^2\text{.} \end{equation*}

It follows that

\begin{align*} \int \rho^2 |\omega |^{2q} |\nabla \omega | ^2 \amp = \amp \int \nabla \omega \nabla ( \rho^2 |\omega |^{2q})\\ \amp = \amp 2 \int \rho \nabla \rho \omega |\omega |^{2q} + 2 q \int \rho^2 |\omega | ^{2q-1} |\nabla \omega |^2\text{.} \end{align*}

We use the Young inequality to get

\begin{equation} 2q |\omega |^{2q-1} \leq \frac{2q-1}{2q} |\omega | ^{2q} + ( 2q) ^{2q-1} .\tag{$\triangle$}\tag{2.9} \end{equation}

Inserting the above inequality into the equation, we get

\begin{align*} \int \rho ^2 |\omega |^{2q} |\nabla \omega |^2 \amp \leq \amp (2q)^{2q} + 2 \sqrt{\int \rho^2 |\omega |^{2q} | \nabla \omega |^2}\\ \amp \times \amp \sqrt{\int |\nabla \rho |^{2} | \omega |^{2q} }\text{.} \end{align*}

Using the Cauchy inequality \(ab \leq \varepsilon a^2 + \frac{1}{\varepsilon} b^2\text{,}\) we get

\begin{equation*} \int \rho^2 |\omega |^{2q} |\nabla \omega |^2 \leq C (( 2g) ^{2q} + \int |\omega | ^{2q} | \nabla \rho |^2)\text{.} \end{equation*}

With a slightly larger constant \(C\text{,}\) we have

\begin{equation*} \int \rho^2 |\nabla | \omega | ^{q+1} |^2 \leq C q^2 (( 2q)^{2q} + \int |\omega | ^{2q} | \nabla \rho |^2)\text{.} \end{equation*}

Thus we have

\begin{equation*} \int (\nabla ( \rho^2 |\omega |^{q+1}) |^2 \leq 2 C q^2((2q) ^{2q} + \int |\omega | ^{2q+2} | \nabla \rho |^2)\text{.} \end{equation*}

Thus replacing \(q + 1\) by \(q\) and choosing the suitable cut-off function, we get

\begin{equation*} \left( \int _{B(\bar{\sigma})} |\omega | {2^{qk}} \right) ^{\frac{1}{k}} \leq C \left(( 2q) ^{2q} + \tau ^{-2} q ^2 \int _{B(q + \tau)} |\omega | ^{2q}\right)\text{.} \end{equation*}

Let \(\kappa = \frac{n}{n-1}, \; q_i = \kappa ^{i-1} , \; \delta_0 = \bar{\sigma} > 1\text{.}\)

\begin{equation*} \delta _i = \delta _{i-1} - \frac{\bar{\sigma} - 1}{2^i}\text{.} \end{equation*}

Then we have

\begin{align*} \left( \int_{B(\delta i)} | \omega |^{2 +1}\right) ^{\frac{1}{k}}\amp \leq \amp C \kappa ^{2(i-1)} \kappa^{i-1}\\ \amp + \amp C (4 k )^i \int _{B (\delta _{i-1})} |\omega |^{2\kappa ^{i-1}} \end{align*}

We let

\begin{equation*} I_j = \left( \int _{B\partial _{1}} |\omega |^{2 k ^{j}} \right) ^{\frac{1}{2k^{3}}}\text{.} \end{equation*}

Then we get

\begin{equation*} I_1 \leq C ^{\frac{1}{k^{i-1}}} k ^{i-1} + C ^{\frac{1}{k^{i-1}}} (4 k) ^{\frac{1}{k^{i-1}}}I_{i-1}\text{.} \end{equation*}

Using the standard iteration we get

\begin{equation*} I_i \leq C + \frac{1}{2k ^i}\text{.} \end{equation*}

For any \(q \geq 2\) let \(j\) be such that

\begin{equation*} 2 \kappa ^{i-1} \leq q \leq 2\kappa^{ j}\text{.} \end{equation*}

Using Hölder and \(|| \omega ||_{L^{2}} \leq C\) we get

\begin{equation*} \Vert \omega \Vert_{L^{q}} C I_j \leq \tilde{C} q \; \geq 2\text{.} \end{equation*}

Since \(q ^q \leq e^q q !\text{,}\) we have

\begin{equation*} \int_{B_{1}} |\omega | ^q \leq \tilde{C} ^q q ^q \leq ( \tilde{C} e ^q) q !\text{.} \end{equation*}

For \(\varepsilon\) enough

\begin{equation*} \int_{B_{1}} e ^{\varepsilon |\omega|} \leq \sum^\infty_{q = 2} (C \varepsilon e )^q \leq C\text{.} \end{equation*}

This proved the weak Harnack inequality.

Theorem 2.3.3. Harnack inequality.

Let \(u\) be a positive harmonic function on \(B(a)\text{.}\) Then for any \(0 \lt \theta \lt 1\) we have

\begin{equation*} \sup _{B(a\theta)} u \leq C \inf_{B(a \theta) }u \end{equation*}

where \(C\) only depends on \(n, (1 - \theta )^{-1}\text{,}\) and the Sobolev constants.
