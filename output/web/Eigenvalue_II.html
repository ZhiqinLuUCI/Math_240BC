Section 5.2 Eigenvalue problems(II)

By the variational characterizing of the eigenvalues, we know that it is usually more difficult to get the lower bound estimate of eigenvalues. Among all the eigenvalues, the lower bound of the first eigenvalue is particularly important.

The Cheeger's result did give a lower bound estimate of the first eigenvalues. But the bounds are not “computable.” In geometry, “computable” bounds provide effective versions of Poincaré and Sobolev inequalities.

The following Lichnerowicz theorem gives a good lower bound of the first eigenvalue for closed manifold.

Theorem 5.2.1. Lichnerowicz.

Let \(M\) be a closed \(n\)-dimensional Riemannian manifold. Assume that

\begin{equation*} { Ric} (M) \geq (n-1) k > 0 \end{equation*}

Then \(\lambda _1 \geq n k\)

Proof.

One line proof: let \(u\) be the first eigenfunction. Then using the Ricci identity we have

\begin{equation*} \frac{1}{2} \Delta |\nabla u|^2 \geq \sum u _{ij} ^2 + \nabla u \Delta u + { Ric}(\nabla u, \nabla u)\text{.} \end{equation*}

We have

\begin{align*} \sum u^2_{ij} \amp \geq \amp \sum u^2 _{ii} \geq \frac{1}{2} \left(\sum u_{ii} \right) ^2 = \frac{\lambda_1^2}{n} uU^2\\ { Ric} (\nabla u, \nabla u) \amp \geq \amp (n-1) k |\nabla u |^2 \end{align*}

Thus we have

\begin{equation*} \frac{1}{2} \Delta |\nabla u|^2 \geq \frac{\lambda ^2_1}{n} u^2 - \lambda _1 |\nabla uU |^2 + (n-1)k | \nabla u|^2\text{.} \end{equation*}

Taking integration on both sides, we get

\begin{equation*} \frac{\lambda ^2_1}{n} - \lambda ^2_1 + (n-1) k \lambda _1 \leq 0\text{.} \end{equation*}

The theorem follows.

In 1962, Obata proved, if \(\lambda _1 = nk\text{,}\) then \(M\) has to be the standard sphere.

For the rest of this section, we use the gradient estimates to find “computable” lower bounds of the first eigenvalue.

We prove the following theorem.

Theorem 5.2.2. Li-Yau.

Let \(M\) be a closed manifold and

\begin{equation*} { Ric} (M)\geq 0\text{.} \end{equation*}

Then \(\lambda _ 1 \geq \pi ^2 / 2 d ^2\text{,}\) \(d\) is the diameter.

\begin{proof} Let \(u\) be the first eigenfunction. After normalization we may assume that

\begin{equation*} 1 = \sup u > \inf u = - k \geq -1 \end{equation*}

for some \(1 \geq k > 0\text{.}\) Let

\begin{equation*} \tilde{u} = \frac{u - \frac{1-k}{2}}{\frac{1+k}{2}}\text{.} \end{equation*}

Then after this linear change of \(u\text{.}\) We have

\begin{equation*} \left\{ \begin{array}{l} \Delta \tilde{u} = - \lambda _1 ( \tilde{u} + 1 )\\ \sup \tilde{u} = a \\ \inf \tilde{u} = - 1 \end{array} \right. \end{equation*}

for \(a = \frac{1-k}{1+k}\text{.}\) \(1 > a \geq 0\text{.}\)

Let \(g = \frac{1}{2} ( | \nabla \tilde{u} |^2 + ( \lambda _ 1 + \varepsilon ) \tilde{u} ^2)\) for some \(\varepsilon > 0\) to be determined later. Assume that at \(x _0\)

\begin{equation*} g(x_0) = { Max}\, g\text{.} \end{equation*}

Using the maximum principle, at \(x_0\text{,}\) we have

\begin{equation*} \tilde{u}_j \tilde{u}_{j} + (x_1 + \varepsilon ) \tilde{u} u _i = 0 \end{equation*}

and

\begin{align*} 0 \geq \Delta g \amp = \amp \hat{u} ^2 _{ij} { Ric} ( \nabla \tilde{u}, \nabla \tilde{u} ) + \nabla \tilde{u} \nabla \Delta \tilde{u}\\ \amp \amp + ( \lambda _1 + \varepsilon ) |\nabla \tilde{u}|^2 + ( \lambda _1 + \varepsilon ) \tilde{u} \Delta \tilde{u}\\ \amp \amp \geq \tilde{u}^2 _{ij} - \lambda _1 |\nabla \tilde{u}|^2 + ( \lambda _1 + \varepsilon ) |\nabla \tilde{u}|^2\\ \amp \amp - \lambda _1 (\lambda _1 + \varepsilon ) \tilde{u} ( \tilde{u} + a \end{align*}

if at \(x _0 , \nabla \tilde{U} = 0\text{.}\) Then we have

\begin{equation*} |\nabla \tilde{u}|^2 + ( \lambda _1 + \varepsilon ) \tilde{u} ^2 \leq ( \lambda _1 + \varepsilon )\text{.} \end{equation*}

In particular, we have

\begin{equation*} |\nabla \tilde{u} |^2 + \lambda _1 (1+a ) \tilde{u} ^2 \leq (1+a ) \end{equation*}

if \(\nabla \tilde{u} (x_0) \neq 0\text{.}\) Then using the Cauchy inequality

\begin{equation*} \tilde{u} ^2 _{ij} \geq \frac{(\tilde{u} _i \tilde{u}_j \tilde{u}_{ij} )^2}{|\nabla \tilde{u}|^4} = ( \lambda _1 + \varepsilon )^2 \tilde{u} ^2\text{.} \end{equation*}

Thus we have

\begin{align*} 0 \geq \Delta g (x_0) \amp \geq \amp ( \lambda _1 + \varepsilon )^2 \tilde{u}^2 + \varepsilon |\nabla \tilde{u}|^2\\ \amp \amp - \lambda _1 (\lambda _1 + \varepsilon) \tilde{u}^2 - \lambda _1 (\lambda _1 + \varepsilon )\\ \amp \amp \geq 2 \varepsilon g - \lambda _1 (\lambda _1 + \varepsilon )a\text{.} \end{align*}

For any \(\varepsilon > \lambda _1 a\text{,}\) the above gives

\begin{equation*} |\nabla \tilde{u}| ^2 + \lambda _1 (1 + a) \tilde{u}^2 \leq \lambda _1 (1 + a)\text{.} \end{equation*}

Let

\begin{equation*} f(t) = { arc} \sin \tilde{U} ( \sigma (t)) \end{equation*}

where \(\sigma (t)\) is the arc-length curve connecting the minimal point and the maximum point of \(\tilde{u}\text{.}\) Then by the above argument, we have

\begin{equation*} |f^\prime (t)|\leq \sqrt{\lambda _1 (1 + a) }\text{.} \end{equation*}

Let \(d\) be the diameter of the manifold, then we have

\begin{align*} d \sqrt{\lambda_1 (1+a)} \amp \geq \amp \int ^d _0 ( f ^\prime (t) ) d t \geq { arc} \sin 1\\ \amp \amp - { arc} \sin (-1) = \pi\text{.} \end{align*}

Thus

\begin{equation*} \lambda +1 \geq \frac{1}{1+a} \frac{\pi^2}{d^2}\text{.} \end{equation*}

Since \(a \lt 1\text{,}\) this gives

\begin{equation*} \lambda_1 \geq \frac{\pi^2}{2d^2} \end{equation*}

We let \(\theta = { arc} \sin \tilde{u}\text{.}\) Then Zhong-Yau proved \(\oint\) the following surprising theorem.

Theorem 5.2.3. ***.

Let

\begin{equation*} \psi (\theta) = \left(\frac{4}{\pi} \right)\theta + \cos \theta \sin \theta ) - 2 \sin \theta\text{.} \end{equation*}

Then

\begin{equation*} \frac{|\nabla \tilde{u}|^2}{1 - \tilde{u}^2} \leq \lambda _1 (1 + a \psi (\theta))\text{.} \end{equation*}

The method is maximal principle, very surprising and mysterious.

Using the above sharpened inequality, observed that \(\psi (\theta)\) is an odd function, we can prove that

\begin{equation*} \lambda _1 \geq \frac{\pi^2}{d^2}\text{.} \end{equation*}

Assume that \({ Ric} (M) \geq - (n-1) k\) for \(k > 0\text{.}\) Then by estimating \(| \nabla u|^2 + \lambda _1 ( 1 - u )^2\text{,}\) Li-Yau was able to prove that

\begin{equation*} \lambda _1 \geq \frac{C}{d^2} \exp ( - C_1 \sqrt{kd^2})\text{.} \end{equation*}

*** was able to modify the above and proved that

\begin{equation*} \lambda _1 \geq \frac{\pi^2}{d^2} \exp ( - C_1 \sqrt{kd^2})\text{.} \end{equation*}

When \({ Ric}(M) > 0\text{,}\) or \({ Ric}(M) \geq (n-1) k > 0\) the above inequality is not optimal. In fact, it is far from being optimal. Let \(M = s ^n\text{.}\) Then by Lichnerowicz theorem, \(\lambda _1\geq n\) (in fact, \(\lambda _ 1 = n\)). \(d (s^n) = \pi\text{.}\) Thus \(**\) gives

\begin{equation*} \lambda _1 \geq 1\text{.} \end{equation*}

In this direction, we have the following Peter Li Conjecture.

Conjecture 5.2.4. P. Li.

If \({ Ric} (M) \geq (n-1) k\text{.}\) Then

\begin{equation*} \lambda _ 1 \geq \frac{\pi^2}{d^2} + (n-1)k\text{.} \end{equation*}

Such a conjecture, if true, will sharpen both the result of Zhong-Yang and Lichnerowicz because by Myer's theorem

\begin{equation*} \frac{\pi^2}{d^2} \geq k\text{.} \end{equation*}

Not much was known to the proof of the conjecture. D. Yang proved that

\begin{equation*} \lambda _1 \geq \frac{\pi^2}{d^2} + \frac{1}{4} ( n-1) k \end{equation*}

Ling Jun proved a bigger number

\begin{equation*} \lambda _1 \geq \frac{\pi^2}{d^2} + \alpha ( n-1) k \alpha > \frac{1}{4}\text{.} \end{equation*}

On the other end, if \({ Ric} (M) \geq ( n-1) k, k > 0\text{.}\) Then

\begin{equation*} \lambda _1 \geq \frac{\pi^2}{d^2} - ( n-1) k\text{.} \end{equation*}

Recently, *** was able to prove that

\begin{equation*} \lambda _1 > \frac{\pi^2}{d^2}\text{.} \end{equation*}

Very interesting result.
