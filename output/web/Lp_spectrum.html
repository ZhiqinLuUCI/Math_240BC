Section 6.2 The \(L^p\)-spectrum of the Laplacian

Subsection 6.2.1 The Laplacian on \(L^p\) space

Definition 6.2.1.

A one-parameter semi-group on a complex Banach space \(B\) is a family \(T_t\) of bounded linear operators, where \(T_t: B\rightarrow B\) parameterized by real numbers \(t\geqslant 0\) and satisfies the following relations:

  • \ding{172}.

    \(T_0 = 1\text{;}\)

  • \ding{173}.

    If \(0\leqslant s_1 t \lt +\infty\text{,}\) then

    \begin{equation*} T_s T_t = T_{s+t} \end{equation*}
  • \ding{174}.

    The map

    \begin{equation*} t_1 f \mapsto T_t f \end{equation*}

    from \([0, +\infty)\times B\) to \(B\) is jointly continuous.

The (infinitesimal) generator \(Z\) of a one-parameter semi-group \(T_t\) is defined by

\begin{equation*} Zf = \lim_{t\rightarrow 0^+} t^{-1}(T_t f - f) \end{equation*}

The domain \(Dom(Z)\) of \(Z\) being the set of \(f\) for which the limit exists. It is evident that \(Dom(Z)\) is a linear space. Moreover, we have

Lemma 6.2.2.

The subspace \(Dom(Z)\) is dense in \(B\text{,}\) and is invariant under \(T_t\) in the sense that

\begin{equation*} T_t(Dom(Z)) \subset Dom(Z) \end{equation*}

for all \(t\geqslant 0\text{.}\) Moreover

\begin{equation*} T_t Zf = ZT_t f \end{equation*}

for all \(f\in Dom(Z)\) and \(t \geqslant 0\text{.}\)

Proof.

If \(f\in B\text{,}\) we define

\begin{equation*} f_t = \int_0^t T_x fdx \end{equation*}

The above integration exists in the following sense: since \(T_x f\) is a continuous function of \(x\text{,}\) we can define the integration as the limit of the corresponding Riemann sums. In a Banach space, absolute convergence implies the conditional convergence. Thus in order to prove the convergence of the Riemann sums, we only need to verify that

\begin{equation*} \int_0^t \|T_x f\|dx \end{equation*}

is convergent. But this follows easily from the joint continuity in the definition of semi-group. \(\|T_x f\|\) must be uniformly bounded for small \(x\text{.}\)

We compute

\begin{align*} \nonumber \amp \amp \lim_{h\rightarrow 0^+} h^{-1} (T_h f_t - f_t)\\ \nonumber \amp = \amp \lim_{h\rightarrow 0^+}\left\{h^{-1}\int_h^{t+h} T_x fdx - h^{-1}\int_0^t T_xfdx\right\}\\ \nonumber \amp = \amp \lim_{h\rightarrow 0^+}\left\{h^{-1}\int_t^{t+h} T_x fdx - h^{-1}\int_0^h T_xfdx\right\}\\ \nonumber \amp = \amp T_t f - f \end{align*}

Therefore, \(f_t \in Dom(Z)\) and

\begin{equation*} Z(f_t) = T_t f - f \end{equation*}

Since \(t^{-1}f_t \rightarrow f\) in norm as \(t\rightarrow 0^+\text{,}\) we see that \(Dom(Z)\) is dense in \(B\text{.}\)

The generator \(Z\text{,}\) in general, is not a bounded operator. However, we can prove the following

Lemma 6.2.3.

The generator \(Z\) is a closed operator.

Proof.

We first observe that

\begin{equation*} T_t f - f = \int_0^t T_x Z f dx \end{equation*}

if \(f\in Dom(Z)\text{.}\) To see this, we consider the function \(r(t) = T_t f - f - \int_0^t T_x Zfdx\text{.}\) Obviously we have \(r(0) = 0\text{,}\) and \(r'(t)\equiv 0\text{.}\) Thus \(r(t)\equiv 0\text{.}\)

Using the above formula, we have

\begin{equation*} T_f f - f = \lim_{n\rightarrow\infty}(T_t f_n - f_n) = \lim_{n\rightarrow\infty} \int_0^t T_x Z f_n dx \end{equation*}

By the Lebegue theorem, the above limit is equal to

\begin{equation*} \int_0^t T_x g dx \end{equation*}

Thus we have

\begin{equation*} \lim_{t\rightarrow 0^+}t^{-1}(T_t f - f) = g \end{equation*}

and therefore \(f\in Dom(Z)\text{,}\) \(Zf = g\text{.}\)

Lemma 6.2.4.

If \(B\) is a Hilbert space, then \(Z\) must be densely defined and self-adjoint.

Let \(M\) be a manifold of dimension \(n\text{,}\) not necessarily compact or complete. The semi-group can formally be defined as

\begin{equation*} e^{\Delta t} \end{equation*}

More precisely, the following result is true

Theorem 6.2.5.

Let \(M\) be a manifold, then there is a heat kernel

\begin{equation*} H(x,y,t) \in C^{\infty}(M\times M\times \mathbb{R}^t) \end{equation*}

such that

\begin{equation*} (T_t f)(x) = \int_M H(x,y,t)f(y)dy \end{equation*}

satisfying

  • \ding{172}.

    \(H(x,y,t) = H(y,x,t)\text{.}\)

  • \ding{173}.

    \(\lim_{t\rightarrow 0^+} H(x,y,t) = \delta_x(y)\text{.}\)

  • \ding{174}.

    \((\Delta - \frac{\partial}{\partial t})H = 0\text{.}\)

  • \ding{175}.

    \(H(x,y,t) = \int_M H(x,z,t-s)H(z,y,s)dz\text{.}\)

In [Getzler], the above theorem was proved. One of the feature of the above theorem is that the proof is independent to the fact that \(\Delta\) can be extended as a densely defined self-adjoint operator on \(L^2(M)\text{.}\) In particular, we don't need to assume \(M\) to be complete. The infinitesimal generator on \(L^2(M)\) is in fact the Dirichlet Laplacian.

We let \(\Delta_p\) denote the Laplacian on \(L^p\) space. With this notation, for most of the theorems in linear differential geometry, the completeness assumption can be removed.

Example 6.2.6.

Let \(f\in Dom(\Delta_2)\) such that \(f\in L^2(M)\) and \(\Delta f = 0\text{.}\) Then \(f\) has to be a constant.

When \(M\) is a complete manifold, the above is a theorem of Yau. However, it is interesting to see that even when \(M\) is incomplete, the above result is still true, and the proof is exactly the same as the original proof of Yau.

Examining some special cases of the above setting is interesting.

  • \textcircled{A}.

    If \(\partial M\neq \emptyset\) and if \(\partial M\) is an \((n-1)\)-dimensional manifold, then \(A_2\) is the Dirichlet Laplacian.

  • \textcircled{B}.

    If \(M = \mathbb{R}^n - \{0\}\text{.}\) Then if \(f\in L^2(M)\text{,}\) \(\Delta f = 0\) and \(f \in Dom(\Delta_2)\text{.}\) Then \(f(0) = 0\) and \(f\) must be bounded near \(0\text{.}\) By the removable singularity theorem, \(f\) extends to a harmonic function on \(\mathbb{R}^n\text{,}\) which must be a constant.

  • \textcircled{C}.

    \(\Delta_2\) is particularly useful on moduli spaces, where it is very difficult to describe the boundary.

Subsubsection 6.2.1.1 Variational characterization of spectrum

Unlike in the case of compact manifold, in general, a complete manifold doesn't admit any pure point spectrum. For example, there are no \(L^2\)-eigenvalues on \(\mathbb{R}^n\text{.}\) That is, for any \(\lambda\in\mathbb{R}\text{,}\) if \(\Delta f + \lambda f = 0\) and \(f \in L^2(\mathbb{R}^n)\text{,}\) then we have \(f\equiv 0\text{.}\)

The above well-known result was generalized by Escobar, who proved that if \(M\) has a rotational symmetric metric, then there is no \(L^2\)-eigenvalue.

Let \(\Delta\) be the Laplacian on a complete non-compact manifold \(M\text{.}\) By the argument in the previous section, \(\Delta\) naturally extends to a self-adjoint densely defined operator, which we still denote as \(\Delta\) for the sake of simplicity.

It is well-known that there is a spectrum measure \(E\) such that

\begin{equation*} -\Delta = \int_0^{\infty} \lambda dE \end{equation*}

The heat kernel is defined as

\begin{equation*} e^{\Delta t}f(x) = \int H(x,y,t)f(y)dy \end{equation*}

and the Green's function is defined as

\begin{equation*} G(x,y) = \int_0^\infty H(x,y,t)dt \end{equation*}

The pure-point spectrum of \(\Delta\) are these \(\lambda\in\mathbb{R}\) such that

  • \ding{172}.

    There exists an \(L^2\) function \(f\neq 0\) such that

    \begin{equation*} \Delta f + \lambda f = 0 \end{equation*}
  • \ding{173}.

    The multiplicity of \(\lambda\) is finite.

  • \ding{174}.

    In a neighborhood of \(\lambda_1\) it is the only spectrum point.

We define

\begin{equation*} \rho(\Delta) = \{y\in\mathbb{R}|(\Delta-y)^{-1}\mbox{ is a bounded operator} \} \end{equation*}

and we define \(\sigma(\Delta) = \mathbb{R} - \rho(\Delta)\) to be the spectrum of \(\Delta\text{.}\) From the above discussion, \(\sigma(\Delta)\) decomposes as the union of pure point spectrum, and the so-called essential spectrum, which is, by definition, the complement of the pure-point spectrum.

The set of the essential spectrum is denoted as \(\sigma_{ess}(\Delta)\text{.}\) Using the above definition, \(\lambda\in\sigma_{ess}(\Delta)\text{,}\) if either

  • \ding{172}.

    \(\lambda\) is an eigenvalue of infinite multiplicity, or

  • \ding{173}.

    \(\lambda\) is the limiting point of \(\sigma(\Delta)\text{.}\)

The following theorems in functional analysis are well-known. For reference, see Donnelly.

Theorem 6.2.7.

A necessary and sufficient condition for the interval \((-\infty, \lambda)\) to intersect the essential spectrum of an self-adjoint densely defined operator \(A\) is that, for all \(\varepsilon > 0\text{,}\) there exists an infinite dimensional subspace \(G_\varepsilon \subset Dom(A)\text{,}\) for which

\begin{equation*} (Af - \lambda f - \varepsilon f, f) \lt 0 \end{equation*}
Theorem 6.2.8.

A necessary and sufficient condition for the interval \((\lambda-a,\lambda+a)\) to intersect the essential spectrum of \(A\) is that there exists an infinite dimensional subspace \(G\subset Dom(A)\) for which \(\|(A-\lambda I)f\|\leqslant a \|f\|\) for all \(f\in G\text{.}\)

Using the above result, we give the following variational characterization of the lower bound of spectrum and the lower bound of essential spectrum.

Theorem 6.2.9.

Using the above notations, define

\begin{equation*} \lambda_0 = \inf_{f\in C_0^\infty(M)}\frac{\int_M |\nabla f|^2}{\int_M f^2} \end{equation*}

and

\begin{equation*} \lambda_{ess} = \sup_K \inf_{f\in C_0^\infty(M\backslash K)}\frac{\int_M |\nabla f|^2}{\int_M f^2} \end{equation*}

where \(K\) is a compact set running through an exhaustion of the manifold. Then \(\lambda_0\) and \(\lambda_{ess}\) are the least lower bound of \(\sigma(\Delta)\) and \(\sigma_{ess}(\Delta)\text{,}\) respectively.

Corollary 6.2.10.

If \(\lambda_0 \lt \lambda_{ess}\text{,}\) then \(\lambda_0\) is an eigenvalue of \(M\) with finite dimensional eigenspace.

In this case, \(\lambda_0\) is called the ground state.

In the following, we give a non-trivial application of the above principle.

Theorem 6.2.11. Lin-Lu.

Let \(M\) be a complex complete surface embedded in \(\mathbb{R}^3\text{.}\) Assume that \(M\) is not totally geodesic, but asymptotically flat in the sense that the second fundamental form goes to zero at infinity. Define

\begin{equation*} \Omega = \{y\in\mathbb{R}^3 | d(y,M) \leqslant a \} \end{equation*}

for a small positive number \(a > 0\text{.}\) Then \(\Omega\) is a \(3\)-d manifold with boundary. The Dirichlet Laplacian of \(\Omega\) has a ground state.

Sketch of the proof: Since \(M\) is asymptotically flat, at infinity

\begin{equation*} \Omega \approx M \times [-a, a] \end{equation*}

As a result

\begin{equation*} \lambda_{ess} = \frac{\pi^2}{4a^2} \end{equation*}

Thus the main difficulty in the proof of the above theorem is to prove

\begin{equation*} \lambda_0 \lt \frac{\pi^2}{4a^2} \end{equation*}

which can be obtained by careful analysis of the Gauss and the mean curvatures.

Remark 6.2.12.

Exner et al. proved that under the condition

\begin{equation*} \int K \leqslant 0, \int |K| \lt \infty \end{equation*}

and \(M\) being asymptotically flat, the ground state exists. Thus we make the following conjecture to give the complete picture.

Conjecture 6.2.13.

Let \(M\) be a complete, no-totally geodesic, and asymptotically embedded surface in \(\mathbb{R}^3\text{.}\) Let \(\Omega\) be defined as before. Let \(K\) be the Gauss curvature. If

\begin{equation*} \int_M |K| \lt +\infty \end{equation*}

then the ground state exists.

The difficulty of the above conjecture is that even the surface is asymptotically flat, we still don't known the long-range behavior of the surface.

Subsection 6.2.2 On the theorem of Sturm

Let \(M\)be a complete Riemannian manifold. We say that the volume \((M,g)\) grows uniformly sub-exponentially, if for any \(\varepsilon > 0\text{,}\) there is a constant \(C \lt \infty\) such that for all \(r > 0\) and all \(x\in M\text{,}\) we have

\begin{equation*} v(B_r(x))\leqslant Ce^{\varepsilon r}v(B_1(X)) \end{equation*}

Theorem 6.2.14. Sturm.

If the volume of \((M,g)\) grows uniformly and sub-exponentially, then the spectrum \(\sigma(\Delta_p)\) of \(\Delta_p\) acting on \(L^p(M)\) is independent of \(p\in[1, \infty)\text{.}\) In particular, it is a subset of the real line.

One feature of the concept ‚Äúuniformly and sub-exponentially‚Äù is that it is self-dual. Take the following example: A hyperbolic space is not ``uniformly and sub-exponentially". On the other side, let \(\Gamma\) be a discrete group acting on the hyperbolic space \(H\text{,}\) such that \(\Gamma\backslash H\) has finite volume. Since the infinity of \(\Gamma\backslash H\) are cusps, it is still not ‚Äúuniformly and sub-exponentially‚Äù.

A manifold with non-negative Ricci curvature satisfies the assumption that the volume grows ‚Äúuniformly and sub-exponentially‚Äù. However, for such a manifold, its volume is infinite. It doesn't has the finite volume counterpart.

The proof of Sturm's theorem depends on the heat kernel estimates. We begin with the following

Lemma 6.2.15.

If the volume of \((M,g)\) grows uniformly and sub-exponentially, then for any \(\varepsilon > 0\)

\begin{equation*} \sup_{x\in M}\int_M e^{-\varepsilon d(x,y)}(v(B_1(x))^{-\frac{1}{2}}v(B_1(y))^{-\frac{1}{2}} dv(y) \lt \infty \end{equation*}

Proof.

We take \(r = d(x,y)\text{.}\) Then since

\begin{equation*} B_1(y) \subset B_{r+1}(x) \end{equation*}

we must have

\begin{equation*} v(B_1(y))^{-\frac{1}{2}} \geqslant v(B_{r+1}(x))^{-\frac{1}{2}} \geqslant Ce^{-\frac{1}{2}(r+1)}v(B_1(x))^{-\frac{1}{2}} \end{equation*}

for any \(x\text{.}\) Thus the integration in the lemma is less than

\begin{equation*} C\int_M e^{-\varepsilon r}e^{\frac{1}{2}\varepsilon(r+1)} v(B_1(x))^{-1} dy \end{equation*}

We let \(f(r) = v(\partial B_r(x))\) and \(F(r) = \int_0^r f(t)dt\text{.}\) Then up to a constant, the above expression is less than

\begin{equation*} (v(B_1(x)))^{-1} \int_0^\infty e^{-\frac{1}{2}\varepsilon r} f(r)dr \end{equation*}

By the volume growth assumption \(f(r) \leqslant Cr^{n-1} v B_1(x)\text{,}\) the lemma follows.

In fact, the assertion is true if

  • \ding{172}.

    The Ricci curvature of \(M\) has a lower bound;

  • \ding{173}.
    \begin{equation*} \sup_{x\in M} \int_M e^{-\varepsilon d(x,y)}(v(B_1(x))^{-\frac{1}{2}} v(B_1(y))^{-\frac{1}{2}} dv(y) \lt \infty \end{equation*}

The hard part is to prove \(\sigma(\Delta_p) \subset \sigma(\Delta_2)\) for all \(p\in[1,\infty]\text{.}\) If this is done, then it is easy to prove

\begin{equation*} \sigma(\Delta_2) \subset \sigma(\Delta_p) \end{equation*}

as follows:

Let \(\xi\in\rho(\Delta_p)\text{.}\) Then \((\Delta_p - \xi)^{-1}\) is a bounded operator on \(L^p(M)\text{.}\) Let \(\frac{1}{p} + \frac{1}{q} = 1\text{.}\) By dualization \((\Delta_q - \zeta)^{-1}\) is bounded in \(L^q(M)\text{.}\) By the interpolation theorem, \((\Delta_2 - \zeta)^{-1}\) is bounded and this \(\zeta\in\rho(\Delta_2)\text{.}\)

In order to prove \(\sigma(\Delta_p)\subset\sigma(\Delta_2)\text{,}\) or \(\rho(\Delta_2)\subset\rho(\Delta_p)\text{,}\) we need some estimates. Let \(\zeta\in\rho(\Delta_2)\text{.}\) Then

\begin{equation*} (\Delta_2 - \zeta)^{-1} \end{equation*}

is bounded from \(L^2 \rightarrow L^2\text{.}\) In order to prove that the operator is bounded on \(L^p\text{,}\) we need to prove that it has a kernel \(g(x,y)\) such that

\begin{equation*} (\Delta_2 - \zeta)^{-n} f = \int g(x,y)f(y)dy \end{equation*}

Lemma 6.2.16.

If \(g(x,y)\) satisfies

\begin{equation*} \sup_{x\in M}\int_M |g(x,y)|dy \leqslant C \end{equation*}

Then \((\Delta_2 - \zeta)^{-n}\) is a bounded operator on \(L^p(M)\text{.}\)

Proof.

This is essentially H√∂lder inequality:

\begin{align*} \nonumber \amp \amp \int_M \left(\int_M g(x,y)f(y)dy\right)^p dx\\ \nonumber \amp \leqslant \amp \int_M \left(\int_M g^{\frac{1}{q}}g^{\frac{1}{p}}fdy\right)^p dx\\ \nonumber \amp \leqslant \amp \int_M\left(\int_M g\right)^{\frac{1}{q}} \cdot \int gf^p dx\\ \nonumber \amp \leqslant \amp C^{\frac{1}{q}} \int_M\int_M g(x,y)f^p(y)dydx\\ \nonumber \amp \leqslant \amp C^{1+\frac{1}{q}}\int_M f^p(y)dy \end{align*}

If we assume that \(\sigma(\Delta_p)\) is a no-where dense set in \(\mathbb{C}\text{,}\) then we have

Lemma 6.2.17.

If \((\Delta_2 - \zeta)^{-n}\) is bounded, so is \((\Delta_2 - \zeta)^{-1}\text{.}\)

Proof.

For any \(\varepsilon\text{,}\) let \(\zeta '\in\rho(\Delta_\rho)\) and \(|\zeta - \zeta '| \lt \varepsilon\text{.}\) Then from

\begin{equation*} \|(\Delta_2 - \zeta)^{-n}\| \leqslant C \end{equation*}

we get

\begin{equation*} \|(\Delta_2 - \zeta ')^{-n}\| \leqslant C+1 \end{equation*}

provided that \(\varepsilon\) is small enough. Let \(dist(\zeta ', \sigma(\Delta_p))\) be the distance to the spectrum of \(\Delta_p\text{,}\) then we have

\begin{equation*} C+1 \geqslant \lim_{m\rightarrow \infty} \|(\Delta_2 - \zeta)^{-nm}\|^{\frac{1}{m}} \geqslant dist(\zeta ', \sigma(\Delta_p))^{-n} \end{equation*}

Thus

\begin{equation*} dist(\zeta ', \sigma(\Delta_p)) \geqslant \frac{1}{(C+1)^{\frac{1}{n}}} \end{equation*}

Since \(\zeta '\) is arbitrary, we have \(dist(\zeta, \sigma(\Delta_p)) > \delta > 0\text{.}\)
