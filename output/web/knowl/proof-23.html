The second application of the heat kernel estimate is the following resolvent estimate.

Theorem 3.1.6.

For any \(\beta > 0\) there is \(n \in N, \alpha \lt 0 , C \subset \infty\) such that the integral kernel \(g _{\alpha} ^{(\frac{n}{2})}(x, y)\) of \((\Delta - \alpha) ^{-\frac{n}{2}}\) satisfies.

\begin{equation*} g _{\alpha} ^{(\frac{n}{2})}(x, y) \leq C \phi (x) ^2 e ^{-\beta d(x, y)} \end{equation*}

where \(\phi (x) = (v (B_1 (x)) ^{-\frac{1}{2}}\text{.}\)

Proof.

First, using the heat kernel \(H (x, t)\) we can write

\begin{equation*} g _{\alpha} ^{(\frac{n}{2})}(x, y) = C _2 \int ^\infty_0 H (x, y, t) t ^{ \frac{n}{2}-1 } e ^{\alpha t} \, dt\text{.} \end{equation*}

By the volume comparison theorem, we have

\begin{align*} v ^{ \frac{n}{2}} (B_{\sqrt{t}}(x)) v ^{ \frac{-1}{2} }(B_{\sqrt{t}}(y))\amp \leq \amp C_2 \phi (x)^2 \sup (1, t ^{-\frac{n}{2}})\\ \amp \times \amp e ^{\beta _{2} d (x, y)}\text{.} \end{align*}

By an element calculation, for any \(\beta _3 > 0\text{,}\) we have

\begin{equation*} \exp \left( - \frac{d^2 (x, y)}{C_1 4t} \right) \leq \exp ( \beta_3 d (x, y) ) \exp (C_1 \beta ^2_3 t )\text{.} \end{equation*}

Thus for any \(\beta > 0\text{,}\) there exists \(\alpha \lt 0 , C_3 \lt \infty\) such that

\begin{equation*} H (x, y, t) \leq C_3 \phi ^2 (x) e ^{-\beta _{2} d (x, y)} \sup \{ t ^{ -\frac{n}{2} } , 1 \} t ^{ - (\alpha + 1 ) t} \end{equation*}

which implies

\begin{equation*} g _{\alpha} ^{(\frac{n}{2})} (x, y) \leq C_t \phi ^2 (x) e ^{-\beta d (x, y)}\text{.} \end{equation*}

Cheeger-Yau's heat kernel comparison theorem.

Theorem 3.1.7.

Let \(M\) be a complete Riemannian manifold such that Ric\((M) \geq 0\text{.}\) Fixing \(x \in M , r_0 > 0\text{.}\) The heat kernel \(H (x, y, t)\) in \(B(x, r_0)\) and the heat kernel \(\varepsilon (r (x, y) , t)\) in \(V (k, r_0)\) satisfies the following

\begin{equation*} \varepsilon (r (x, y) , t) \leq H (x, y, t)\text{.} \end{equation*}

(For both Dirichlet and Neumann conditions).

Proof.

Using the property of the heat kernel, we have

\begin{align*} \lefteqn{H(x, y, t) - \varepsilon (x, y, t)}\\ \amp \amp = \int ^t _0 \int _{B (x_1 r_0)} \frac{d}{ds} ( \varepsilon (x, z, t - s) H (z, y, s) ) \, d z ds\\ \amp \amp = - \int ^t _0 \int _{B (x_1 r_0) }\left( \frac{d}{ds}\varepsilon (r (x, z), t-s)\right) H(z, y, s) \, d z ds\\ \amp \amp + \int^t_0 \int _{B (x_1 r_0) }\varepsilon (r (x, z), t-s)\frac{d}{ds} H(z, y, s) \, d z ds\\ \amp \amp = - \int^t_0 \int _{B (x_1 r_0)} \tilde{\Delta} \varepsilon (r (x, z), t - s) H(z, y, s) \, d z ds\\ \amp \amp + \int^t_0 \int _{B (x_1 r_0)}\varepsilon (r (x, z), t - s) \Delta H(z, y, s) \, d z ds\text{.} \end{align*}

Using the Green's formula, under either the Dirichlet or Neumann boundary condition, we have

\begin{align*} \lefteqn{\int _{B (x_1 r_0)}\varepsilon (r (x, z), t - s) \Delta H(z, y, s) \, d z}\\ \amp \amp = \int _{B (x_1 r_0)}\Delta \varepsilon (r (x, z), t - s) H(z, y, s) \, d z\text{.} \end{align*}

Since \(H(\xi, y, s) > 0\text{,}\) we just need to prove that

\begin{equation*} \tilde{\Delta} \varepsilon (r (x, z), t - s)\leq \Delta \varepsilon (r (x, z), t - s)\text{.} \end{equation*}

This essentially follows from the Laplacian comparison theorem: Let \(x = (r, \zeta ) , \zeta \in S^{n-1}\text{.}\) Then

\begin{align*} \tilde{\Delta } \amp = \amp \frac{\partial ^2}{\partial r^2} + m (r) \frac{\partial }{\partial r} , m (r) \frac{\partial }{\partial r} \log { det} \sqrt{\tilde{g}}\\ \Delta \amp = \amp \frac{\partial ^2}{\partial r^2} + m (r,\zeta) \frac{\partial }{\partial r} , m (r,\zeta) \frac{\partial }{\partial r} \log { det} \sqrt{{g}}\text{.} \end{align*}

Since Ric\((M) \geq (n-1)_k\text{,}\) using the volume comparison theorem, we have

\begin{equation*} m (r, \zeta ) \leq m (r)\text{.} \end{equation*}

Since

\begin{equation*} \frac{\partial \varepsilon}{\partial r} \lt 0 \end{equation*}

we have

\begin{equation*} \tilde{\Delta} \varepsilon (r, t-s) \leq \Delta \varepsilon (r, t - s)\text{.} \end{equation*}

In the above proof, we didn't take the cut-locus into account. Using some kind of limiting process, we can overcome the difficulty.
