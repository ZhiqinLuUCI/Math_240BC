Definition 1.2.1.

Let \(X,Y\) be vector fields.

  1. If \(f\in \mathcal C^\infty(M)\text{,}\) then \(\nabla_X f=Xf\text{;}\)

  2. \(\nabla _XY\) is defined by the Levi-Civita connection;

  3. If \(\omega\) is a one-form, we use

    \begin{equation*} (\nabla_X\omega)Y=X(\omega(Y))-\omega(\nabla_XY) \end{equation*}

    to define the one form \(\nabla_X\omega\text{;}\)

  4. In general, let \(\{e_1,\cdots,e_n\}\) be a local frame of \(M\) and let \(\omega_1,\cdots,\omega_n\) be the dual frame. Let

    \begin{equation*} T=a_{i_1\cdots i_p j_1\cdots j_q}e_{i_1}\otimes\cdots \otimes e_{i_p}\otimes\omega_{j_1}\otimes\cdots\otimes \omega_{j_q}\text{.} \end{equation*}

    We define

    \begin{align*} \amp \nabla_X T=X(a_{i_1\cdots i_pj_1\cdots j_q})e_{i_1}\otimes\cdots\otimes \omega_{j_q}\\ \amp +\sum_{s=1}^p a_{i_1\cdots i_pj_1\cdots j_q}e_{i_1}\otimes\cdots\otimes \nabla_X e_{i_s}\otimes \cdots\otimes \omega_{j_q}\\ \ \amp +\sum_{t=1}^q a_{i_1\cdots i_pj_1\cdots j_q}e_{i_1}\otimes\cdots\otimes \nabla_X \omega_{j_t}\otimes \cdots\otimes \omega_{j_q}\text{.} \end{align*}
