Proof

This follows from the strong maximum principle. By the above lemma, we have

\begin{equation*} \frac{\partial u}{\partial n} \leqslant 0 \end{equation*}

If \(\frac{\partial u}{\partial n} = 0\text{,}\) then since the function \(u\) vanishes on the boundary, we have \(\nabla u = 0\) at the point. Using the Taylor's expansion for the boundary point, we get \(p^2 \geqslant 0\) and \(\Delta p^2 = 0\text{.}\) Since \(p^2\) is harmonic, by the strong maximum principle, \(\partial p^2/\partial n \lt 0\) unless \(p^2 =0\text{.}\) But if \(p^2 \equiv 0\) then \(u \equiv 0\text{.}\) This completes the proof.
