The key point is that

\begin{equation*} Q (\rho ^2 f, f) = Q \langle \rho^2 f , \Delta f \rangle = 0\text{.} \end{equation*}

Thus using the same method as before, \(f\) is a constant. If \(M\) is a compact manifold with no boundary, we still use \(\Delta\) to denote the Dirichlet extension of the Laplace operator. By the elliptic regularity, the spectrum of \(\Delta\) are discrete. That is, there is a sequence

\begin{equation*} 0 = \lambda _0 \lt \lambda _1 \leq \lambda _2 \leq \ldots \end{equation*}

such that for any \(\lambda_i\text{,}\) there \(\exists f_i , f _i \in L^2\)

\begin{equation*} \Delta f _i = - \lambda _i f_i\text{.} \end{equation*}

We have similar results for manifolds with boundary conditions. To be more precise, the Laplacians acting on functions with the following boundary conditions.

  1. (A).

    Dirichlet boundary condition: \(\Delta\) acting on \(f\) vanishing on the boundary.

  2. (B).

    Neumann boundary condition: \(\Delta\) acting on functions such that \(\frac{\partial f}{\partial n} = 0\text{.}\)

For eigenvalues, we have the following minimax principle. Assume that \(M\) is a closed manifold, then

\begin{equation*} \lambda _1 = \inf_{\int f = 0 } \frac{\int |\nabla f|^2}{\int f^2}\text{.} \end{equation*}

To prove the above result, we let \(\phi\) be any smooth function such that

\begin{equation*} \int_M \phi = 0\text{.} \end{equation*}

Then by the definition of \(\lambda _1\text{,}\) we have

\begin{equation*} \frac{\int | \nabla (f + \varepsilon \phi )|^2}{\int (f + \varepsilon \phi )^2} \geq \lambda _1\text{.} \end{equation*}

However, if we take the first order term, we get

\begin{equation*} \int (\delta f + \lambda _1 f ) \phi = 0\text{.} \end{equation*}

Note that

\begin{equation*} \int (\Delta f + \lambda _ 1 f ) = 0\text{.} \end{equation*}

Then

\begin{equation*} \Delta f + \lambda _1 f \equiv 0 \end{equation*}

and \(\lambda _ 1\) is the first eigenvalue.

By elliptic regularity, \(\lambda _ 1 > 0\text{.}\) Thus we have the following Poincar√©-inequality: there exists a constant \(C\text{,}\) such that

\begin{equation*} \int | \nabla f |^2 \geq C \int f^2 \end{equation*}

for any function with

\begin{equation*} \int _M f = 0\text{.} \end{equation*}
