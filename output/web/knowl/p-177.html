If we normalize \(f_\alpha\) such that \(\int_M f_\alpha^2=1\text{,}\) then the sequence \(\{f_\alpha\}\) is bounded in \(H_1(M)\text{.}\) Therefore, there is an \(f\in H_1(M)\) such that

\begin{equation*} f_\alpha\to f \end{equation*}

in the weak sense. In particular, for any \(\phi\) smooth, we have

\begin{equation*} \int_M\nabla f_\alpha\nabla\phi\to\int_M\nabla f\nabla\phi\text{.} \end{equation*}
