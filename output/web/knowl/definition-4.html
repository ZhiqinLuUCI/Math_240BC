Definition 2.2.2.

Let \(H\) be a Hilbert space, Given a densely defined linear operator \(A\) on \(H\text{,}\) its adjoint \(A^*\) is defined as follows:

  1. The domain \({\mathcal Dom}(A^*)\) of \(A^*\) consists of vectors \(x\) in \(H\) such that

    \begin{equation*} y\mapsto \langle x, Ay\rangle \end{equation*}

    is a bounded linear functional, where \(y\in{\mathcal Dom}(A)\text{;}\)

  2. By the Riesz Representation Theorem for linear functionals, if \(x\) is in the domain of \(A^*\text{,}\) there is a unique vector \(z\) in \(H\) such that

    \begin{equation*} \langle x,Ay\rangle=\langle z,y\rangle \end{equation*}

    for any \(y\in {\mathcal Dom}(A)\text{.}\) This vector \(z\) is defined to be \(A^*x\text{.}\) It can be shown that the dependence of \(z\) on \(x\) is linear.

If \(A^*=A\) (which implies that \({\mathcal Dom}\, (A^*)={\mathcal Dom}\,(A)\)), then \(A\) is called self-adjoint.
