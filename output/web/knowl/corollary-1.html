Corollary 1.3.4. Ricci identity.

Using the above notations, we have

\begin{align*} \amp f_{ij}=f_{ji};\\ \amp f_{ijk}-f_{ikj}=f_r R_{rijk}\text{.} \end{align*}

In particular, we have the following useful Ricci identity:

\begin{equation*} \sum_i f_{iji}-\sum_i f_{iij}=(Ric)_{rj}f_r\text{.} \end{equation*}
