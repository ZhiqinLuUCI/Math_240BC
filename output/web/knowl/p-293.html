We write

\begin{equation*} \frac{\partial \omega _j}{\partial r} =\sum_{j>1} a_{ij} \omega _j \end{equation*}

for \(i>1\) for some matrix valued function \((a_{ij})\text{.}\) The Uhlenbeck's trick is that, by an orthogonal change of the co-frame, we can make \((a_{ij})\) a symmetric matrix.
