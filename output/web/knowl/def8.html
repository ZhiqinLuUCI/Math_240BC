Definition 9.2.1.

Let \(\Omega\) be a bounded domain in \(\R^n\) with smooth boundary. Let \(y\in\Omega\text{.}\) Define

\begin{equation*} \chi(y)=\{p\in\R^n\mid u(x)\leq u(y)+p(x-y),\,\forall x\in \Omega\}\text{.} \end{equation*}

Then \(\chi\) defines a map from \(\Omega\) to the set of subsets of \(\R^n\text{.}\) We call it the normal map.
