By assumption, there is a sequence \(\{u_m\}\subset \mathcal C^2(\bar\Omega)\) such that \(u_m\to u\) in \(W_{{ loc}}^{2,n}(\Omega)\text{.}\) That is, for any \(\Omega'\subset\subset\Omega\text{,}\) if \(m\to\infty, u_m\to u\) in \(W^{2,n}(\Omega')\text{.}\) For any \(\eps>0\text{,}\) let

\begin{equation*} \Omega_\eps=\{x\in\Omega\mid { dist}(x,\pa\Omega)>\eps\}\text{.} \end{equation*}

Let

\begin{align*} \amp v_\eps=u-\sup_{\Omega\backslash\Omega_\eps} u-\eps\\ \amp v_{m,\eps}=u_m-\sup_{\Omega\backslash\Omega_\eps} u_m-\eps \end{align*}

By Theorem 9.2.11, we have

\begin{equation*} \sup_\Omega v_{m,\eps}\leq\frac{d}{\sqrt[n]{w_n}}\left(\int_{\Gamma_{v_{m,\eps}}^+}\det(-D^2 u_m) dx\right)^{1/n}\text{.} \end{equation*}

Since \(\Gamma^+_{m,\eps}\subset\Omega_\eps\text{,}\) we write

\begin{align*} \amp \int_{\Gamma_{v_{m,\eps}}^+}\det(-D^2 u_m) dx\\ \amp =\int_{\Gamma_{v_{m,\eps}}^+}(\det(-D^2 u_m) -\det(-D^2 u))dx+\int_{\Gamma_{v_{m,\eps}}^+}\det(-D^2 u) dx\\ \amp =I_{1,m}+I_{2,m} \end{align*}

Obviously, we have \(I_{1,m}\to 0\) as \(m\to\infty\text{.}\) On the other hand, we have

\begin{equation*} \varlimsup_{m\to\infty}\Gamma_{v_{m,\eps}}^+\subset\Gamma_{v_\eps}^+\text{.} \end{equation*}

In fact, if

\begin{equation*} x_0\in\varlimsup_{m\to\infty}\Gamma_{v_{m,\eps}}^+\text{.} \end{equation*}

Then there is a subsequence \(m_k\) such that \(x_0\in\Gamma_{v_{m_k,\eps}}\text{.}\) By Lemma 9.2.7,

\begin{equation*} |p_k|\leq\frac{4\sup |u_m|}{{ dist}(x,\pa\Omega)}\leq\frac{4}{\eps}\sup |u_m| \end{equation*}

for any \(p_k\in\chi_{v_{m_k,\eps}}(x_0)\text{.}\)

Assuming \(p_k\to p_0\text{.}\) Then \(p_0\in\chi_{v_\eps}(x_0)\text{,}\) hence \(x_0\in\Gamma^+_{v_\eps}\text{.}\) Thus

\begin{equation*} \varlimsup_{m\to\infty} I_{2,m}\leq\int_{\Gamma^+_{v_\eps}}\det(-D^2 u) dx\text{.} \end{equation*}

The theorem is proved.
