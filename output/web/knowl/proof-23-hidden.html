Let \(f = H (x, y, t)\text{.}\) Then

\begin{equation*} \int f ^2 = \int f ^{\frac{2n}{n+2}+ \frac{4}{n+2}}\leq \left( \int f^{\frac{2n}{n-2}}\right) ^{\frac{n+2}{2n}}\left. \left( \int f \right) ^{\frac{4}{n+2}}\right)\text{.} \end{equation*}

Thus we have

\begin{equation*} \frac{\partial }{\partial t}\int _M H(x, y, t)^2 \, dy \leq - C \left( \int_M |H (x, y, t) ^2 | dy \right) ^{\frac{2+n}{n}}\text{.} \end{equation*}

Since

\begin{equation*} \lim_{t \rightarrow 0} H (x, x, t) = \infty\text{.} \end{equation*}

Thus

\begin{equation*} H (x, x, 2t) \leq \left( \frac{4}{n} Ct \right) ^{-\frac{n}{2}}\text{.} \end{equation*}

Using the same method as before, we have

\begin{equation*} \lambda _k \geq C \left( \frac{k}{{ vol} (M)}\right) ^{\frac{2}{n}}\text{.} \end{equation*}
