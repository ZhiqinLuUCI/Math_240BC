If \({ Ric} \geq 0\) then \(k = 0\text{.}\) We have

\begin{equation*} \Delta \phi \geq - (2 - \varepsilon ) \frac{\nabla \phi \nabla u}{u} + \varepsilon \phi^3\text{.} \end{equation*}

For \(\varepsilon = \frac{2}{n-1}\text{.}\) Thus using the generalized maximal principal, \(\phi \equiv 0\text{.}\)
