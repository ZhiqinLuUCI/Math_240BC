Using the cut-off function, we may assume that \({ supp}\, g\) is contained in a coordinate chart. Thus we may assume that the Riemannian metric is given under the global coordinates \((x_1, \ldots, x _n)\) as follows

\begin{equation*} d s^2 = \sum g _{ig} d x _i ds _j \end{equation*}

by definition

\begin{equation*} \int _M g = \int_M g \sqrt{{ det} (g _{ij} ) } \ dx_1, \ldots, dx _n\text{.} \end{equation*}
