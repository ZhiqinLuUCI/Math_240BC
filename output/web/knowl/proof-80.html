Proof

The proof of this theorem is quite long so we divide it into the following 5 steps.

Step 1. Using rescaling

\begin{equation*} x\mapsto\frac{x-y}{2R}\text{,} \end{equation*}

we assume that \(y=0\text{,}\) \(R=1/2\text{.}\) Let

\begin{equation*} \tilde u=u+\left|\left|\frac{f}{\lambda}\right|\right|_{L^n(B_1)}, \Gamma=\{x\in B_1\mid\tilde u(x)\geq 1\}\text{.} \end{equation*}

We want to prove that there exists \(C>0\text{,}\) \(0\lt \delta\lt 1\) such that whenever

\begin{equation} |\Gamma\cap K_\alpha|\geq\delta|K_\alpha|\text{,}\tag{9.3.4} \end{equation}

we have

\begin{equation} \inf_{K_{3\alpha}}\,\tilde u\geq C^{-1}\text{,}\tag{9.3.5} \end{equation}

where \(K_\alpha\) is the tube centered at \(0\) with side length \(2\alpha\text{,}\) \(\alpha=1/(6\sqrt n)\text{,}\) \(C,\delta\) only depends on \(n,\gamma\text{.}\)

For any \(\eps>0\text{.}\) Let

\begin{equation*} w=\log(\tilde u+\eps)^{-1}, g=\frac{f}{\tilde u+\eps}\text{.} \end{equation*}

Then we have

\begin{equation*} -a^{ij}D_{ij} w=\frac{1}{\tilde u+\eps}a^{ij} D_{ij} u-a^{ij}D_iwD_jw \leq -g-a^{ij} D_iw D_jw\text{.} \end{equation*}

Let \(\eta=(1-|x|^2)^\beta\text{.}\) Let \(v=\eta w\text{.}\) Then we have

\begin{align*} \amp Lv=\eta Lw+w L\eta-2a^{ij}D_i wD_j \eta\\ \amp \leq -\eta g-\eta a^{ij}D_i w D_j w-2a^{ij} D_iw D_j\eta+vL\eta/\eta\\ \amp \leq -\eta g+\frac 1\eta a^{ij} D_i\eta D_j\eta+\frac{v L\eta}{\eta}\text{.} \end{align*}

By (9.3.3), we know when

\begin{equation*} (2\beta-1)\lambda|x|^2\geq n\Lambda\text{,} \end{equation*}

we have \(L\eta\leq 0\text{.}\) Let \(\alpha\in(0,1)\text{.}\) If \(|x|>\alpha\text{,}\) we take

\begin{equation*} \beta-1\geq\frac{n\gamma}{2\alpha^2}\text{.} \end{equation*}

Then we have \(L\eta\leq 0\) for \(|x|\geq \alpha\text{.}\) Thus we have

\begin{equation*} Lv\leq|g|+4\beta^2\Lambda+\sup_{B_\alpha}\left(\frac{L\eta}{\eta}\right)\chi(B_\alpha) v\text{,} \end{equation*}

where \(\chi(B_\alpha)\) is the characteristic function of \(B_\alpha\text{.}\) Note that

\begin{equation*} \left|\left|\frac{g}{\lambda}\right|\right|_{L^n(B_1)}\leq 1\text{.} \end{equation*}

By the Aleksandrov maximum principle, we have

\begin{equation} \sup_{B_1} v\leq C[1+||v^+||_{L^n(B_\alpha)}]\text{,}\tag{9.3.6} \end{equation}

where \(C\) only depends on \(n,\gamma,\alpha\text{.}\)

In order to use the measure theory, we change ball into cube. We have

\begin{equation*} \sup_{B_1} v\leq C[1+||v^+||_{L^n(B_\alpha)}]\leq C(1+|K_\alpha^+|^{1/n}\sup_{B_1} v)\text{,} \end{equation*}

where

\begin{equation*} K_\alpha^+=\{x\in K_\alpha\mid v>0\}=\{x\in K_\alpha\mid\tilde u+\eps\lt 1\}\text{.} \end{equation*}

If

\begin{equation*} \frac{|K_\alpha^+|}{K_\alpha}\leq\theta\overset{\Delta}{=}\frac{1}{(2C)^n|K_\alpha|}=\frac{1}{(4C\alpha)^n}\text{.} \end{equation*}

Then we have

\begin{equation*} \sup_{B_1} v\leq 2C\text{,} \end{equation*}

or equivalently

\begin{equation*} \inf_{B_{1/2}}\,(\tilde u+\eps)\geq\frac 1C\text{.} \end{equation*}

Letting \(\eps\to 0\text{,}\) we get (9.3.5). On the other hand, if \(\frac{|K_\alpha^+|}{K_\alpha}>\theta\text{,}\) we let \(\delta=1-\theta\text{,}\) \(\alpha=\frac{1}{6\sqrt n}\text{.}\) Then we have \(K_{3\alpha}\subset B_{1/2}\text{.}\) If \(|\Gamma\cap K_\alpha|\geq\delta|K_\alpha|\text{,}\) then by Lemma 9.3.2, we have

\begin{equation*} |K_\alpha^+|=|K_\alpha\backslash (\Gamma\cap K_\alpha)|\leq\theta|K_\alpha| \end{equation*}

which contradicts to the assumption (9.3.4).

Step 2. For any positive integer \(m\text{,}\) if

\begin{equation*} |\Gamma\cap K_\alpha|\geq\delta^m|K_\alpha|\text{,} \end{equation*}

then

\begin{equation*} \inf_{K_\alpha}\tilde u\geq C^{-m}\text{,} \end{equation*}

where \(C\) is the constant in Step 1.

If \(m=1\text{,}\) the statement is proved by the above step. Assume that the claim is true for \(m\geq 1\text{.}\) Assume that

\begin{equation*} |\Gamma\cap K_\alpha|\geq\delta^{m+1}|K_\alpha|\text{.} \end{equation*}

Let \(\tilde K_0=K_\alpha\text{,}\)

\begin{equation*} \Gamma_\delta=\bigcup\{K_{3r}(x)\cap \tilde K_0\mid K_r(x)\subset\tilde K_0, |\Gamma\cap K_r(x)|\geq\delta|K_r(x)|\}\text{.} \end{equation*}

By Lemma 9.3.2, we have either

\begin{equation*} \Gamma_\delta=\tilde K_0, { or} |\Gamma\cap\tilde K_0|\leq\delta|\Gamma_\delta|\text{.} \end{equation*}

By the definition of \(\Gamma_\delta\) and Step 1, we have

\begin{equation*} \inf_{\Gamma_\delta}\tilde u\geq C^{-1}\text{.} \end{equation*}

If \(\Gamma_\delta=\tilde K_0=K_\alpha\text{,}\) then the above implies the claim. If \(|\Gamma\cap\tilde K_0|\leq\delta|\Gamma_\delta|\text{.}\) Let \(v=Cu\text{.}\) Then \(v\) satisfiws

\begin{equation*} -a^{ij}D_{ij} v\geq Cf\text{.} \end{equation*}

Let

\begin{equation*} \tilde v=v+\left|\left|\frac{Cf}{\lambda}\right|\right|_{L^n(B_1)}=C\tilde u\text{.} \end{equation*}

Let

\begin{equation*} \tilde\Gamma=\{x\in B_1\mid\tilde v\geq 1\}\text{.} \end{equation*}

Then \(\Gamma_\delta\subset\tilde\Gamma\) and we have

\begin{equation*} |\tilde\Gamma\cap\tilde K_0|\geq|\Gamma_\delta|\geq\frac{1}{\delta}|\Gamma\cap\tilde K_0| =\frac 1\delta|\Gamma\cap K_\alpha|\geq\delta^m|K_\alpha|=\delta^m|\tilde K_0|\text{.} \end{equation*}

By the inductive assumption, we have

\begin{equation*} \inf_{K_\alpha}\tilde v\geq C^{-m}\text{,} \end{equation*}

which is equivalent to

\begin{equation*} \inf_{K_\alpha}\tilde u\geq C^{-(m+1)}\text{.} \end{equation*}

Step 3. Let

\begin{equation*} \Gamma_t=\{x\in B_1\mid\tilde u(x)>t\}\text{.} \end{equation*}

Then there is a \(C>1, \mu>0\) such that

\begin{equation} |B_\alpha\cap \Gamma_t|\leq C|B_\alpha|\left(\frac{\inf_{B_\alpha}\tilde u}{t}\right)^\mu\text{,}\tag{9.3.7} \end{equation}

where \(C\) and \(\mu\) depend only on \(n,\gamma\text{.}\)

Let \(v=u/t\) and \(\tilde v=\tilde u/t\text{.}\)

\begin{equation*} \tilde\Gamma\overset{\Delta}{=}\{x\in B_1\mid\tilde v(x)>1\}=\Gamma_t\text{.} \end{equation*}

If \(|B_\alpha\cap\Gamma_t|=0\text{,}\) then (9.3.7) is obvious. Now assume that \(|B_\alpha\cap\Gamma_t|\neq 0\text{,}\) then there is a positive number \(m\) such that

\begin{equation*} \delta^m|K_\alpha|\leq|\tilde\Gamma\cap K_\alpha|\leq\delta^{m-1}|K_\alpha|\text{.} \end{equation*}

That it,

\begin{equation*} \log\frac{|\tilde\Gamma\cap K_\alpha|}{K_\alpha}\cdot(\log\delta)^{-1}\leq m\leq 1+\log\frac{|\tilde\Gamma\cap K_\alpha|}{K_\alpha}\cdot(\log\delta)^{-1}\text{.} \end{equation*}

By Step 2, we have

\begin{equation*} \inf_{K_\alpha}\tilde v\geq C^{-m}\geq C^{-1}\left[\frac{|\tilde\Gamma\cap K_\alpha|}{K_\alpha}\right]^{\frac{\log C}{\log \delta^{-1}}}\text{.} \end{equation*}

Let \(\mu=\log\delta^{-1}/\log C\text{.}\) Then we have

\begin{equation*} |\Gamma_t\cap K_\alpha|\leq (C\inf_{K_\alpha}\tilde v)^\mu|K_\alpha|\text{.} \end{equation*}

Step 4. There is a \(p>0\) such that

\begin{equation*} \left(\fint_{B_\alpha}u^p dx\right)^{1/p}\leq C\left( \inf_{B_\alpha} u+\left|\left|\frac f\lambda\right|\right|_{L^n(B_1)}\right)\text{.} \end{equation*}

We have

\begin{equation*} \int_{B_{\alpha}} u^p dx=p\int_0^\infty t^{p-1}|B_\alpha\cap\Gamma_t| dt =p\int_0^b t^{p-1}|B_\alpha\cap\Gamma_t| dt+p\int_b^\infty t^{p-1}|B_\alpha\cap\Gamma_t| dt\text{,} \end{equation*}

where \(b\) is a number to be determined. Therefore, we have

\begin{equation*} \int_{B_{\alpha}} u^p dx\leq p\int_0^b\int_0^b t^{p-1}|B_\alpha| dt+p\int_b^\infty Cm_0^\mu|B_\alpha| t^{p-\mu-1} dt\text{,} \end{equation*}

where \(m_0={\displaystyle \inf_{B_\alpha}}\,\tilde u\text{.}\) Let \(p=\mu/2\text{.}\) Then

\begin{equation*} \int_{B_\alpha} u^p dx\leq b^p|B_\alpha|+Cm_0^{2p}b^{-p}|B_\alpha|\text{.} \end{equation*}

Let \(b=C^{1/2p}m_0\text{.}\) Then we have

\begin{equation*} \int_{B_\alpha}u^p dx\leq 2C^{1/2}m_0^p\leq 2C^{1/2}\left(\inf_{B_\alpha} u+\left|\left|\frac f\lambda\right|\right|_{L^n(B_1)}\right)^p\text{.} \end{equation*}

Step 5. Using covering technique to prove

\begin{equation*} \left(\int_{B_{1/2}}u^p dx\right)^{1/p}\leq C\left(\inf_{B_{1/2}} \, u+\left|\left|\frac f\lambda\right|\right|_{L^n(B_1)}\right)\text{.} \end{equation*}

We note that \(u\in W^{2,n}(B_{\frac 12})\text{,}\) there is a \(x_0\in\bar{B_{1/2}}\) such that

\begin{equation*} u(x_0)=\inf_{B_{\frac 12}}\,u\text{.} \end{equation*}

Consider the integration

\begin{equation*} \int_{B_{\frac 12+\frac\alpha 4}}dy\cdot\int_{B_{\frac\alpha 4}(y)} u^p dx\geq|B_{\frac \alpha 4}| \int_{B_{\frac 12}}u^p(y) dy\text{.} \end{equation*}

Using the mean value theorem, there is a \(y\) such that

\begin{equation*} y_0\in {B_{\frac 12+\frac\alpha 4}} \end{equation*}

such that

\begin{equation*} \int_{B(y_0)} u^p dx\geq\int_{B_{\frac 12}}u^p dx\text{.} \end{equation*}

The theorem is proved.
