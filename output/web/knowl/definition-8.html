Definition 5.1.6.

Let \(M\) be a compact Riemannian manifold if \(\partial M \neq \emptyset\text{.}\)

\begin{equation*} h_D (M) = \inf \left\{ \left.\frac{{ vol} ( \partial \Omega )}{{ vol} ( \Omega)} \right| \Omega \subset \subset M\right\} \end{equation*}

if \(\partial M = \emptyset\text{.}\)

\begin{equation*} h_N = \inf \left\{ \left.\frac{{ vol}( H )}{\min ({ vol} (M_1), { vol} (M_2) )} \right| H \ \mbox{is a hypersurface} \right\}\text{.} \end{equation*}
