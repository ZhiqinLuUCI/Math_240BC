If we compare \(G\) to the operator \(A\) (they have the same eigenspaces), we shall see that \(G\) is also a compact operator because \(||G||\leq C||A||\) for some constant \(C\text{.}\) Furthermore, by the elliptic regularity, \(G\) maps smooth \(p\)-forms to smooth \(p\) forms. Using this fact, the Hodge decomposition is given by

\begin{equation*} \phi=\eta_0+d\delta G\phi+\delta dG\phi\text{.} \end{equation*}
