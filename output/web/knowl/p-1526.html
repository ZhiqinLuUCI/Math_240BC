By using the transformation

\begin{equation*} x\mapsto\frac{x-y}{2R}\text{,} \end{equation*}

we can assume that \(y=0\) and \(R=1/2\text{.}\) Taking

\begin{equation*} \eta(x)=(1-|x|^2)^\beta \end{equation*}

as the cut-off function, then we have

\begin{align} \begin{split} \amp D_i\eta=-2\beta x_i(1-|x|^2)^{\beta-1};\tag{9.3.3}\\ \amp D_{ij}\eta=-2\beta\delta_{ij}(1-|x|^2)^{\beta-1}+4\beta(\beta-1)x_ix_j(1-|x|^2)^{\beta-2};\notag\\ \amp L\eta=[2\beta\, \sum_{i=1}^na^{ii}(1-|x|^2)-4\beta(\beta-1)\sum_{i,j=1}^na^{ij}x_ix_j](1-|x|^2)^{\beta-2}. \end{split}\notag \end{align}
