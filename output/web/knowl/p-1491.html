If \(E\) is a measurable subset of \(\Gamma_u\text{,}\) then there is an open set \(G\supset E\backslash S\) such that on \(G\text{,}\) \(J(x)>0\text{.}\) On the other side, since \(E\backslash S\) is measurable, we can find open sets \(\{O_\ell\}\) such that \(E\backslash S\subset O_\ell\text{,}\) and \(|O_\ell\backslash(E\backslash S))|\to 0\) as \(\ell\to\infty\text{.}\) Thus we have

\begin{equation*} \int_{Du(E)}g(x(p))dp\leq\int_{Du(G\cap O_\ell)} g(x(p))dp\leq\int_{G\cap O_\ell} g(x)J(x) dx\text{.} \end{equation*}
