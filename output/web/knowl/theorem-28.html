Theorem 4.1.1.

Let \(M\) be a complete Riemannian, then there is a heat kernel \(H (x, y, t) \in C^\infty (M \times M \times \mathbb{R} ^+ )\) such that

\begin{equation*} (e ^{- \Delta t} f) (x) = \int_M H (x, y, t )f (y) \end{equation*}

\(\forall f \in L^2 (M)\) such that

  1. { (1)}.

    \(\displaystyle H(x, y, t) = H (y, x, t)\)

  2. { (2)}.

    \(\displaystyle \lim_{t \rightarrow 0^{+} } H (x, y, t) = \delta _x (y)\)

  3. { (3)}.

    \(\displaystyle \left( \Delta - \frac{\partial }{\partial t} \right) H = 0\)

  4. { (4)}.

    \(H(x, y, t) = \int H (x, z, t - s) H (z, y, s) \, dz\text{.}\)
