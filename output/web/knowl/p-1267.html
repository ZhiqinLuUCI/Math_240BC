We first define the Sobolev \(s\)-norms for \(\rho_\alpha S\) on each \(U_\alpha\text{.}\) Since \(U_\alpha\) is a coordinate patch, we can assume, without loss of generality, that \(U_\alpha\) is an open set of a torus \(T=T_{n}=\R^{n}/\Z^{n}\text{.}\) Thus the section \(\rho_\alpha S\) can be extended as a smooth \(\C^r\)-valued function on \(T\text{.}\) Let

\begin{equation*} \rho_\alpha S=\sum_{\xi\in\mathbb Z^{n}}S_\xi e^{i\langle\xi,x\rangle} \end{equation*}

be the Fourier expansion of \(S_\alpha\text{.}\) We define

\begin{equation*} ||\rho_\alpha S||_s^2=\sum_{\xi\in\mathbb Z^{n}}(1+||\xi||^2)^s |(\rho_\alpha S)_\xi|^2\text{.} \end{equation*}
