Corollary 3.2.6. Harnack **.

Let \(M\) be \(n\)-dimensional Riemannian manifold, \({ Ric} (M) \geq - (n-1) k\text{.}\) If \(u\) is a positive harmonic function in \(Ba\text{.}\) Then

\begin{equation*} \sup_{Ba/2} u \leq C (n, a, k) \inf_{B a/2} u \end{equation*}

where \(C (n, a, k)\) are constants depending only on \(n, a, k\text{.}\)
