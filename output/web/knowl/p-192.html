The set of the essential spectrum is denoted as \(\sigma_{ess}(\Delta)\text{.}\) Using the above definition, \(\lambda\in\sigma_{ess}(\Delta)\text{,}\) if either

  1. \ding{172}.

    \(\lambda\) is an eigenvalue of infinite multiplicity, or

  2. \ding{173}.

    \(\lambda\) is the limiting point of \(\sigma(\Delta)\text{.}\)
