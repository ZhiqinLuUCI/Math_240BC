To construct the required \(u_0\text{,}\) we first pick up any smooth function \(w>0\) on \(\Omega\) with \(w|_{\partial\Omega} = 0\text{,}\) \(\nabla w|_{\partial \Omega} \neq 0\text{.}\) Let

\begin{equation*} u_0 = w e^{-C\sum x_j^2} \end{equation*}

for a constant \(C > 0\) sufficiently large. By the above lemma, the function is log-concave near the boundary. Away from the boundary, since \(w > \delta > 0\) for some constant \(\delta > 0\text{,}\) we can choose \(C\) large enough so that \(u_0\) is log-concave.
