To prove the second assertion of the theorem, we note that the biggest possible eigenvalue of \(A\) is \(1\text{.}\) By the property of compact operators, there is a gap between the eigenvalue \(1\) and the rest of the eigenvalues. Thus if \(\phi\in\mathcal H^{p}(M)^\perp\) and if \(\phi\) is smooth, we have

\begin{equation*} ||\Delta\phi||_0\geq\eps||\phi||_0 \end{equation*}

for some \(\eps>0\text{.}\) Thus we can define the inverse operator \(G\) on \(\mathcal H^{p}(M)^\perp\text{,}\) called the Green's operator, such that \(G=\Delta^{-1}|_{\mathcal H^{p}(M)^\perp}\text{.}\)
