This is a general fact about the spectrum of any linear operator on a Banach space. We assume that \(\Delta\) is densely defined in a Banach space \(\mathcal B\text{.}\) Assume that \({ Spec} \,(\Delta)=\emptyset\text{.}\) Let \(f\in \mathcal B\) and let \(\ell\) be abounded linear functional on \(\mathcal B\text{,}\) then

\begin{equation*} \ell((\lambda I-\Delta)^{-1}f) \end{equation*}

is a bounded holomorphic function of \(\C\text{,}\) which has to be a constant. More over, it must be the zero function. Since \(\ell\) and \(f\) are arbitrary, we conclude that \((\lambda I-\Delta)^{-1}=0\) which is not possible for all \(\lambda\text{.}\) This is a contradiction.
