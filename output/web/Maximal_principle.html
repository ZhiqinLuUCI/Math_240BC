Section 3.2 Gradient estimates and maximal principle

If \(X\) is a compact manifold, then any smooth real function on \(X\) reaches its maximum point. At the maximum point, the first derivatives are zero, and the Hessian matrix at the point is non-positive.

For non-compact manifold, the above statement is not true in general. In order to estimate a function we sometimes need a differential inequality. Usually such a differential inequality is obtained by doing the so-called gradient estimate.

We start with the following generalized maximum principle:

Theorem 3.2.1.

Let \(f\) be a positive smooth function on a complex non-compact Riemannian manifold \(X\text{.}\) We assume that \({ Ric} (X) \geq - (n-1) k^2\) for some number \(k\text{.}\) Let \(\phi_1, \phi_2\) be two smooth functions on \(X\) with \(\phi_1\) bounded. Assume that there are constants \(\alpha > 0 , C_1, C_2\) such that

\begin{equation} \nabla f \geq C_1 f ^{l+ \alpha} + \phi _0 \nabla \phi_1 \nabla f - C_2 .\tag{$\star$}\tag{3.2.1} \end{equation}

Then \(f\) is bounded. Furthermore, there is a constant

\begin{equation*} C = C (\alpha, C_1, C_2 \Vert \phi _0 \Vert C_0 \nabla \phi _1 ) \end{equation*}

such that

\begin{equation*} f \leq C\text{.} \end{equation*}

\begin{proof} We first assume that \(f\) is bounded. That is

\begin{equation*} \sup f \lt + \infty\text{.} \end{equation*}

We claim that there is a sequence \(\{ x_k\}\) in \(X\) such that for any \(\varepsilon > 0\text{,}\) if \(k\) is large enough, we have

\begin{align*} \amp \amp f(x_k) > \sup f - \varepsilon\\ \amp \amp | \nabla f | (x_k) \lt \varepsilon\\ \amp \amp \Delta f (x_k) \lt \varepsilon\text{.} \end{align*}

To prove the claim, we first take a sequence \(\{ y_k\}\) such that

\begin{equation*} \lim_{k \rightarrow \infty} f (y_k) = \sup f\text{.} \end{equation*}

Define the cut-off function \(\rho\) as follows

\begin{equation*} \phi : \mathbb{R} \rightarrow \mathbb{R} \end{equation*}

smooth, \(0 \leq \rho \leq 1, \rho (t) = 1\) for \(0 \leq t \leq 1\text{.}\) \(\rho (t) = 0\) for \(t > 2\) and \(\rho^\prime (t) \leq 0\) for all \(t \in \mathbb{R}\text{.}\) Let \(R\) be a large number to be determined later. Let \(d(x) = { dist} (x, y_k)\) be the distance function. In general, the function \(d(x)\) is only a continuous function. But let

\begin{equation*} g(x) = \rho \left( \frac{d^2(x)}{R^2} \right) f (x)\text{.} \end{equation*}

If the maximum point of \(g(x)\) happens to be not smooth, we can always perturb the reference point by a little. So without loss of generality, we assume that \(g(x)\) is smooth. Let \(x_k\) be the maximum point of \(g(x)\text{.}\) By the definition of \(x_k\text{,}\) we have

\begin{equation*} g(x_k) \geq g (y_k) = f (y_k) \geq \sup f - \varepsilon\text{.} \end{equation*}

Using the definition, we also have

\begin{equation*} 1 - \rho \left( \frac{d^2 (x_k) }{R^2} \right) \ \lt \frac{\varepsilon }{\sup f }\text{.} \end{equation*}

Since

\begin{equation*} 0 = \nabla g (x_k) = \rho^\prime \frac{2}{k^2} d \nabla d f (x_k) + \rho \nabla f (x_k) \end{equation*}

we have

\begin{equation*} |\nabla f| (x_k)= - \frac{\rho^\prime}{\rho} \frac{2}{k^2} d | \nabla d |f (x_k)\text{.} \end{equation*}

Because \(| \nabla d | = 1\text{,}\) if \(R\) is big enough, we have

\begin{equation*} |\nabla f| (x_k) \lt \varepsilon\text{.} \end{equation*}

Finally, we have

\begin{equation*} 0 \geq \Delta g (x_k) = \rho f + 2 \nabla \rho \nabla f + f \Delta \rho\text{.} \end{equation*}

In order to prove the claim, we just need to prove that

\begin{equation*} \Delta \rho \left( \frac{d^2 (x, y_k )}{R^2} \right) \lt \varepsilon \end{equation*}

for \(k\) large enough. A straight forward computation gives that

\begin{equation*} \Delta \rho = \frac{2}{R^2} \rho^\prime + \frac{4d^2}{R^4} \rho^{\prime \prime} + \frac{2}{R^2} \rho^\prime d \Delta d\text{.} \end{equation*}

Since the Ricci curvature of \(X\) is bounded from below, by Laplacian comparison theorem, we have

\begin{equation*} d \Delta d \geq -C \end{equation*}

for some constant \(C\) depending only on the lower bound of the Ricci curvature. Since \(\rho^\prime \leq 0\text{,}\) for \(R\) large enough we have

\begin{equation*} \Delta \rho \lt \varepsilon \end{equation*}

for any \(\varepsilon\text{.}\) The claim is proved.

Using the differentiable inequality (\(\star\)), we have

\begin{equation*} \varepsilon \geq C_1 f (x_k) ^{1 + \alpha} = \Vert \phi_0 \Vert \, \Vert \nabla \phi_1 \Vert \varepsilon - C_2\text{.} \end{equation*}

Thus

\begin{equation*} f (x_k) \leq {}^{1+\alpha}\sqrt{\frac{C_2 + \Vert\phi _0\Vert \,\Vert \nabla \phi _1 \Vert}{C_1}} \\text{.} \end{equation*}

Taking \(k \rightarrow \infty\text{,}\) we get the effective bound for \(f\text{.}\)

Now we assume that \(\sup f = + \infty\text{.}\) Let

\begin{equation*} v = 1 - \frac{1}{(1+f) ^\beta} \end{equation*}

for \(\beta > 0\) to be determined later. Then by the previous result, there is a sequence \(\{ x_k\}\) such that for any \(\varepsilon > 0\text{,}\) if \(k\) is large enough, we have

\begin{align*} \amp \amp v (x_k) > 1 - \varepsilon\\ \amp \amp |\nabla v | (x_k) \lt \varepsilon\\ \amp \amp \Delta v (x_k) \lt \varepsilon \text{.} \end{align*}

By the definition of \(v\text{,}\) we have

\begin{equation*} \Delta v = \beta \frac{\Delta f }{(1+ \delta )^{1+ \beta}} - \frac{1+\beta}{\beta} | \nabla v|^2 (1 +f )^\beta \end{equation*}

Applying the above inequality to \((\star)\text{,}\) we get

\begin{equation*} \varepsilon > \beta \frac{C_1 f^{1+\alpha}- \Vert\phi _0\Vert \,\Vert \nabla \phi _1 \Vert \cdot \varepsilon - C_2}{(1+f ) ^{1 + \beta}} - \frac{1+\beta }{\beta} \cdot \varepsilon (1 + f)^\beta\text{.} \end{equation*}

Letting \(k \rightarrow \infty\text{,}\) we get a uniform bound of \(\sup f\) from the above inequality.

Let \(f\) be a smooth function on a complete non-compact manifold. We say that \(f\) is a harmonic function, if

\begin{equation*} \Delta f = 0\text{.} \end{equation*}

Then we have the following:

Theorem 3.2.2. Yau.

Let \(p > 1\text{,}\) if \(f \in L ^p , \Delta f = 0\text{.}\) Then \(f\) is a constant.

For the sake of simplicity, we only prove the theorem for \(p = 2\text{.}\)

Let \(f\) be an \(L^2\) harmonic function. Take a fixed point \(x_0\text{.}\) Let

\begin{equation*} d (x) = { dist} (x, x_0)\text{.} \end{equation*}

Let \(\rho\) be a cut-off function such that \({ supp} \rho\) is compact. We consider

\begin{equation*} g (x) = \rho \left( \frac{d(x)}{R} \right) f (x)\text{.} \end{equation*}

We have

\begin{equation*} \int_M \rho ^2 | \nabla f (x) |^2 = - 2 \int \rho f \nabla \rho \nabla f \frac{1}{R}\text{.} \end{equation*}

Using the Cauchy inequality, we get

\begin{equation*} \int_M \rho^2 | \nabla f | ^2 \leq \frac{2}{R} \left( \int_M \rho^2 |\nabla f|^2 \right) ^{\frac{1}{2}} \int_M |\nabla \rho| ^2 f\text{.} \end{equation*}

Since \(| \nabla \rho | \leq C\text{.}\) We get

\begin{equation*} \int_M \rho^2 |\nabla f | ^2 \leq C \frac{1}{R} \int_M f^2\text{.} \end{equation*}

Letting \(R\rightarrow \infty\text{,}\) we get \(\nabla f \equiv 0\) so \(f\) is a constant.

By the above theorem, for a complete manifold, the only interesting harmonic functions may be bounded harmonic functions.

In order to study the bounded harmonic functions, we first introduce the Ricci identity.

Let \(f\) be a smooth function of \(M\text{.}\) Define the derivative of \(f\) using the following formula

\begin{equation*} f_i \omega _i = df\text{.} \end{equation*}

Using the same idea, we define

\begin{equation*} f_{ij} \omega _j = d f_i - f _s \omega _{s j}\text{.} \end{equation*}

The matrix \((f_{ij} )\) is called the Hessian matrix. We have

\begin{equation*} f_{ij} \omega _j \wedge \omega _i = d f _i \wedge \omega _i - f _s \omega _{s j} \wedge \omega _i = 0\text{.} \end{equation*}

Thus the Hessian matrix is always symmetric.

The 3\(^{ rd}\) order co-variant derivatives are defined as

\begin{equation*} f_{ijk} \omega _k = d f _{ij} - f _{is} \wedge \omega _{sj} - f _{sj} \wedge \omega _{si}\text{.} \end{equation*}

A careful computation gives

\begin{equation*} f_{ijk} \omega _k \wedge \omega _{j}= - \frac{1}{2}f_s R_{si kj} \omega _k \wedge \omega _j\text{.} \end{equation*}

Thus we have

\begin{equation*} f_{ijk} - f _{ikj} = + f _s R_{sikk} \end{equation*}

In particular, we have the following Ricci identity

\begin{equation*} f_{iik} - f _{kii} = - f_s { Ric} _{sk} \end{equation*}

Remark 3.2.3.

We have \(\Delta f = f _{ii}\text{.}\)

With the preparation above, we prove the following.

Theorem 3.2.4.

Let \(M\) be a complete Riemannian manifold, \(\dim M = n \geq 2\text{.}\) \({ Ric}(M) \geq - (n-1) k , k \geq 0\text{.}\) Let \(u\) be a positive harmonic function. Then on any geodesic ball \(B_a (x)\text{,}\) we have

\begin{equation*} \frac{|\nabla u|}{u} \leq C_n \left( \frac{1+ a k ^{\frac{1}{2}}}{a} \right)\text{.} \end{equation*}

where \(C_n\) is a constant only depends on \(n\text{.}\) We first prove that

\begin{equation} \frac{1}{2} \Delta |\nabla u | ^2 \geq \sum _{ij} u_{ij} ^2 - (n-1) k | \nabla u|^2. \tag{$\triangle$}\tag{3.2.2} \end{equation}

To see this, we do the following

\begin{equation*} \left( \sum u^2_j\right) _i = 2 u_j u_{ji}\text{.} \end{equation*}

Thus,

\begin{equation*} \frac{1}{2} \Delta |\nabla u | ^2 + ( u_j u_{ji} )_i = u ^2_{ji} + u_j u_{jii}\text{.} \end{equation*}

Using the Ricci identity, we have

\begin{equation*} \frac{1}{2} \Delta |\nabla u| ^2 = u^2_{ji} + u _j (\Delta u)_j + { Ric} (\nabla u, \nabla u)\text{.} \end{equation*}

Thus \((\triangle)\) follows from the assumption on the Ricci curvature and harmonicity of \(u\text{.}\)

We now consider the points such that \(\nabla u \neq 0\text{.}\) By changing a frame, we may assume that

\begin{equation*} u_i \neq 0 , \ u_j = 0 \ \mbox{for} \ \ j \lt 1\text{.} \end{equation*}

From \((\triangle)\text{,}\) we conclude that

\begin{equation*} \Delta | \nabla u| = \frac{\Delta | \nabla u|^2}{2 | \nabla u|} - \frac{1}{4} \frac{| \nabla | \nabla u| ^2 |^2}{| \nabla u|^3}\text{.} \end{equation*}

Using the above information, we get

\begin{equation*} \Delta |\nabla u| \geq \frac{1}{| \nabla u|} \left( \sum_{j > 1} u^2_{ij} + \sum _{j>1} u^2_{jj} \right) - (n-1) k |u_1|\text{.} \end{equation*}

Since

\begin{equation*} \sum _{j>1} u^2_{jj} \geq \frac{1}{n-1} \left( \sum_{j>1} u_{jj} \right) ^2 = \frac{1}{n-1} u^2_{11}\text{.} \end{equation*}

we have

\begin{equation*} \Delta |\nabla u| \geq \frac{1}{n-1} \frac{1}{|\nabla u|} \sum_j u^2_{ij} - (n-1) k |u_1|\text{.} \end{equation*}

Now we assume that \(\phi = |\nabla u|/u\text{.}\) Then we have

\begin{equation*} \Delta \phi = \frac{\Delta |\nabla u|}{u} + 2 \nabla |\nabla u| \nabla \frac{1}{u} + |\nabla u| \Delta \left( \frac{1}{u} \right)\text{.} \end{equation*}

Since

\begin{equation*} 2 \nabla | \nabla u| \nabla \frac{1}{u} = 2 \nabla (\phi u) \nabla \left(\frac{1}{u} \right) = - 2 \frac{\nabla \phi \nabla u}{u} - 2 \phi ^3\text{.} \end{equation*}

We have

\begin{equation*} \nabla \phi = \frac{\Delta |\nabla u|}{u} - 2 \frac{\nabla \phi \nabla u}{u}\text{.} \end{equation*}

We have

\begin{equation*} \frac{\nabla \phi \nabla u}{u} = - \phi ^3 - 2 \nabla |\nabla u| \nabla \frac{1}{u}\text{.} \end{equation*}

Thus if \(\varepsilon > 0\) is small enough

\begin{equation*} \frac{\Delta |\nabla u|}{u} - \varepsilon \frac{\nabla \phi \nabla u}{u} \geq \varepsilon \phi ^3\text{.} \end{equation*}

Thus we get

\begin{equation*} \Delta \phi \geq - (n-1) k \phi - (2 - \varepsilon ) \frac{\nabla \phi \nabla u}{u} + \varepsilon \phi^3\text{.} \end{equation*}

Using the maximum principle, we have \(\phi\) is bounded.

Corollary 3.2.5.

Let \(M\) be a complete Riemannian manifold, if \({ Ric} (M) \geq 0\text{.}\) Then any positive harmonic function is a constant.

Proof.

If \({ Ric} \geq 0\) then \(k = 0\text{.}\) We have

\begin{equation*} \Delta \phi \geq - (2 - \varepsilon ) \frac{\nabla \phi \nabla u}{u} + \varepsilon \phi^3\text{.} \end{equation*}

For \(\varepsilon = \frac{2}{n-1}\text{.}\) Thus using the generalized maximal principal, \(\phi \equiv 0\text{.}\)

Corollary 3.2.6. Harnack **.

Let \(M\) be \(n\)-dimensional Riemannian manifold, \({ Ric} (M) \geq - (n-1) k\text{.}\) If \(u\) is a positive harmonic function in \(Ba\text{.}\) Then

\begin{equation*} \sup_{Ba/2} u \leq C (n, a, k) \inf_{B a/2} u \end{equation*}

where \(C (n, a, k)\) are constants depending only on \(n, a, k\text{.}\)

\begin{proof} By the above theorem, we have

\begin{equation*} \sup_{Ba} \frac{|\nabla u|}{u} \leq C (n, a, k)\text{.} \end{equation*}

Thus

\begin{equation*} | \nabla lg u |\leq C (n, a, k) \end{equation*}

the conclusion follows.

Corollary 3.2.7.

Suppose \({ Ric} (M) \geq 0\text{.}\) Then any positive harmonic function must be constant.

Let the Riemann metric be written as

\begin{equation*} (dr)^2 + h_{ij} (r, \theta) \, d \theta _i d \theta _j\text{.} \end{equation*}

What is the asymptotic behavior of \(h_{ij} (r, \theta)\) when \(r \rightarrow 0\text{.}\)

Note that the Riemannian metric at the reference point is regular. Thus we can define \(( \theta _2, \ldots, \theta _n)\) as

\begin{equation*} \theta _j = \frac{x_j}{r}, \;\; r = \sqrt{\sum x^2_j}\text{.} \end{equation*}

Suppose

\begin{equation*} ds ^2 = g_{ij} d x_i dx_j\text{.} \end{equation*}

Let's compare

\begin{equation*} g_{ij} d x_i dx_j \sim (dr ) ^2 + h_{ij} d \theta _i d \theta _j\text{.} \end{equation*}

We have

\begin{align*} g_{ij} d x _i dx_j \amp = \amp g_{ij} ( dr \theta _i + r d \theta _i ) ( d r \theta _j + r d \theta _j )\\ \amp = \amp g _{ij} \theta_i \theta_j (dr) ^2 + 2 g _{ij} x_i dr d \theta _j + g _{ij} r ^2 d \theta _i d \theta _j \text{.} \end{align*}

By comparison, we have

\begin{align*} \amp \amp g_{ij} \theta_i \theta _j = 1\\ \amp \amp \sum _i g_{ij} x _i = 0 \end{align*}

and

\begin{equation*} h_{ij} = r^2 \left(g_{ij} - g_{ij} \frac{\theta_i}{\theta_1} - g_{i1} \frac{\theta_j}{\theta_1} + g_{11} \frac{\theta_i \theta_j}{\theta^2_1} \right)\text{.} \end{equation*}

Thus

\begin{equation*} \sqrt{{ det} ( h _{ij} ) } = \gamma ^{n-1} \cdot \mu \end{equation*}

for \(\mu\) being a regular function (at least for fixed \(\theta_i\)). Thus

\begin{equation*} \Delta \rho = \frac{\partial f}{\partial r} \sim \frac{n-1}{r} + \ \mbox{small terms}\text{.} \end{equation*}

If we choose \((g_{ij})\) to be normal, we can actually compute

\begin{equation*} { det}\left( \delta_{ij} + \frac{\theta_i \theta_j}{\theta^2_1} \right) = \frac{1}{\theta^2_1}\text{.} \end{equation*}

Thus

\begin{equation*} \sqrt{{ det}\,h_{ij}} \end{equation*}

can be extended as a regular function near 0.
