Section 2.3 Variational characterization of spectrum

Let \(\Delta\) be the Laplace operator on functions of a manifold \(M\text{.}\) As in the last section, we make the convention that the operator is a negative operator.

Let \(\lambda\) be a complex number, \((\lambda I-\Delta)^{-1}\) is called the resolvent of \(\Delta\text{.}\) If for some \(\lambda\text{,}\) \((\lambda I-\Delta)^{-1}\) doesn't exist, or it does exist but is an unbounded operator, then we call \(\lambda\) spectrum point of \(\Delta\text{.}\) We use \(\sigma(\Delta)\) or \({ Spec}\, (\Delta)\) to denote the set.

Theorem 2.3.1.

Using the above notations, we have

\begin{equation*} { Spec} \,(\Delta)\neq\emptyset\text{.} \end{equation*}

Proof.

This is a general fact about the spectrum of any linear operator on a Banach space. We assume that \(\Delta\) is densely defined in a Banach space \(\mathcal B\text{.}\) Assume that \({ Spec} \,(\Delta)=\emptyset\text{.}\) Let \(f\in \mathcal B\) and let \(\ell\) be abounded linear functional on \(\mathcal B\text{,}\) then

\begin{equation*} \ell((\lambda I-\Delta)^{-1}f) \end{equation*}

is a bounded holomorphic function of \(\C\text{,}\) which has to be a constant. More over, it must be the zero function. Since \(\ell\) and \(f\) are arbitrary, we conclude that \((\lambda I-\Delta)^{-1}=0\) which is not possible for all \(\lambda\text{.}\) This is a contradiction.

When \(M\) is a compact manifold, the best possible things happen: by the Hodge theorem, the spectrum of \(\Delta\) is made from eigenvalues. More precisely, we have

\begin{equation*} 0=\lambda_0\lt \lambda_1\leq\lambda_2\cdots\leq\lambda_k\to +\infty\text{,} \end{equation*}

such that for each \(\lambda_j\text{,}\) the space

\begin{equation*} E_j=\{f\mid\Delta f=-\lambda_j f\} \end{equation*}

is not trivial and is of finite dimensional. By the above result, we can prove the following variational (or min-max) principal.

Theorem 2.3.2.

We have

\begin{equation} \lambda_k = \inf \left\{ \left. \frac{ \int_{\Omega} |\nabla f |^2 }{\int_{\Omega} f^2 } \, \right| \, \, \, \left. f\right|_{\pa \Omega} = 0, \, f \not\equiv 0 = \int_{\Omega} f \phi_j, \, \, \forall \, j \lt k \right\}\text{,}\tag{2.2} \end{equation}

where \(\phi_j\) are the eigenfunctions of \(\lambda_j\text{.}\)

Proof.

Let \(\{ f_\alpha\}\) be a sequence such that

\begin{equation*} \int_M f_\alpha\phi_j=0, \qquad \forall j\lt k\text{,} \end{equation*}

and

\begin{equation*} \frac{\int_M|\nabla f_\alpha|^2}{\int_M f_\alpha^2}\to\lambda_k\text{.} \end{equation*}

If we normalize \(f_\alpha\) such that \(\int_M f_\alpha^2=1\text{,}\) then the sequence \(\{f_\alpha\}\) is bounded in \(H_1(M)\text{.}\) Therefore, there is an \(f\in H_1(M)\) such that

\begin{equation*} f_\alpha\to f \end{equation*}

in the weak sense. In particular, for any \(\phi\) smooth, we have

\begin{equation*} \int_M\nabla f_\alpha\nabla\phi\to\int_M\nabla f\nabla\phi\text{.} \end{equation*}

We have

\begin{equation*} \left|\int_M\nabla f\nabla\phi\right|\leq\int_M\nabla f_\alpha\nabla\phi\leq ||\nabla f_\alpha||_{L^2}||\nabla\phi||_{L^2}\to\sqrt{\lambda_k}||\nabla\phi||_{L^2}\text{.} \end{equation*}

Since smooth functions are dense in \(H_1(M)\text{,}\) we have

\begin{equation*} \int_M|\nabla f|^2\leq\liminf_{k\to\infty}\int_M|\nabla f_\alpha|^2\text{,} \end{equation*}

which is known as the Fatou Lemma.

On the other hand, by the Rellich Lemma, \(f_\alpha\to f\) strongly in \(L^2(M)\text{.}\) Thus we have

\begin{equation*} \int_Mf^2=1\text{,} \end{equation*}

hence

\begin{equation*} \lambda_k=\frac{\int_M|\nabla f|^2}{\int_M f^2} \end{equation*}

Let \(\phi\) be a smooth function such that

\begin{equation*} \int_M\phi\phi_j=0 \end{equation*}

Then for any \(\eps\text{,}\) we must have

\begin{equation*} \frac{\int_M|\nabla f+\eps\nabla\phi|^2}{\int_M (f+\eps\phi)^2}\geq\frac{\int_M|\nabla f|^2}{\int_M f^2} \end{equation*}

Since this is true for any \(\eps\text{,}\) we must have

\begin{equation*} \Delta f=-\lambda_k f \end{equation*}

in the weak sense. By the elliptic regularity, \(f\) has to be smooth.

Unlike in the case of compact manifold, in general, a complete non-compact manifold doesn't admit any pure point spectrum. For example, there are no \(L^2\)-eigenvalues on \(\mathbb R^n\text{.}\) That is, for any \(\lambda\in\mathbb R\text{,}\) if \(\Delta f+\lambda f=0\) and \(f\in L^2(\mathbb R^n)\text{,}\) then we have \(f\equiv 0\text{.}\)

Let \(\Delta\) be the Laplace operator on a complete non-compact manifold \(M\text{.}\) We extend \(\Delta\) naturally to a self-adjoint densely defined operator on \(L^2(M)\text{,}\) which we still denote as \(\Delta\) for the sake of simplicity.

The pure point spectrum of \(\Delta\) are those \(\lambda\in\mathbb R\) such that

  1. there exists an \(L^2\) function \(f\neq 0\) such that

    \begin{equation*} \Delta f+\lambda f=0\text{.} \end{equation*}

  2. the multiplicity of \(\lambda\) is finite;

  3. in a neighborhood of \(\lambda\text{,}\) it is the only spectrum point.

From the above discussion, \(\sigma(\Delta)\) decomposes as the union of pure point spectrum, and the so-called essential spectrum, which is, by definition, the complement of the pure point spectrum.

The set of the essential spectrum is denoted as \(\sigma_{ess}(\Delta)\text{.}\) Using the above definition, \(\lambda\in\sigma_{ess}(\Delta)\text{,}\) if either

  1. \ding{172}.

    \(\lambda\) is an eigenvalue of infinite multiplicity, or

  2. \ding{173}.

    \(\lambda\) is the limiting point of \(\sigma(\Delta)\text{.}\)

The following theorems in functional analysis are well-known.

Theorem 2.3.3.

A necessary and sufficient condition for the interval \((-\infty,\lambda)\) to intersect the essential spectrum of an self-adjoint densely defined operator \(A\) is that, for all \(\eps>0\text{,}\) there exists an infinite dimensional subspace \(G_\eps\subset \mathcal Dom(A)\text{,}\) for which \((Af-\lambda f-\eps f,f)\lt 0\text{.}\)

Theorem 2.3.4.

A necessary and sufficient condition for the interval \((\lambda-a,\lambda+a)\) to intersect the essential spectrum of \(A\) is that there exists an infinite dimensional subspace \(G_\eps\subset \mathcal Dom(A)\) for which \(||(A-\lambda I)f||\leq a||f||\) for all \(f\in G_\eps\text{.}\)

Using the above result, we give the following variational characterization of the lower bound of spectrum and the lower bound of essential spectrum.

Theorem 2.3.5.

Using the above notations, define

\begin{equation*} \lambda_0=\inf_{f\in C_0^\infty(M)}\frac{\int_M|\nabla f|^2}{\int_M f^2}\text{,} \end{equation*}

and

\begin{equation*} \lambda_{ess}=\sup_K\inf_{f\in C_0^\infty(M\backslash K)}\frac{\int_M|\nabla f|^2}{\int_M f^2}\text{,} \end{equation*}

where \(K\) is a compact set running through an exhaustion of the manifold. Then \(\lambda_0\) and \(\lambda_{ess}\) are the least lower bound of \(\sigma(\Delta)\) and \(\sigma_{ess}(\Delta)\text{,}\) respectively.

Proof.

We prove the formula for \(\lambda_{ess}\text{.}\) The formula for \(\lambda_0\) is similar. Let

\begin{equation*} \lambda'_{ess}=\inf \sigma_{ess}(\Delta)\text{.} \end{equation*}

If \(\sigma_{ess}(\Delta)=\emptyset\text{,}\) we define \(\lambda'_{ess}=+\infty\text{.}\) When \(\lambda'_{ess}=+\infty\text{,}\) we prove that \(\lambda_{ess}\) is also infinity.

Assume not, then there is a constant \(C\text{,}\) such that for any compact set \(K\text{,}\) there is a function \(f\in \mathcal C_0^\infty(M\backslash K)\text{,}\) we have

\begin{equation*} \frac{\int_M |\nabla f|^2}{\int_M f^2}\lt C\text{.} \end{equation*}

By inductively choosing \(f\text{,}\) we can make sure that the support of these functions are disjoint. Thus the functions span an infinite dimensional space \(G\) with

\begin{equation*} (-\Delta f-Cf,f)\lt 0\text{,} \end{equation*}

which contradicts to the fact that the essential spectrum is an empty set.

The proof of the general case is similar. We fist prove that \(\lambda_{ess}\leq\lambda_{ess}'\) By definition, for \(\eps>0\text{,}\)

\begin{equation*} (-\infty,\lambda_{ess}'+\eps)\cap \sigma_{ess}(\Delta)\neq\emptyset\text{.} \end{equation*}

Therefore we can find infinite dimensional space \(V\) such that for any \(f\in V\)

\begin{equation*} (-\Delta f-(\lambda_{ess}'+\eps) f, f)\lt 0\text{.} \end{equation*}

In what follows, we shall prove that, using the cut-off functions, there exist infinitely many elements in \(V\) such that their supports are disjoint.

We assume that \(K\) is a compact set. Let \(K'\) be a large ball containing \(K\text{.}\) Let \(\rho\) be the cut-off function such that \(\rho=1\) on \(K\) but \(\rho=0\) outside \(K'\text{.}\) We claim that for any \(\eps>0\text{,}\) there is an \(f\in V\) with \(\int f^2=1\) but

\begin{equation*} \int_M\rho^2f^2\lt \eps\text{.} \end{equation*}

If the above is not true, then there is an \(\eps_0>0\) such that for any \(f\in V\text{,}\)

\begin{equation*} \int_M\rho^2f^2\geq\eps_0\text{,} \end{equation*}

Since the set \(f\in V\) is of infinite dimensional, the set \(\rho f\) is of infinite dimensional as well, otherwise the above inequality is not valid. Thus we can find an orthogonal basis

\begin{equation*} \int_M\rho^2f_if_j=0\text{,} \end{equation*}

if \(i\neq j\text{,}\) while we still keeping \(\int_Mf_i^2=1\text{.}\) We consider

\begin{equation*} \int_M|\nabla(\rho f_i)|^2\leq 2\int_M|\nabla \rho|^2 f_i^2+2\int_M|\nabla f_i|^2\text{.} \end{equation*}

By \(\int_M|\nabla f_i|^2\leq \lambda'_{ess}+\eps\text{,}\) we have Thus we have

\begin{equation*} \frac{\int_M|\nabla(\rho f_i)|^2}{\int_M(\rho f_i)^2} \leq \frac{2C+2(\lambda_{ess}'+\eps)}{\eps_0} \end{equation*}

for any \(i\text{.}\) This is a contradiction because on the compact set \(K'\text{,}\) the are only finitely many eigenvalues (counting multiplicity) below a fixed number.

Now we can prove our theorem. For all \(\eps>0\text{.}\) We can find an \(f\) with \(\int_M f^2=1\) but

\begin{equation*} \int_M\rho^2f^2\lt \eps\text{.} \end{equation*}

We take \(K=B(r)\) and \(K'=B(R+1)\) of balls of radius \(r, R+1\text{,}\) respectively. Both \(r,R\) are large numbers. Let \(\tilde\rho\) be a cut-off function such that

\begin{equation*} \{\tilde\rho=1\}\subset B(R)\text{.} \end{equation*}

We also assume that \(|\nabla\tilde\rho|\leq C\text{.}\) Let \(\rho_1=1-\tilde\rho\text{.}\) Then we have

\begin{equation*} { supp}\,\rho_1\cap K=\emptyset\text{.} \end{equation*}

We have

\begin{equation*} \int_M|\nabla(\rho_1f)|^2=\int_M\rho_1^2|\nabla f|^2+2\int_M\rho_1f\nabla\rho_1\nabla f+\int_M f^2|\nabla \rho_1|^2\text{.} \end{equation*}

From

\begin{equation*} \int_M f^2|\nabla \rho_1|^2\leq C\eps\text{,} \end{equation*}

and

\begin{equation*} 2\int_M\rho_1 f\nabla \rho_1\nabla f\leq 2 \sqrt{\int_M |\nabla(\rho_1 f)|^2}\cdot \sqrt{\int_M |\nabla\rho_1|^2f^2}\leq C\sqrt\eps \sqrt{\int_M |\nabla(\rho_1 f)|^2}\text{,} \end{equation*}

we get

\begin{equation*} \int_M|\nabla (\rho_1f)|^2\leq\lambda_{ess}'+C\eps\text{.} \end{equation*}

Since

\begin{equation*} \int_M\rho_1^2f^2\geq 1-C\eps\text{,} \end{equation*}

we get

\begin{equation*} \lambda_{ess}\leq\frac{\lambda_{ess}'+C\eps}{1-C\eps}\text{.} \end{equation*}

and thus \(\lambda_{ess}\leq\lambda'_{ess}\text{.}\) The other direction is easier to prove.

It is particularly interesting to get the lower bound estimate for the essential spectrum because of the following theorem.

Theorem 2.3.6. Variational principal.

Suppose \(\lambda_0\lt \lambda_{ess}\text{,}\) then \(\lambda_0\) is en eigenvalue of \(M\) with finite dimensional eigenspace.

That is, there exists an \(L^2\) function \(f\neq 0\text{,}\) such that

\begin{equation*} \Delta f=-\lambda_0 f\text{,} \end{equation*}

which is a very strong result.

Example 2.3.7.

Let \(H=-\Delta+|x|^2\) on \(\mathbb R^n\text{.}\) Then the set of essential spectrum is empty. As a result, all spectrum of \(H\) are eingevalues of finite multiplicity.
