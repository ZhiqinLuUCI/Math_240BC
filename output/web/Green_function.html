Section 4.2 Green's function and parobolicity

Let \(M\) be a complete non-compact Riemannian manifold. The Green's function is a smooth function on

\begin{equation*} M \times M \backslash { diag} (M) \end{equation*}

such that

  1. \(G (x, y) = G (y, x)\) and fixing \(y\text{,}\) we have \(\Delta _x G (x, y) = 0 \, \forall x \neq y\text{.}\)

  2. \(G(x, y) \geq 0\text{.}\)

  3. Fixing \(y\text{,}\) when \(x \rightarrow y\text{,}\) we have the following

\begin{equation*} G(x, y) = \left\{ \begin{array}{ll} \rho _y (x) ^{2-n} (1 + 0(1) )\amp n>2 \\ - \log \rho_y (x) (1 + 0(1)) \amp n = 2 \end{array} \right.\text{.} \end{equation*}

This last asymptotical expansion of the Green's function also implies that \(\Delta _x G(x, y) = - \delta _{x \cdot y}\text{.}\)

From the asymptotical behavior we can find that \(n = 2\) and \(n> 2\text{,}\) the Green's functions are very different.

We make the following.

Definition 4.2.1.

A complete manifold is said to be parabolic, if and only if it doesn't admit a positive Green's function. Otherwise it is said to be non-parabolic.

Definition 4.2.2.

An End, \(E\text{,}\) with respect to a compact subset \(\Omega \subset M\) is an unbounded connected component of \(M \backslash \Omega\text{.}\) The number of ends with respect to \(\Omega\text{,}\) denoted by \(N_{\Omega}(M)\text{,}\) is the number of unbounded connected component of \(M \backslash \Omega\text{.}\)

Definition 4.2.3.

An End \(E\) is said to be parabolic, if it doesn't admit a positive harmonic function \(f\) satisfying

\begin{equation*} f \equiv \ \mbox{on} \ \partial E \end{equation*}

and

\begin{equation*} \frac{\lim}{n\rightarrow E (\infty)} f(y) \lt 1 \end{equation*}

where \(E(\infty)\) denotes the infinity of \(E\text{.}\) Otherwise, \(E\) is said to be non-parabolic and the function \(f\) is said to be a barrier function of \(E\text{.}\)

We prove the following result:

Theorem 4.2.4.

Let \(E\) be an parabolic end. Let \(A(R) = E \cap \partial B(R)\) where \(B(R)\) is the ball of radius \(R\) with respect to some reference point. Let \(f\) be a harmonic function on \(E\) such that

\begin{equation*} f|_{\partial E} = 1 , f | _{A(R) } = 0\text{.} \end{equation*}

Then

\begin{equation*} \lim_{R\rightarrow \infty} \int_E |\nabla f|^2 \rightarrow 0\text{.} \end{equation*}

Proof.

Using the Green's formula, we have

\begin{equation*} \int _E |\nabla f|^2 = - \int _E \frac{\partial f}{\partial r} \end{equation*}

where \(\frac{\partial }{\partial r}\) is the outer normal direction. We claim that \(\frac{\partial f}{\partial r} \rightarrow 0\text{.}\) To see this, we take a sequence \(R_1 \lt R_2 \lt \ldots R_k \rightarrow \infty\text{.}\) The corresponding harmonic function \(f_i = f_{R_{i}}\text{.}\) By the maximal principle, \(f_i\) are increasing sequence on any compact set of \(E\text{.}\) Let

\begin{equation*} \lim_{i \rightarrow \infty} f_i = f\text{.} \end{equation*}

Then \(f\) must be a positive harmonic function. By the parabolicity, \(f \equiv 1\text{.}\) By the maximal principle again

\begin{equation*} \frac{\partial f_i}{\partial r} \rightarrow 0 \end{equation*}

as \(i \rightarrow \infty\text{.}\)

Examples: \(\mathbb{R}^2\) and \(\mathbb{R}^n\text{.}\)

Let \(R > 0\) be a big number. Let

\begin{equation*} F(R) = \{ f \in C^\infty _0 (\mathbb{R}^n) | f = 1 \; \mbox{for} \; |x| \lt R, f \; \mbox{rotational symmetric} \}\text{.} \end{equation*}

If \(n > 2\text{,}\) then for any \(C > 0\text{,}\) there exists an \(R_0\) such that for any \(R > R_0\)

\begin{equation*} \int_{\mathbb{R}^n}|\nabla f | ^2 > C \end{equation*}

for any \(f \in F(R)\text{.}\) If \(n = 2\text{,}\) then for any \(\varepsilon > 0\) there exists \(R_0 > 0\) such that for any \(R> R_0\text{,}\) we can find an \(f_R \in F(R)\) for which

\begin{equation*} \int_{\mathbb{R}^2} | \nabla f|^2 \lt \varepsilon\text{.} \end{equation*}

Proof.

If \(n > 2\text{,}\) then

\begin{equation*} \int ^\infty_{\mathbb{R}^2} \frac{1}{r^{n-1}} \, d r = \frac{1}{n-2} \frac{1}{R^{n-2}}\text{.} \end{equation*}

Thus we have

\begin{align*} \int_{\mathbb{R}^n} | \nabla f|^2 \amp \geq \amp (n-2) \subset R^{n-2} \int ^\infty_R r ^{n-1} \left( \frac{\partial f}{\partial r} \right)^2 dr \int^\infty_0 \frac{1}{r^{n-1}} dr\\ \amp \geq\amp (n-1) \subset R^{n-2} \rightarrow + \infty \text{.} \end{align*}

However, for \(n = 2\text{,}\) we define \(f_R = \sigma _R (|x|(\) such that

\begin{equation*} \sigma _R (t) = \left\{ \begin{array}{ll} 1 \amp t \leq R\\ \left( 1 - \frac{\log R}{R}\right) ^{-1} \left( \frac{\log R}{\log t} - \frac{\log R}{R} \right) \amp R \lt t \lt e ^R\\ 0 \amp t \geq e^R \end{array} \right.\text{.} \end{equation*}

Then a straightforward computation gives

\begin{equation*} \int^\infty_0 + | \sigma ^\prime _R (t) |^2 dt \leq \frac{4}{3} \frac{1}{\log R} \; \;\;\; \mbox{for} \; R >>0\text{.} \end{equation*}

The main result of this section is the following characterization of parabolicity.

Theorem 4.2.5.

Let \(M\) be a complete manifold. If \(M\) is non-parabolic, then for any point \(p \in M\text{,}\) we have

\begin{equation*} \int^\infty_1 \frac{dt}{Ap(t)} \lt + \infty \end{equation*}

where \(A_p (r)\) denotes the area of \(\partial B_p (r)\text{.}\)

Proof.

For \(p\in M\text{,}\) let \(G(p, y)\) be the Green's function (assuming that it exists). Let \(G_i (p, y)\) be the Green's function on \(B_p (R_i)\) with the Dirichlet boundary condition, where \(R_i \rightarrow \infty\text{.}\)

For any \(|\lt R\lt R_i\text{,}\) let's denote that

\begin{align*} S_i(1) \amp = \amp \sup_{y \in \partial B_{p}(1)} \; G_i (p, y)\\ i_i(R) \amp = \amp \inf_{y \in \partial B_{p}(R)} \; G_i (p, y) \text{.} \end{align*}

Let \(f\) be the harmonic function defined on \(B_p (R) \backslash B_p (1)\) satisfying the boundary conditions

\begin{align*} \amp \amp f(y) = S_i (1) \; \;\mbox{on} \; \partial B_p(1)\\ \amp \amp f(y) = G_i (p, y) \;\;\mbox{on} \; \partial B_p(R) \text{.} \end{align*}

The maximum principle implies that

\begin{equation*} f (y) \geq G_i (p, y) \;\mbox{on} \; B_p (R) \backslash B_p (1)\text{.} \end{equation*}

In particular, we have

\begin{equation*} \frac{\partial f}{\partial r} \leq \frac{\partial G_i}{\partial r} \; \mbox{on} \; \partial B_p (R)\text{.} \end{equation*}

On the other hand, since \(f(y)\) is harmonic, Stokes theorem implies

\begin{equation*} 0 = \int _{B_{p} (R) | B_{p} (1) } \Delta f = \int _{\partial B_{p} (R)} \frac{\partial f}{\partial r} - \int _{\partial B_{p} (1)}\frac{\partial f}{\partial r}\text{.} \end{equation*}

Also, we observe that

\begin{equation*} \int _{\partial B_{p} (R)} \frac{\partial G_{i}}{\partial r} = \int _{ B_{p} (R)} \Delta G_i = - 1\text{.} \end{equation*}

Thus we get

\begin{equation*} \int _{\partial B_{p} (1)} \frac{\partial f}{\partial r} \leq - 1\text{.} \end{equation*}

consider \(h\) to be the harmonic function defined on \(B_p (R) \backslash B_p (1)\) satisfying the boundary conditions

\begin{align*} h(y) \amp = \amp S_i (1) \; \mbox{on} \; \partial B_p (1)\\ h(y) \amp = \amp i_i (R) \; \mbox{on} \; \partial B_p (R) \text{.} \end{align*}

Again, the maximum principle implies that

\begin{equation*} h(y) \leq f (y) \; \mbox{on} \; B_p (R) \backslash B_p (1) \end{equation*}

and

\begin{equation*} \frac{\partial h}{\partial r} \leq \frac{\partial h}{\partial r} \; \mbox{on} \; B_p (1)\text{.} \end{equation*}

Thus we have

\begin{equation*} \int _{\partial B_{p} (R)} \frac{\partial h}{\partial r} = \int _{\partial B_{p} (1) }\frac{\partial h}{\partial r} \leq - 1\text{.} \end{equation*}

Define the function

\begin{equation*} g(r) = (s_i (1) - i_i (R) ) \left( \int^R_1 \frac{dt}{A_p (t)} \right) ^{-1} \int ^R_r \frac{dt}{A_p (t) } + i_i (R)\text{.} \end{equation*}

Then \(g (r (y))\) will have the same boundary conditions as \(h(y)\text{.}\) The Dirichlet integral minimizing property for harmonic functions implies that

\begin{align*} \amp \int_{B_{p} (R) \backslash B_{p} (1)} |\nabla h |^2 \leq \int_{B_{p} (R) \backslash B_{p} (1)} |\nabla g|^2\\ \amp = \int^R_1 \left( ( s_i (1) - i _i (R)) \left( \int^R_1 \frac{dt}{A_p (t)} \right) ^{-1} \frac{1}{A_p (r)} \right) ^2 \, A_p (r) \, dr\\ \amp = \left( s_i (1) - i _i (R)\right) \left( \int^R_1 \frac{dt}{A_p (t)} \right) ^{-1}\text{.} \end{align*}

On the other hand, integration by parts yields

\begin{align*} \int_{B_{p} (R) \backslash B_{p} (1)} |\nabla h |^2 \amp = \amp i_i(R)\int_{\partial B_{p} (R) } \frac{\partial h}{\partial r } -s_i(1) \int_{\partial B_{p} (R) }\frac{\partial h}{\partial r }\\ \amp \leq \amp s_i (1) - i _i (R)) \text{.} \end{align*}

By taking the limit

\begin{equation*} \int ^k_1 \frac{dt}{A_p (t)} \leq \sup _{y \in \partial B_{p} (1) } G (p, y) - \inf _{y \in \partial B_{p} (R)} G (p , y)\text{.} \end{equation*}

Corollary 4.2.6.

Let \(M\) be a complete manifold such that

\begin{equation*} { vol} (B (R)) \leq CR^2 \end{equation*}

then \(M\) has to be parabolic.

Corollary 4.2.7.

Let \(M\) be a Riemann surface such that

\begin{equation*} \int_M |k| \lt + \infty \end{equation*}

Then \(M\) is parabolic.

Proof of a theorem of Brascamp and Lieb.

Theorem 4.2.8.

Let \(\Omega\) be a bounded convex domain of \(\mathbb{R} ^n\text{.}\) Let \(u\) be the first Dirichlet eigenfunction. Let

\begin{equation*} \Delta u = - \lambda _1 u \;\; u > 0\text{.} \end{equation*}

Then \(\log u\) must be concave.

\begin{proof} We choose any function \(u_0 > 0 , u_0 |_{\partial \Omega} = 0\) such that \(- \log u_0\) is concave. Such a function always exists. For example, we can take the convex hull of the graph of \(-\log u\text{.}\)

Consider the flow

\begin{equation*} \frac{\partial u}{\partial t} = \Delta u + \lambda _1 u \;\; \frac{u}{\partial \Omega} = 0\text{.} \end{equation*}

We assume that \(u_t \rightarrow u\text{,}\) the first eigenfunction. We are going to use the maximum principle. Let \(T\) be the biggest number such that \(\det (- \nabla ^2 \log u)\) is degenerated. Thus there is an \(x_0 \in M\) and a direction \(i\) such that

\begin{equation*} - ( \log u) _{ii} (x_0) = 0 \end{equation*}

and for other \(j, ( - \log u) _{jj} (x_0) \geq 0\text{.}\) Let \(\phi = - \log u\text{.}\) Then the envolution of \(\phi\) is

\begin{equation*} \frac{\partial \phi}{\partial t} = \Delta \phi - |\nabla \phi |^2 - \lambda _1\text{.} \end{equation*}

By the maximum principle, \(\phi _{ii} k = 0 , \frac{\partial \phi _{ii}}{\partial t} \leq 0 \; \Delta \phi _{ii} \geq 0\text{.}\) Thus

\begin{align*} 0 \amp \geq \amp \frac{\partial \phi _{ii}}{\partial t} = \Delta \phi_{ii} - 2 \phi _k \phi_{kii} - 2 \phi ^2_{kl}\\ \amp \geq \amp - 2 \phi _{ki}^2\text{.} \end{align*}

However by convexity, \(\phi_{ki}^2 - \phi _{ii} \phi_{kk}= 0, \phi_{ki} \equiv 0\text{.}\) The theorem follows from the strong maximum principle.
