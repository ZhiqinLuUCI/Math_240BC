Section 7.1 The Poincaré Lemma

In this section, we prove the following

Theorem 7.1.1.

Let \(U\) be the unit ball of the original point of \(\R^n\text{.}\) Let \(\omega\) be a \(p\)-form on \(U\) with \(p\geq 1\text{.}\) Then there is a \(p-1\) form \(\eta\) such that \(\omega=d\eta\text{.}\)

Proof.

If \(\omega\) is a \(1\)-form, the proof is straightforward. Let

\begin{equation*} f(x)=\int_{\ell}\omega\text{,} \end{equation*}

where \(\ell\) is a curve connecting the original point to \(x\text{.}\) Then

\begin{equation*} df=\omega\text{.} \end{equation*}

In general, we use the math induction. We write

\begin{equation*} \omega=\omega_1+dx_n\wedge\omega_2\text{,} \end{equation*}

where \(\omega_1,\omega_2\) are forms without the factor \(dx_n\text{.}\) Let

\begin{equation*} \omega=\sum a_{i_1\cdots i_{p-1}}(x_1,\cdots,x_n)dx_{i_1}\wedge\cdots\wedge dx_{i_{p-1}}\text{.} \end{equation*}

Define

\begin{equation*} \eta=\int a_{i_1\cdots i_{p-1}}(x_1,\cdots,x_n)dx_n\, dx_{i_1}\wedge\cdots\wedge dx_{i_{p-1}}\text{.} \end{equation*}

Then \(d(\omega-d\eta)=0\) and \(\omega-d\eta\) has not \(dx_n\) factor. As a result, the coefficients of \(\omega-d\eta\) are independent to \(x_n\text{.}\) Thus the theorem follows from the inductive assumption.

The following theorem is a nonlinear generalization of the above result:

Theorem 7.1.2.

Let \(U\) be the unit tube of the original point of \(\R^n\text{.}\) Let \(\omega\) is a skew-symmetric matrix valued \(1\)-form on \(U\text{.}\) Assume that

\begin{equation*} d\omega+\omega\wedge\omega=0\text{.} \end{equation*}

Then on \(U\text{,}\) the equation

\begin{equation*} dg+\omega g=0 \end{equation*}

is solvable.

Proof.

We assume that \(g(0)=1\text{.}\) Assume that for \(r>0\text{,}\) we can define \(g(x_1,\cdots,x_r,0,\cdots,0)\) such that \(dg+\omega g=0\) on \(\{x_{r+1}=\cdots x_n=0\}\text{.}\) We define \(g(x_1,\cdots, x_{r+1},0,\cdots,0)\) by the following ODE:

\begin{equation*} \left\{ \begin{array}{l} \frac{\pa g}{\pa x_{r+1}}+\omega_{r+1} g=0\\ g(x_1,\cdots,x_r,0,\cdots,0) \text{ were defined. } \end{array} \right. \end{equation*}

To complete the proof, we need to prove that for any \(k\leq r\text{,}\) we have

\begin{equation} \frac{\pa g}{\pa x_k}+\omega_k g=0\text{.}\tag{7.1.1} \end{equation}

Taking the derivative with respect to \(x_{r+1}\text{,}\) we get

\begin{equation*} \frac{\pa ^2 g}{\pa x_{r+1}\pa x_k}+\frac{\pa\omega_k}{\pa x_{r+1}}g+\omega_k\frac{\pa g}{\pa x_{r+1}}\text{.} \end{equation*}

Using the ODE, we get

\begin{equation*} -\frac{\omega_{r+1}}{\pa x_k}g-\omega_{r+1}\frac{\pa g}{\pa x_k}+\frac{\pa\omega_k}{\pa x_{r+1}}g+\omega_k\frac{\pa g}{\pa x_{r+1}}\text{.} \end{equation*}

Using the fact that \(d\omega+\omega\wedge\omega=0\text{,}\) we can prove that the expression in (7.1.1) satisfies the ODE with the zero initial value, hence must be zero.
