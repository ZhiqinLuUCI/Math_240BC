Section 6.1 Spectrum on complete non-compact manifolds

Unlike in the case of compact manifold, in general, a complete manifold doesn't admit any eigenvalues. For example, there are no \(L^2\)-eigenvalue on \(\mathbb{R}^n\text{.}\) That is, \(\forall \lambda \in \mathbb{R}\text{.}\) If

\begin{equation*} \Delta f + \lambda f = 0 \end{equation*}

and \(f \in L^2 (\mathbb{R}^n)\text{,}\) then \(f \equiv 0\text{.}\)

Escobar proved that if \(M\) has a rotational symmetric metric, then there is no \(L^2\)-eigenvalue. Let \(\Delta\) be the Laplace operator on a complete non-compact manifold \(M\text{.}\) By the argument before, \(\Delta\) naturally extends to a self-adjoint densely defined operator, which we still denote \(\Delta\) for the sake of simplicity.

It is well-known that there is a spectrum measure such that

\begin{equation*} \Delta = \int^\infty _0 \lambda d E \end{equation*}

(Here we assume \(\Delta\) is the geometric Laplacian, which is a positive operator).

Define \(e^{-\Delta t}\) for any \(t > 0\text{.}\) Obviously, it is a bounded operator. Thus by the Hahn-Banach \(e^{-\Delta t}\) is a bounded operator.

The heat kernel is defined as

\begin{equation*} e^{-\Delta t} f(x) = \int H(x, y, t) f (y) \, dy\text{.} \end{equation*}

The Green's function is defined as

\begin{equation*} G(x, y) = \int^\infty_0 H (x, y, t) \, dt\text{.} \end{equation*}

Of course, we need to prove the existence of these functions when \(M\) is complete non-compact.

The pure point spectrum of \(\Delta\) are those \(\lambda \in \mathbb{R}\) such that

  1. There exist a \(L^2\) function \(f \neq 0\) such that

    \begin{equation*} \Delta f + \lambda f = 0 \end{equation*}

  2. The multiplicity of \(\lambda\) is finite

  3. In a neighborhood of \(\lambda\text{,}\) it is the only spectrum point.

We define

\begin{equation*} \rho (\Delta ) = \left\{ y \in \mathbb{R} \left| \begin{array}{r} (\Delta - y ) ^{-1} \ \mbox{is a bounded } \\ \mbox{operator} \end{array} \right. \right\} \end{equation*}

\(\sigma (\Delta ) = \mathbb{R} - \rho (\Delta)\) is the spectrum set of \(\Delta\text{.}\) From the above discussion, \(\sigma (\Delta )\) decomposes \(\emptyset\) as the union of pure point spectrum, and the so-called essential spectrum, which is by definition, the complement of pure point spectrum.

Using the above definition, \(\lambda \in \sigma (\Delta )\) belongs to the set \(\sigma _{ess} ( \Delta )\text{,}\) if either

  1. \(\lambda\) is an eigenvalue of infinite multiplicity, or

  2. \(\lambda\) is the limiting point of \(\sigma (\Delta ) \text{.}\)

The following theorems in functional analysis characterizing the essential spectrum.

Theorem 6.1.1.

A necessary and sufficient condition for the interval \(( - \infty , \lambda )\) to intersect the essential spec of an self-adjoint densely defined operator \(A\) is that, for all \(\varepsilon > 0\text{,}\) there exists an infinite dimensional subspace \(G _\varepsilon \subset { Dom} (A)\text{,}\) for which \(( A f - \lambda f - \varepsilon f, f) \lt 0\text{.}\)

Theorem 6.1.2.

A necessary and sufficient condition for the interval \(( \lambda - \sigma , \lambda + \sigma )\) to intersect the essential spectrum of \(A\) is that there exists an infinite dimensional subspace \(G \lt { Dom} (A)\) for which \(\Vert A - \lambda I) f \Vert \lt \sigma\Vert f\Vert, f \in G\text{.}\)

For reference, see Donnelly, Topology 20, 1‚Äì14, 1981.

Using the above result, we give the following variational characterization of the lower bound of spectrum and the lower bound of essential spectrum.

Theorem 6.1.3.

Let

\begin{equation*} \lambda_0 = \inf_{f \in C^\infty_0 (M)} \frac{\int_M |\nabla f| ^2}{\int_M f ^2} \end{equation*}

and

\begin{equation*} \int_{ess} = \sup_k \inf_{f \in C^\infty_0 (M\backslash k)} \frac{\int_M |\nabla f| ^2}{\int_M f ^2} \end{equation*}

where \(k\) is a compact set running through an exhaustion of the manifold. Then \(\lambda _0\) and \(\lambda_{ess}\) are the lower bound of \(\sigma (\Delta )\) and \(\sigma _{ess} ( \Delta )\) respectively.

\begin{proof} We first prove the formula for \(\lambda _{ess}\text{.}\) Let

\begin{equation*} \lambda _{ess} ^\prime = \inf \sigma_{ess} ( \Delta ) \end{equation*}

is \(\sigma _{ess} (\Delta ) = \emptyset\text{,}\) we define, \(\lambda _{ess} ^\prime = + \infty\text{.}\) By the definition, \(\forall \epsilon > 0 , ( - \infty, \lambda ^\prime _{ess} + \varepsilon ) \cap \sigma _{ess} (\Delta ) \neq \emptyset\text{.}\) Then we can find infinite dimensional space \(V\)such that for any \(f \in V\)

\begin{equation} \langle \Delta f - ( \lambda _{ess} ^\prime + \varepsilon ) f , f\rangle \lt 0. \tag{$\star$}\tag{6.1.1} \end{equation}

Without loss of generality, we may assume that

\begin{equation*} \int_M f ^2 = 1\text{.} \end{equation*}

Also, without loss of generality, we may assume that all elements in \(V\) are smooth.

We leave as an exercise, to prove that (\(\star\)) implies

\begin{equation*} \int_M |\nabla f | ^2 \leq ( \lambda _{ess}^\prime + \varepsilon )\text{.} \end{equation*}

Now we assume that \(k\) is a compact set. Let \(k ^\prime\) be a larger ball containing \(k\text{.}\) Let \(\rho\) be the cut-off function such that \(\rho \equiv 1\) on \(k\) but \(\rho \equiv 0\) outside \(k^\prime\text{.}\) We claim (***) that \(\forall \varepsilon > 0\text{,}\) there is an \(f \in V\) with \(\int f^2 = 1\) but

\begin{equation*} \int _M \rho ^2 f^2 \lt \varepsilon\text{.} \end{equation*}

If the above is not true, then for any \(f \in V \text{.}\)

\begin{equation*} \int_M \rho^2 f ^2 \geq \varepsilon _0, \; \mbox{if} \; \int f ^2 = 1\text{.} \end{equation*}

Since the set \(f \in V\) is of infinite dimensional, the set \(\rho f\) is of infinite dimensional. Thus we can find an orthogonal basis

\begin{equation*} \int\rho^2 f _i f _j = 0 \;\; i \neq j \end{equation*}

while we can still keep \(\int f _i^2 = 1\text{.}\) We consider

\begin{align*} \int | \nabla (\phi f_i )| ^2 \amp \leq \amp 2 \int | \nabla \rho |^2 f^2_i + 2 \int \rho ^2 |\nabla f_i |^2\\ \amp \leq \amp 2 C + 2 (\lambda _{ess} ^\prime + \varepsilon ) \text{.} \end{align*}

Thus

\begin{equation*} \frac{\int \nabla ( \rho f_i )^2}{\int ( \rho f_i )^2 } \leq \frac{2C + 2 ( \lambda _{ess} ^\prime + \varepsilon }{\varepsilon _0} \end{equation*}

for infinitely dimensional space. This is a contradiction because on the compact set \(k ^\prime\) the eigenvalues go to infinity.

With the above preparation, we can prove our theorem \(\forall \varepsilon > 0\text{.}\) We find an \(f\) with \(\int f ^2 = 1\) but

\begin{equation*} \int \rho ^2 f ^2 \lt \varepsilon\text{.} \end{equation*}

Consider \(\rho_1 = 1 - \rho\)

\begin{align*} \int ( \nabla (\rho _1 f ) |^2 \amp = \amp \int \rho^2_1 | \nabla f |^2 + 2 \int \rho_1 f \nabla \rho _1 \nabla f\\ \amp \amp + \int f ^2 |\nabla \rho_1 |^2 \text{.} \end{align*}

Using the above inequality, we have

\begin{align*} \lefteqn{\int f ^2 |\nabla \rho_1 |^2 \leq C \varepsilon }\\ \amp \amp 2 \int \rho_1 f \nabla \rho _1 \nabla f = - \frac{1}{2} \int \Delta \rho^2 _1 \cdot f ^2\\ \amp \amp \leq C \varepsilon \end{align*}

because the supp of \(\nabla \rho_1, \Delta \rho^2_1\text{,}\) are within \(k^\prime\text{.}\) Thus

\begin{align*} \int |\nabla ( \rho _1 f ) |^2 \amp \leq \amp ( \lambda _{ess}^\prime + \varepsilon ) + C \varepsilon\\ \int \rho^2 _1 f ^2 \amp \geq \amp 1 - \varepsilon\text{.} \end{align*}

Using the definition, we get

\begin{equation*} \lambda _{ess} \leq \lambda _{ess} ^\prime + \varepsilon \end{equation*}

and thus \(\lambda _{ess} \leq \lambda _{ess} ^\prime\text{.}\) The other direction is easier to prove.

Using the same method, we can prove the case for \(\lambda _0\text{,}\) provided that we need a similar theorem like Theorem 1.

It is an interesting question to compute the set of essential. It is particularly interesting to get the lower bound estimate for the essential spectrum because of the following theorem.

Theorem 6.1.4.

Suppose \(\lambda _0 \lt \lambda _{ess}\text{,}\) then \(\lambda _ 0\) is an eigenvalue of \(M\) with finite dimensional eigenspace.

That is, there exists an \(L^2\) function \(f \neq 0\text{,}\) such that

\begin{equation*} \Delta f = - \lambda _0 f \end{equation*}

which is a very strong result.

In what follows we use \(\mathbb{R}^n\) to elaborate our theorem.

First let's compute \(\lambda _{ess}\) for \(\mathbb{R}^n\text{.}\) We claim that \(\lambda _{ess} = 0\text{.}\) By definition, this is equivalent to say that \(\forall \varepsilon > 0 \, \forall \, R > 0\text{,}\) there is a function \(f \in C^\infty_0 ( \mathbb{R} ^n - B (R))\) such that

\begin{equation*} \int |\nabla f |^2 \lt \varepsilon \int f ^2\text{.} \end{equation*}

This is obvious: let \(B (x_0, R ^\prime )\) be a ball of radius \(R^\prime\) with center \(x _0\) such that \(|x_0| > 2 R+ 1 + R^\prime\text{.}\) Then

\begin{equation*} B(x_0, R^\prime ) \subset \mathbb{R}^n \backslash (R)\text{.} \end{equation*}

Let \(f\) be the first Dirichlet eigenfunction of \(B (X_0, R^\prime )\text{.}\) Then if \(R^\prime \rightarrow \infty\)

\begin{equation*} \int |\nabla f |^2 /\int f ^2 \lt \varepsilon \end{equation*}

zero extending \(f\) to \(\mathbb{R} ^n \backslash B(R)\) we get the result.

Using Theorem 2, we can even prove that

\begin{equation*} \sigma _{ess} ( \Delta ) = [ 0, + \infty)\text{.} \end{equation*}

To prove the above, we make the following observation \(\forall \lambda > 0 , \forall m \in \mathbb{Z}\text{,}\) we can find a square of size \(m \bar{\pi} \sqrt{ \frac{\lambda }{n}}\) such that

\begin{equation*} f = \sin \sqrt{ \frac{\lambda }{n}} x _1 \cdot \sin \sqrt{ \frac{\lambda }{n}} x_2 \ldots \sin \sqrt{ \frac{\lambda }{n}} x_n \end{equation*}

is an eigenfunction with Dirichlet condition: \(\Delta f + \lambda f = 0\text{.}\)

However, we can't use Theorem 2 directly. The reason is that \(f\) is not second differentiable near the boundary. Thus we need to use cut-off function. Without loss of generality, we may assume that the square is in the first quadratic. We denote such a square to be \(s\text{.}\)

Let \(s _1\) be a square with the same center as \(s_1\) but with smaller size. We assume that the distance of the boundary of \(s_1\) to \(s\) is \(d\text{,}\) which is to be determined later.

We choose a cut-off function \(\rho\) such that \(\rho\equiv 1\) on \(s_1\) and \(\rho \equiv 0\) outside \(s\text{.}\) By the same argument as before, we can prove that

\begin{equation*} \Vert \Delta ( \rho f ) + \lambda \rho f \Vert _{L_{2}} \subset \varepsilon \Vert \rho f \Vert_{L_{2}}\text{.} \end{equation*}

Thus \(ess (\Delta ) = [ 0 , + \infty)\text{.}\)

Unfortunately, the above argument doesn't apply to the general case. Thus the following result of *** is very surprising and interesting.

Theorem 6.1.6. J-P. Wang.

Let \(M\) be a complete manifold with non-negative Ricci curvature. Then

\begin{equation*} ess (\Delta ) = [ 0, \infty) \end{equation*}

Note that \(\lambda _{ess}\) was known before, e.g., P.Li-Wang, Brooks.

In this lecture, we study the essential spectrum of a complete non-compact manifold with non-negative Ricci curvature. We are going to prove that the set of essential spectrum is \([ 0 , \infty)\text{.}\) The reference papers for this lecture are

  1. Sturm, J. Funct. Anal. 118, 442‚Äì453, 1993

  2. Wang, Math. Research Letters. 4, 473‚Äì479, 1997

Through this lecture, \(M\) is a complete Riemannian manifold with non-negative Ricci curvature. We say the volume \((M, g)\) grows uniformly sub-exponentially, if for any \(\varepsilon > 0\text{,}\) there is a constant \(C \lt \infty\) such that for all \(r > 0\) and all \(x \in M\)

\begin{equation*} v (B_r (x)) \leq C e ^{\varepsilon r} v (B, (x)) \end{equation*}

Theorem 6.1.7.

If the volume of \((M, g)\) grow uniformly sub exponentially, then the spectrum \(\sigma (\Delta p )\) of \(\Delta p\) acting on \(L^p (M)\) is independent of \(p \in [1, \infty)\text{.}\) In particular, it is a subset of the real line.

Note that by the Bishop volume comparison theorem, Ricci non-negative implies uniformly sub-exponentially volume growth.

The theorem can be proved using the resolvent estimates, which are based on the previous heat kernel estimate.

We begin with the following.

Lemma 6.1.8.

If the volume of \((M, g)\) grows uniformly subexponentially, then for any \(\varepsilon > 0\)

\begin{equation*} \sup_{x \in M} \int_M e ^{-\varepsilon d (x, y)} (v (B_1 (x))) ^{-\frac{1}{2}} ( v (B _1 (y))) ^{-\frac{1}{2}} d \nu (y) \lt \infty\text{.} \end{equation*}

\begin{proof} We take \(r = d (x, y)\text{.}\) Then since

\begin{equation*} B_1 (y) \subset B_{r+1} (x) \end{equation*}

we must have

\begin{equation*} \nu (B_1 (y)) ^{-\frac{1}{2}} \geq \nu (B_{r+1} (x)) ^{-\frac{1}{2}} \geq C e ^{-\frac{1}{2}(r+1)}\nu (B_1(x)) ^{-\frac{1}{2}} \end{equation*}

for any \(x\text{.}\) Thus the integration in the lemma if less than

\begin{equation*} C \int_M e^{-\varepsilon r} \cdot e ^{\frac{1}{2}\varepsilon (r + 1)} \nu (B_1(x)) ^{-1} dy\text{.} \end{equation*}

We let \(f (r) = { vol} (\partial B_r (x))\) and \(F (r) = \int^r_0 f(t) \, dt\text{.}\) Then up to a constant, the above expression is less than

\begin{equation*} (\nu (B_1 (x)) ) ^{-1} \int^\infty_0 e ^{-\frac{1}{2}\varepsilon r} f (f) \, dr\text{.} \end{equation*}

By volume comparison again \(f(r) \leq c r ^{n-1} \nu (B _1 (x)) \text{.}\) The lemma follows.

Lemma 6.1.9.

For any \(\beta > 0\text{,}\) there is an \(n \in N, \alpha \lt 0 , c \lt \infty\) such that the integral kernel \(g _\alpha ^{(\frac{n}{2})} (x, y)\) of \((\Delta - \alpha) ^{-\frac{n}{2}}\) exists and satisfies

\begin{equation*} g_\alpha ^{\frac{n}{2}} (x, y) \leq C e ^{-\beta d (x, y)} \phi(x)^2 \end{equation*}

where \(\phi (x) = ( \nu (B_1 (x))) ^{-\frac{1}{2}}\text{.}\)

The lemma was proved on page 67, using the heat kernel estimates.

Before going further, let's make some remarks on the kernel of an operator. Let \(A\) be an operator on functions. If there is a function \(g(x, y)\) such that

\begin{equation*} A f (x) = \int_M g (x, y) f (y) \, dy \end{equation*}

then we call \(g (x, y)\) the ‚Äúkernel‚Äù of \(A\text{.}\) However, in general, the kernel doesn't exist.

To see why the kernel in general does not exist, we let \(\xi \in \rho (\Delta )\text{.}\) To be more specific, \(\Delta\) is an operator on \(L^2 (M)\) so we assume that \(\xi \in \rho (\Delta_2)\text{.}\) The operator \(( \Delta _2 - \xi) ^{-1}\) is called the resolvent. It is a bounded operator form \(L^2(M) \rightarrow L^2 (M)\text{.}\) (By Hahn-Banach theorem, it can be extended to whole \(L^2(M)\text{.}\) However, in general, the kernel doesn't exist: if not, we let \(f_i \rightarrow \delta\) be a sequence converges to the \(\delta\)-function in \(L^1(M)\) then \(\Delta _2f_i\) could have been bounded. We need an estimate to extend \(\Delta _2\) from \(L^2 (M)\) to \(L^1(M)\text{.}\)

Lemma 2 told us that for \(\alpha \lt 0 , (\Delta - \alpha ) ^{- \frac{n}{2}}\) has a kernel. The operator can be extended to \(L^1(M)\text{.}\) Further more, the kernel exponentially decays.

The Laplacian we used here is the geometric Laplacian. That is, it is a positive operator.

For our purpose, we just need to prove \(\sigma (\Delta _1) \subset \sigma (\Delta _2)\text{,}\) which is also the major part of the paper of Sturm.

Recall that \(x \in (A)\text{,}\) if, \((A - x I ) ^{-1}\) is a bounded operator. We define the spectrum \(\sigma (A) = \mathbb{R} - \rho (A)\text{.}\) Note that \(\sigma (\Delta _2) \subset [ 0 , \infty)\text{.}\) Then for \(\alpha \lt 0 , (\Delta _2 - \alpha I) ^{-1}\) is bounded. Since

\begin{equation*} \Delta - \alpha I = (\Delta - \zeta I) ( 1 - (\alpha - \zeta ) ( \Delta - \zeta I)^{-1})\text{.} \end{equation*}

we have

\begin{equation*} (\Delta - \zeta I ) ^{-1} = ( 1 - (\alpha - \zeta ) ( \Delta - \zeta )^{-1} ( \Delta - \alpha I)^{-1}\text{.} \end{equation*}

or for any \(n\)

\begin{equation*} ( \Delta - \zeta I ) ^{-n} = ( 1 - (\alpha - \zeta ) ( \Delta - \zeta )^{-1})^n (\Delta - \alpha I) ^{-n}\text{.} \end{equation*}

From the above identity, the kernel for \((\Delta - \zeta I) ^{-n}\) exists. Let

\begin{equation*} A = ( 1 - ( \alpha - \zeta )( \Delta - \zeta )^{-1} )^n\text{.} \end{equation*}

Then

\begin{equation*} A _x ( g _{\alpha} ^{(\frac{n}{2})} (x, y ) \end{equation*}

is a kernel of \((\Delta - \zeta I) ^{-n}\text{.}\)

Lemma 6.1.10.

Let \(g (x, y)\) be the kernel of \((\Delta - \zeta I) ^{-n}\text{.}\) Then

\begin{equation*} |g (x, y)|\leq C e ^{- \varepsilon d (x, y)} \phi (x) \phi (y) \end{equation*}

where \(\phi (x) = v (B _1 (x)) ^{-\frac{1}{2}}\text{.}\)

We omit the proof of the above lemma. By Lemma 1,

\begin{equation*} \sup_x \int | g (x, y) \\, dy \lt + \infty\text{.} \end{equation*}

Thus \((\Delta _2 - \zeta )^{-n}\) is an operator from \(L^1 \rightarrow L^1\text{.}\)

Since \(\Delta _2\) and \(\Delta _1\) are the same acting on \(C^\infty\) functions, \((\Delta _1 - \zeta )^{-n}\) is a bounded operator from \(L^1 (M) \rightarrow L^1 (M)\text{.}\)

Lemma 6.1.11.

If \(( \Delta_1 - \zeta) ^{-n}\) is a bounded operator, then \(\zeta \in \rho (\Delta _1)\text{.}\)

\begin{proof} Since \(( \Delta _1 - \zeta )^{-n}\) is bounded, there is a neighborhood of \(\zeta\) such that for any \(\zeta^\prime\) in the neighborhood, \((\Delta _1 - \zeta ^\prime )^{-n}\) is also bounded. By

\begin{equation*} ( \Delta _1 - \zeta )^{-1} = ( \Delta _1 - \zeta^\prime )^{-1} ( 1 - ( \zeta - \zeta ^\prime )( \Delta _1 - \zeta^\prime )^{-1}) ^{-1} \end{equation*}

and the fact that the latter can be expended to a convergent series, \(( \Delta _1 - \zeta )^{-1}\) is bounded in \(L^\prime (M)\text{.}\) Thus \(\zeta \in \rho (\Delta _1)\text{.}\)

From the above lemma, we have

\begin{equation*} \sigma (\Delta _1) \subset \sigma (\Delta _2 ) \subset [ 0, \infty)\text{.} \end{equation*}

The theorem thus follows from

Theorem 6.1.12. Wang.

Let \(M\) be a complete Riemannian manifold with non-negative Ricci curvature. Then

\begin{equation*} \sigma (\Delta_1) = [0, \infty)\text{.} \end{equation*}

Before giving the proof, we use the following theorem.

Theorem 6.1.13.

Let \(M\) be a complete non-compact Riemannian manifold. Ric\((M) \geq 0\text{.}\) Then

\begin{equation*} { vol} B_p (R) \geq C (n , { vol} B_p (1) ) R\text{.} \end{equation*}

\begin{proof} Fixing \(x_0 \in \partial B_p (R)\text{.}\) Using the comparison theorem, we have

\begin{equation*} \Delta p^2 \leq 2 n\text{.} \end{equation*}

For any \(\phi \in C^\infty _0 (M), \phi \geq 0\text{,}\) we have

\begin{equation*} \int _M \phi \Delta \rho^2 \leq 2 n \int_M \phi\text{.} \end{equation*}

We choose a standard cut-off function \(\phi = \psi ( \rho (x))\text{,}\) where

\begin{equation*} \psi (t) = \left\{ \begin{array}{ll} 1 \amp 0 \leq t \leq R - 1\\ \frac{1}{2} (R+1 - t), \amp R-1 \leq t \leq R+1 \\ 0 \amp t \geq R + 1 \end{array} \right.\text{.} \end{equation*}

By Stokes theorem

\begin{align*} \int_M \phi \Delta \rho^2 \amp = \amp - \int \nabla \phi \nabla \rho^2 = - 2 \int \psi ^\prime \rho |\nabla \rho| ^2\\ \amp = \amp \int_{Bx_{0} (R + 1) \backslash B x_{0} (R-1) } \rho\\ \amp \geq \amp (R - 1) { vol} ( x_0 (R+1) \backslash Bx_0 (R-1)) \text{.} \end{align*}

Thus

\begin{equation*} (R-1) { vol} (B x_{0} (R+1)- B x_{0} (R-1) ) \leq 2 n \int \phi \leq 2 n \, { vol} B x_{0} (R+1) \end{equation*}

Obviously

\begin{equation*} B_p (1) \subset B x_{0} (R+1) \backslash B x_{0} (R-1)\text{.} \end{equation*}
