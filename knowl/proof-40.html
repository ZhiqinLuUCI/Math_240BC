Proof

Let \(u\) be the first eigenfunction such that

\begin{equation*} \max u^2 = 1\text{.} \end{equation*}

Let

\begin{equation*} g(x) = \frac{1}{2}(|\nabla u|^2 + (\lambda_1 + \varepsilon)u^2)\text{,} \end{equation*}

where \(\varepsilon > 0\text{.}\)

The function \(g\) is a smooth function. Let \(x_0\) be the maximal point of \(g\text{.}\) Then at \(x_0\) we have

\begin{equation} u_j u_{ji} + (\lambda_1 + \varepsilon) u u_i = 0\tag{5.3.1} \end{equation}

and

\begin{equation*} 0 \geqslant u_{ji}^2 + u_j u_{jii} + (\lambda_1 + \varepsilon) |\nabla u|^2 + (\lambda_1 + \varepsilon)u\Delta u\text{.} \end{equation*}

Using the Ricci identity, we have

\begin{equation*} u_j u_{jii} = u_j(\Delta u)_j + Ric(\nabla u) \geqslant u_j(\Delta u)_j\text{.} \end{equation*}

Thus we have

\begin{equation} 0 \geqslant u_{ji}^2 + u_j(\Delta u)_j + (\lambda_1 + \varepsilon) |\nabla u|^2 + (\lambda_1 + \varepsilon)u\Delta u\text{.}\tag{5.3.2} \end{equation}

Suppose that at the maximum point of \(g(x)\text{,}\) \(\nabla u \neq 0\text{.}\) Then we have

\begin{equation*} u_{ji}^2 \geqslant |\nabla u|^{-4}(\sum_{i,j}u_j u_i u_{ij})^2 \end{equation*}

by the Cauchy inequality. Using the first order condition, we conclude

\begin{equation*} u_{ji}^2 \geqslant (\lambda_1 + \varepsilon)^2 u^2\text{.} \end{equation*}

Putting the above inequality into \(\circledast\text{,}\) we get

\begin{equation*} 0 \geqslant \varepsilon|\nabla u|^2 + \varepsilon(\lambda_1 + \varepsilon)u^2 \end{equation*}

which is not possible. Thus at the maximum point, we must have \(\nabla u = 0\text{.}\) Therefore we have

\begin{equation*} g(x) \leqslant \frac{1}{2}(\lambda_1 + \varepsilon)\max u^2 = \frac{1}{2} (\lambda_1 + \varepsilon)\text{.} \end{equation*}

From the above estimate, we get

\begin{equation*} \frac{|\nabla u|^2}{1-u^2} \leqslant \lambda_1 + \varepsilon\text{.} \end{equation*}

Since \(\varepsilon\) is arbitrary, we let it go to zero and obtain

\begin{equation*} \frac{|\nabla u|^2}{1 - u^2} \leqslant \lambda_1\text{.} \end{equation*}

By changing the sign of \(u\text{,}\) we may assume that \(\max u = 1\text{.}\) Let \(u(p) = 1\) for \(p\in M\text{.}\) Since \(\int u = 0\text{,}\) there is a point \(q\in M\) such that \(u(q)=0\text{.}\) Let \(\sigma(t)\) be the minimal geodesic line connecting \(q\) and \(p\text{.}\) Consider the function

\begin{equation*} \arcsin u(\sigma(t)) \end{equation*}

By the above inequality, we get

\begin{equation*} |(\arcsin u(\sigma(t)))'| \leqslant \sqrt{\lambda_1}|\sigma '(t)| \end{equation*}

Integrating the above inequality along the geodesic line, we get

\begin{equation*} \frac{\pi}{2} \leqslant \sqrt{\lambda_1}d \end{equation*}

and the theorem is proved.
