The problem is that in general, we don't know whether the first eigenfunction is always symmetric. More precisely, if we assume that

\begin{equation*} 1 = \sup u > \inf u = -k \geqslant -1 \end{equation*}

we don't know whether \(k = 1\text{.}\) From Li-Yau's basic estimate, we can improve the estimate \(\lambda_1 \geqslant \pi^2/4d^2\) to

\begin{equation*} \lambda_1 \geqslant (\frac{\pi}{2} + \arcsin k)^2 d^{-2}\text{.} \end{equation*}
