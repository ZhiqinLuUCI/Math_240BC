We prove the formula for \(\lambda_{ess}\text{.}\) The formula for \(\lambda_0\) is similar. Let

\begin{equation*} \lambda'_{ess}=\inf \sigma_{ess}(\Delta)\text{.} \end{equation*}

If \(\sigma_{ess}(\Delta)=\emptyset\text{,}\) we define \(\lambda'_{ess}=+\infty\text{.}\) When \(\lambda'_{ess}=+\infty\text{,}\) we prove that \(\lambda_{ess}\) is also infinity.

Assume not, then there is a constant \(C\text{,}\) such that for any compact set \(K\text{,}\) there is a function \(f\in \mathcal C_0^\infty(M\backslash K)\text{,}\) we have

\begin{equation*} \frac{\int_M |\nabla f|^2}{\int_M f^2}\lt C\text{.} \end{equation*}

By inductively choosing \(f\text{,}\) we can make sure that the support of these functions are disjoint. Thus the functions span an infinite dimensional space \(G\) with

\begin{equation*} (-\Delta f-Cf,f)\lt 0\text{,} \end{equation*}

which contradicts to the fact that the essential spectrum is an empty set.

The proof of the general case is similar. We fist prove that \(\lambda_{ess}\leq\lambda_{ess}'\) By definition, for \(\eps>0\text{,}\)

\begin{equation*} (-\infty,\lambda_{ess}'+\eps)\cap \sigma_{ess}(\Delta)\neq\emptyset\text{.} \end{equation*}

Therefore we can find infinite dimensional space \(V\) such that for any \(f\in V\)

\begin{equation*} (-\Delta f-(\lambda_{ess}'+\eps) f, f)\lt 0\text{.} \end{equation*}

In what follows, we shall prove that, using the cut-off functions, there exist infinitely many elements in \(V\) such that their supports are disjoint.

We assume that \(K\) is a compact set. Let \(K'\) be a large ball containing \(K\text{.}\) Let \(\rho\) be the cut-off function such that \(\rho=1\) on \(K\) but \(\rho=0\) outside \(K'\text{.}\) We claim that for any \(\eps>0\text{,}\) there is an \(f\in V\) with \(\int f^2=1\) but

\begin{equation*} \int_M\rho^2f^2\lt \eps\text{.} \end{equation*}

If the above is not true, then there is an \(\eps_0>0\) such that for any \(f\in V\text{,}\)

\begin{equation*} \int_M\rho^2f^2\geq\eps_0\text{,} \end{equation*}

Since the set \(f\in V\) is of infinite dimensional, the set \(\rho f\) is of infinite dimensional as well, otherwise the above inequality is not valid. Thus we can find an orthogonal basis

\begin{equation*} \int_M\rho^2f_if_j=0\text{,} \end{equation*}

if \(i\neq j\text{,}\) while we still keeping \(\int_Mf_i^2=1\text{.}\) We consider

\begin{equation*} \int_M|\nabla(\rho f_i)|^2\leq 2\int_M|\nabla \rho|^2 f_i^2+2\int_M|\nabla f_i|^2\text{.} \end{equation*}

By \(\int_M|\nabla f_i|^2\leq \lambda'_{ess}+\eps\text{,}\) we have Thus we have

\begin{equation*} \frac{\int_M|\nabla(\rho f_i)|^2}{\int_M(\rho f_i)^2} \leq \frac{2C+2(\lambda_{ess}'+\eps)}{\eps_0} \end{equation*}

for any \(i\text{.}\) This is a contradiction because on the compact set \(K'\text{,}\) the are only finitely many eigenvalues (counting multiplicity) below a fixed number.

Now we can prove our theorem. For all \(\eps>0\text{.}\) We can find an \(f\) with \(\int_M f^2=1\) but

\begin{equation*} \int_M\rho^2f^2\lt \eps\text{.} \end{equation*}

We take \(K=B(r)\) and \(K'=B(R+1)\) of balls of radius \(r, R+1\text{,}\) respectively. Both \(r,R\) are large numbers. Let \(\tilde\rho\) be a cut-off function such that

\begin{equation*} \{\tilde\rho=1\}\subset B(R)\text{.} \end{equation*}

We also assume that \(|\nabla\tilde\rho|\leq C\text{.}\) Let \(\rho_1=1-\tilde\rho\text{.}\) Then we have

\begin{equation*} { supp}\,\rho_1\cap K=\emptyset\text{.} \end{equation*}

We have

\begin{equation*} \int_M|\nabla(\rho_1f)|^2=\int_M\rho_1^2|\nabla f|^2+2\int_M\rho_1f\nabla\rho_1\nabla f+\int_M f^2|\nabla \rho_1|^2\text{.} \end{equation*}

From

\begin{equation*} \int_M f^2|\nabla \rho_1|^2\leq C\eps\text{,} \end{equation*}

and

\begin{equation*} 2\int_M\rho_1 f\nabla \rho_1\nabla f\leq 2 \sqrt{\int_M |\nabla(\rho_1 f)|^2}\cdot \sqrt{\int_M |\nabla\rho_1|^2f^2}\leq C\sqrt\eps \sqrt{\int_M |\nabla(\rho_1 f)|^2}\text{,} \end{equation*}

we get

\begin{equation*} \int_M|\nabla (\rho_1f)|^2\leq\lambda_{ess}'+C\eps\text{.} \end{equation*}

Since

\begin{equation*} \int_M\rho_1^2f^2\geq 1-C\eps\text{,} \end{equation*}

we get

\begin{equation*} \lambda_{ess}\leq\frac{\lambda_{ess}'+C\eps}{1-C\eps}\text{.} \end{equation*}

and thus \(\lambda_{ess}\leq\lambda'_{ess}\text{.}\) The other direction is easier to prove.
