We shall establish

\begin{equation} \int_{B(1) } e ^{p |\omega |}\leq C \tag{$\star$}\tag{3.3.4} \end{equation}

for \(p > 0\) small enough. We first prove that

\begin{equation*} \int_{B(\sigma)} | \nabla \omega |^2 \leq C \end{equation*}

for some \(\sigma > 1\text{.}\) To see this, let \(\rho\) be the cut-off function whose support is within \(B(\bar{\sigma})\) for some \(\bar{\sigma} > \sigma\text{.}\) Since \(u\) is harmonic, we have

\begin{equation*} \int _{B(\bar{\sigma})} \nabla u \nabla ( u^{-1} \rho ^2) = 0\text{.} \end{equation*}
