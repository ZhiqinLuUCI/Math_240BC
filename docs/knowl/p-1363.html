Let \(M\) be a Riemannian manifold, then there is a heat kernel

\begin{equation*} H(x,y,t)\in C^\infty(M\times M\times\mathbb R^+)\text{,} \end{equation*}

such that

\begin{equation*} (T_t f)(x)=\int_M H(x,y,t) f(y)dy \end{equation*}

satisfying

  1. \(H(x,y,t)=H(y,x,t)\text{;}\)

  2. \(\underset{t\to 0^+}{\lim}\, H(x,y,t)=\delta_x(y)\text{;}\)

  3. \((\Delta-\frac{\pa}{\pa t})H=0\text{;}\)

  4. \(H(x,y,t)=\int_M H(x,z,t-s)H(z,y,s) dz\) for any \(0\lt s\leq t\text{.}\)
