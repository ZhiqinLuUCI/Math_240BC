Proof

By using the transformation

\begin{equation*} x\mapsto\frac{x-y}{2R}\text{,} \end{equation*}

we can assume that \(y=0\) and \(R=1/2\text{.}\) Taking

\begin{equation*} \eta(x)=(1-|x|^2)^\beta \end{equation*}

as the cut-off function, then we have

\begin{align} \begin{split} \amp D_i\eta=-2\beta x_i(1-|x|^2)^{\beta-1};\tag{9.3.3}\\ \amp D_{ij}\eta=-2\beta\delta_{ij}(1-|x|^2)^{\beta-1}+4\beta(\beta-1)x_ix_j(1-|x|^2)^{\beta-2};\notag\\ \amp L\eta=[2\beta\, \sum_{i=1}^na^{ii}(1-|x|^2)-4\beta(\beta-1)\sum_{i,j=1}^na^{ij}x_ix_j](1-|x|^2)^{\beta-2}. \end{split}\notag \end{align}

Let \(v=\eta u\text{.}\) Then we have

\begin{equation*} Lv=uL\eta+\eta Lu-2a^{ij} D_i\eta D_j u\text{.} \end{equation*}

On the contact set \(\Gamma_v\) defined in Definition 9.2.2, we have

\begin{equation*} |Dv|\leq\frac{v(x)}{{ dist}(x,\pa B_1)}\leq\frac{v(x)}{1-|x|} \end{equation*}

by Lemma 9.2.7. Then we have

\begin{equation*} |Du|\leq\frac1\eta|Dv-uD\eta|\leq\frac 1\eta\left[\frac{v(x)}{1-|x|}+u|D\eta|\right]\leq 2(1+\beta) \eta^{-1/\beta} u\text{.} \end{equation*}

Thus we have

\begin{equation*} Lv\leq\eta f+(16\beta^2+2n\beta)\Lambda\eta^{-2/\beta} v\leq C\lambda\eta^{-2/\beta} v+f \end{equation*}

almost everywhere on \(\Gamma_v\text{,}\) where \(C\) only depends on \(n,\beta,\gamma\text{.}\)

By Theroem 9.2.13, we have

\begin{align*} \amp \sup _{B_1} v\leq C\left(||\eta^{-2/\beta}v^+||_{L^n(B_1)}+\left|\left|\frac{f}{\lambda}\right|\right|_{L^n(B_1)}\right)\\ \amp \leq C\left((\sup v^+)^{1-2/\beta}||(u^+)^{2/\beta}||_{L^n(B_1)}+\left|\left|\frac{f}{\lambda}\right|\right|_{L^n(B_1)}\right)\text{.} \end{align*}

Using the Young's inequality, we have

\begin{equation*} \sup_{B_1} v\leq C \left|\left|\frac{f}{\lambda}\right|\right|_{L^n(B_1)}+\eps\sup_{B_1} v^++C_\eps||(u^+)^{2/\beta}||^{\beta/2}_{L^n(B_1)}\text{.} \end{equation*}

We take \(\eps=1/2\) and if \(p\lt n\text{,}\) we let \(\beta=2n/p>2\text{.}\) Then we have

\begin{equation*} \sup_{B_{1/2}} u\leq C\left( ||u^+||_{L^p(B_1)}+\left|\left|\frac{f}{\lambda}\right|\right|_{L^n(B_1)}\right)\text{.} \end{equation*}

If \(p\geq n\text{,}\) then the estimate follows from the Hölder inequality.
