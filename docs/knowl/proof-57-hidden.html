Let \(N\) be any finite cover of \(M\text{.}\) Let \(\lambda\) be an eigenvalue of \(N\text{.}\) Then

\begin{equation*} \lambda\in\sigma_{ess}(\widetilde M) \end{equation*}

In fact, let \(\rho\) be a cut-off function. Since

\begin{equation*} \Delta f + \lambda f = 0 \mbox{ on } N \end{equation*}

Then on \(\widetilde M\)

\begin{equation*} \|\Delta(\rho f) + \lambda\rho f\|_{L^2} \leqq \|f\Delta\rho\|_{L^2} + \|\alpha\nabla\rho\nabla f\|_{L^2} \end{equation*}

If \(|\nabla\rho|\text{,}\) \(|\Delta\rho|\) are small, then

\begin{equation*} \|\Delta(f\rho) + \lambda\rho f\|_{L^2} \leqslant \varepsilon \|f\rho\|_{L^2} \end{equation*}

If the result is not true, since \(\sigma_{ess}(\Delta)\) is a closed set, there is an interval \((a,b)\) such that for any \(N\text{,}\) there is no eigenvalues in \((a,b)\text{.}\)

We prove this by contradiction. Let \(\lambda\text{,}\) \(\mu\) be two consecutive eigenvalues such that \(\lambda \lt a\) and \(\mu > b\text{.}\) By the above argument, we can find a \(C_0^\infty\) function on \(\widetilde M\) such that

\begin{align*} \nonumber \|\Delta f + \lambda f\|_{L^2} \amp \lt \amp \varepsilon\|f\|_{L^2}\\ \nonumber \|\Delta g + \mu g\|_{L^2} \amp \lt \amp \varepsilon\|g\|_{L^2} \end{align*}

Let \(k,l\) be integers such that

\begin{equation*} \frac{k\int|\nabla f|^2 + l\int|\nabla g|^2}{k\int f^2 + l\int g^2}\in(a,b) \end{equation*}

Then by repeating \(f\) \(k\)-times and \(g\) \(l\)-times we are done.
