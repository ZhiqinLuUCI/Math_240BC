smooth, \(0 \leq \rho \leq 1, \rho (t) = 1\) for \(0 \leq t \leq 1\text{.}\) \(\rho (t) = 0\) for \(t > 2\) and \(\rho^\prime (t) \leq 0\) for all \(t \in \mathbb{R}\text{.}\) Let \(R\) be a large number to be determined later.
