A manifold is called orientable, if there is a cover \(\{U_j\}\) on \(M\) such that if \(x_j^1,\cdots,x_j^n\) are the coordinates on \(U_j\text{,}\) then we have

\begin{equation*} \det\frac{\pa (x_j^1,\cdots,x_j^n)}{\pa(x_i^1,\cdots,x_i^n)}>0 \end{equation*}

for any \(U_i\cap U_j\neq\emptyset\text{.}\) Such a set of coordinate charts is called an orientation of the manifold.
