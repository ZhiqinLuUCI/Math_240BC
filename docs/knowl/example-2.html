Example 1.2.10.

Define

\begin{equation*} S^n=\{(x_1,\cdots,x_{n+1})\mid \sum x_k^2=1\} \end{equation*}

and the Riemannian metric to be the inherited metric

\begin{equation*} ds^2=dx_1^2+\cdots +dx_n^2+\frac{x_ix_j}{x^2_{n+1}}\, dx_i\,dx_j\text{.} \end{equation*}

Define

\begin{equation*} \omega_i=dx_i-\frac{x_i}{1+x_{n+1}}\,dx_{n+1}\text{.} \end{equation*}

Then

\begin{equation*} \omega_{ij}=\frac{1}{1+x_{n+1}}(x_i\,dx_j-x_j\,dx_i)\text{.} \end{equation*}

Thus we have

\begin{equation*} d\omega_j+\omega_i\wedge\omega_{ij}=0\text{,} \end{equation*}

and

\begin{equation*} d\omega_{ij}+\omega_{is}\wedge\omega_{sj}=\omega_i\wedge\omega_j\text{.} \end{equation*}

Thus the sectional curvature is \(1\text{.}\)
