Proof

For \(\phi\in\Lambda^p(M)\text{,}\) we define a linear functional

\begin{equation*} l(\psi)=(\phi,\psi) \end{equation*}

for \(\psi\in\Lambda^p(M)\text{.}\) With respect to the inner product \(\mathfrak D(\phi,\psi)\text{,}\) \(l\) is bounded:

\begin{equation*} |l(\psi)|\leq ||\phi||_0\sqrt{\mathfrak D(\psi,\psi)}\text{.} \end{equation*}

Thus by the Riesz representation theorem, there is an \(\eta\) such that

  1. \((\phi,\psi)=(\eta, (1+\Delta)\psi)\) for any \(\psi\in\Gamma(X,\mathcal A^{p,q}(E))\text{;}\)

  2. \(||\eta||_1^2\leq C\mathfrak D(\eta,\eta)\lt +\infty\text{.}\) \text{ (G\aa rding inequality) }

Let \(A: \Lambda^p(M)\rightarrow\Lambda^p(M)\) be the linear operator defined by sending \(\phi\in\Lambda^p(M)\) to the unique \(\eta\) defined above. Since \(A\) is a bounded operator, it can be extended to an operator on \(L^2(M)\text{.}\) Then since \(\eta\) is actually in \(\mathcal H^{p}(M)\text{,}\) the operator \(A\) is a compact operator by the Rellich Lemma. On the other hand, it is not hard to see that \(A\) is a self-adjoint operator\footnote {Since \(A\) is a bounded operator, in order to prove that \(A\) is self-adjoint, we just need to verify the equality \((A\phi,\psi)=(\phi,A\psi)\) for smooth \(\phi\) and \(\psi\text{,}\) which follows easily from integration by parts.}.

According to the spectral theorem for compact, self-adjoint operators, there is a Hilbert-space decomposition

\begin{equation*} \Lambda^p(M)=\bigoplus_m E(\rho_m)\text{,} \end{equation*}

where \(\rho_m\) are the eigenvalues of \(A\) and \(E(\rho_m)\) are the finite-dimensional eigenspaces corresponding to the eigenvalue \(\rho_m\text{.}\) Since \(A\) is one-to-one, all \(\rho_m\neq 0\text{.}\)

We claim that

\begin{equation*} \mathcal H^p(M)=E(1)\text{,} \end{equation*}

where \(\mathcal H^{p}(M)\) is the space of harmonic forms. To see this, we assume that \(\phi\in\mathcal H^{p}(M)\text{,}\) and \(A\phi=\phi\text{.}\) By the Weitzenb√∂ck formula, \(\Delta\) is an elliptic operator. Using the Sobolev Lemma and Schauder estimate, we can prove that \(\phi\) is smooth. Thus \((1+\Delta)\phi=\phi\) and \(\phi\) is harmonic. Since \(E(1)\) is of finite dimensional, we have

\begin{equation*} \dim\,\mathcal H^{p}(M)\lt +\infty\text{.} \end{equation*}

This proves the first assertion of the theorem. To prove the second assertion of the theorem, we note that the biggest possible eigenvalue of \(A\) is \(1\text{.}\) By the property of compact operators, there is a gap between the eigenvalue \(1\) and the rest of the eigenvalues. Thus if \(\phi\in\mathcal H^{p}(M)^\perp\) and if \(\phi\) is smooth, we have

\begin{equation*} ||\Delta\phi||_0\geq\eps||\phi||_0 \end{equation*}

for some \(\eps>0\text{.}\) Thus we can define the inverse operator \(G\) on \(\mathcal H^{p}(M)^\perp\text{,}\) called the Green's operator, such that \(G=\Delta^{-1}|_{\mathcal H^{p}(M)^\perp}\text{.}\)

If we compare \(G\) to the operator \(A\) (they have the same eigenspaces), we shall see that \(G\) is also a compact operator because \(||G||\leq C||A||\) for some constant \(C\text{.}\) Furthermore, by the elliptic regularity, \(G\) maps smooth \(p\)-forms to smooth \(p\) forms. Using this fact, the Hodge decomposition is given by

\begin{equation*} \phi=\eta_0+d\delta G\phi+\delta dG\phi\text{.} \end{equation*}

This completes the proof of the Hodge Theorem.
