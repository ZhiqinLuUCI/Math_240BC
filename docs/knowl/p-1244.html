To complete the proof, we need to prove that the above coupling is non-degenerated. What we need to prove is that, for any \(p\)-form \(\omega\) such that \(d\omega=0\) and for any \(p\)-dimensional submanifold \(K\) without boundary \(\int_K\omega=0\text{,}\) then there is a \(p-1\) form \(\eta\) such that \(\omega=d\eta\text{.}\) Let \(U,V\) be two open sets and suppose that the theorem is proved for \(U,V\text{.}\) Let \(\omega=d\eta_1\) on \(U\) and \(\omega=d\eta_2\) on \(V\text{.}\) Then on \(U\cap V\text{,}\) \(d(\eta_1-\eta_2)=0\text{.}\) We need to prove that for any \(p-1\) dimensional submanifold \(L\) in \(U\cap V\text{,}\) if \(\int _L(\eta_1-\eta_2)=0\text{,}\) then there is a \(\xi\) such that \(\eta_1-\eta_2=d\xi\text{.}\) Thus \(\eta_1-d\xi=\eta_2\) provides the required solution.
