We assume that \(U(x,y,t)\) be a function on \(M\times M\times\mathbb R^+\) such that

\begin{equation*} \limt U(x,y,t)=\delta_x(y) \end{equation*}

be the the Dirac function. Let

\begin{equation*} R(x,y,t)=\frac{d U(x,y,t)}{dt}-\Delta_x U(x,y,t)\text{,} \end{equation*}

where \(\Delta_x\) means the Laplace operator for the variable \(x\text{.}\) Note that formally, any function \(g(x,y,t)\) defines one-parameter family of operators \(G_t\) by the formula

\begin{equation*} G_t f(x)=\int_M g(x,y,t) f(y) dy\text{.} \end{equation*}
