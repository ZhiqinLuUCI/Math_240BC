To prove the above, we make the following observation \(\forall \lambda > 0 , \forall m \in \mathbb{Z}\text{,}\) we can find a square of size \(m \bar{\pi} \sqrt{ \frac{\lambda }{n}}\) such that

\begin{equation*} f = \sin \sqrt{ \frac{\lambda }{n}} x _1 \cdot \sin \sqrt{ \frac{\lambda }{n}} x_2 \ldots \sin \sqrt{ \frac{\lambda }{n}} x_n \end{equation*}

is an eigenfunction with Dirichlet condition: \(\Delta f + \lambda f = 0\text{.}\)
