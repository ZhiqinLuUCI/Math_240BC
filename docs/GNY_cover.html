Section 10.2 The Grigor'yan-Netrusov-Yau cover

In this section, we give a simplified proof of (CITE GNY) {Theorem 3.5}.

Theorem 10.2.1.

Let a pseudo-metric space \((X,d)\) satisfy \((2,P)\)-covering property. Let \(m\) be a Borel measure on \(X\text{,}\) and assume that there exist a positive real number \(v\) and \(\rho\) such that

\begin{equation} \forall x\in X, m(B(x,\rho/2))\leq v, and \exists x_0\in X, m(B(x_0,\rho))>v\text{.}\tag{10.2.1} \end{equation}

Then, for any \(\lambda>1\text{,}\) there exists a family \(\mathfrak A\) of \([\eps_0m(X)/v]\) annuli in \(X\) satisfying the following properties

  1. \(m(A)\geq v\) for any \(A\in\mathfrak A\text{;}\)

  2. the annuli \(\{\lambda A\}_{A\in\mathfrak A}\) are disjoint.

Here \(\eps_0\) is a number independent to \(v\text{.}\)

Let \(N\geq 1\) be the maximum number such that there exists a family of annuli \(\{A_i=B(x_i, r_i, R_i)\}\) with

  1. \(2C_3v>m(A_i)>C_3v\text{;}\)

  2. \(\{\lambda A_i\}\) disjoint;

  3. \begin{equation*} m(\bigcup_{i=1}^NB(x_i, 2\lambda^2 R_i))\lt C_1Nv \end{equation*}

    for a constant \(C_1>0\) independent to \(v\text{.}\)

Here \(C_3\) is the constant such that when the radius is multiple by \(8\text{,}\) then the volume is increased by \(C_3\) times.

By¬†(10.2.1), we know that the family of annuli exists at least for \(N=1\text{.}\)

We shall get a contradiction by assuming that

\begin{equation} N\lt \eps_0 m(X)/v\text{.}\tag{10.2.2} \end{equation}

Definition 10.2.2.

We say a subset

\begin{equation*} x_{i_1},\cdots, x_{i_p} \end{equation*}

of \(\{x_1,\cdots, x_N\}\) is \red{admissible}, if there are positive numbers \(T_{i_j}>0\) such that

  1. \begin{equation*} \bigcup_{j=1}^{p} B(x_{i_j}, \frac{1}{2\lambda}T_{i_j})\supset \bigcup_{i=1}^{N} B(x_i, \lambda^2R_i); \end{equation*}
  2. \begin{equation*} m(\bigcup_{j=1}^{p} B(x_{i_j}, T_{i_j}))\leq C_1Nv+C_2v(N-p)\text{.} \end{equation*}

Definition 10.2.3.

Let

\begin{equation*} \sigma=\sup \{r\mid m(B(x,r)\backslash \bigcup_{j=1} ^p B(x_{i_j}, T_{i_j})\leq v, \forall x\in X \}\text{.} \end{equation*}

Then \(\sigma\lt \infty\text{.}\) We call \(\sigma\) the \red{Lu's number} with respect to the balls

\begin{equation*} B(x_{i_1}, T_{i_1}),\cdots, B(x_{i_p}, T_{i_p})\text{.} \end{equation*}

Let \(x\in X\) be a point such that

\begin{equation*} m(B(x,2\sigma)\backslash \bigcup_{j=1} ^p B(x_{i_j}, T_{i_j})>v\text{.} \end{equation*}

Definition 10.2.4.

A subset, say, \(x_{i_s},\cdots, x_{i_p}\text{,}\) of \(x_{i_1},\cdots, x_{i_p}\) is called \red{selected}, if

\begin{equation*} B(x,17\lambda^2 \sigma)\cap B(x_{i_j}, \frac 12 T_{i_j})\neq\emptyset \end{equation*}

for \(j\geq s\text{.}\) In this case, we also call \(x_{i_j}\) selected if \(j\geq s\text{,}\) and \(B(x_{i_j}, T_{i_j})\) selected if \(x_{i_j}\) is selected.

Note that there must be at least one selected ball, otherwise, we can add \(B(x,3/2\sigma)\) to \(\mathfrak A\text{,}\) a contradiction.

If there are more than one selected points, say \(x_{i_{p-1}}, x_{i_p}\text{,}\) then we remove \(x_{i_p}\text{,}\) and replace \(B(x_{i_{p-1}}, T_{i_{p-1}})\) by

\begin{equation*} B(x_{i_{p-1}}, \max(300\lambda^3\sigma, T_{i_{p-1}}) )\text{.} \end{equation*}

We claim that the new set of balls is still admissible.

First, we know that

\begin{equation*} m(\bigcup_{j=1}^{p-2} B(x_i,T_{i_j} )\bigcup B(x_{i_{p-1}}, 300\lambda^3\sigma))\leq C_1Nv+C_2(n-p+1)v\text{.} \end{equation*}

We shall prove

\begin{equation*} \bigcup_{j=1} ^{p-2} B(x_{i_j}, \frac{1}{2\lambda}T_{i_j})\bigcup B(x_{i_{p-1}}, 150\lambda^2\sigma)\supset \bigcup_{i=1}^{N} B(x_i, \lambda^2 R_i)\text{.} \end{equation*}

To see this we first observe that

\begin{equation*} d(x,x_{i_j})\leq 17\lambda^2\sigma+\frac 12 T_{i_j} \end{equation*}

for \(j=p-1,p\text{.}\) On the other hand,

\begin{equation*} d(x,x_{i_j})>T_{i_j}-2\sigma \end{equation*}

for any \(j\text{.}\) Therefore, we have

\begin{equation*} T_{i_j}\leq (34\lambda^2+4)\sigma, d(x,x_{i_j})\leq (34\lambda^2+2)\sigma \end{equation*}

for \(j=p-1,p\text{.}\) Thus \(B(x_{i_p}, T_{i_p})\subset B(x_{i_{p-1}}, 150\lambda^2\sigma)\text{,}\) the reduced set is still admissible.

Using the procedure, we can get a minimal admissible set

\begin{equation*} B(x_1,T_1),\cdots, B(x_k, T_k)\text{.} \end{equation*}

Without loss of generality, we can assume that

\begin{equation} m(B(x_j, T_j))>C_3 v\text{.}\tag{10.2.3} \end{equation}

Let \(x\in X\) such that

\begin{equation*} m(B(x,2\sigma')\backslash \bigcup_{j=1} ^k B(x_{j}, T_{j}))>v\text{,} \end{equation*}

where \(\sigma'\) is the Lu's number for the minimal admissible set.

If \(B(x,2\lambda\sigma')\) doesn't touch any balls, then we can place one more ball \(B(x,2\sigma')\text{.}\) If \(B(x,2\lambda\sigma')\) touch one of the balls, it has to touch only one of them, say, \(B(x_1,T_1)\) otherwise the sequence is not minimal.

By the above same computation

\begin{equation*} T_1\leq 4(\lambda+1)\sigma', d(x,x_1)\leq (4\lambda+2)\sigma'\text{.} \end{equation*}

We consider the annulus

\begin{equation*} B(x_1,\frac 12T_1, d(x,x_1)+2\sigma')\text{.} \end{equation*}

We claim

\begin{equation*} B(x_1,\frac 12T_1, d(x,x_1)+2\sigma')\supset B(x,2\sigma')\text{.} \end{equation*}

To prove this, we first observe that \(T_1>8\sigma'\text{,}\) otherwise it contradicts to¬†(10.2.3). Let \(z\in B(x,2\sigma')\backslash \bigcup_{j=1} ^k B(x_{j}, T_{j})\text{.}\) Then for any \(y\in B(x,2\sigma')\text{,}\) we have

\begin{equation*} d(y,x_1)\geq d(z,x_1)-4\sigma'\geq T_1-4\sigma'\geq \frac 12 T_1\text{.} \end{equation*}

The inequality \(d(y,x_1)\leq d(x,x_1)+2\sigma'\) is obvious. The claim is proved.

It remains to prove that the annulus

\begin{equation*} B(x_1,\frac{1}{2\lambda}T_1, \lambda(d(x,x_1)+2\sigma')) \end{equation*}

doesn't touch any other elements in \(\mathfrak \lambda A\text{.}\)

To see this, we first observe that if \(y\in B(x_2, \frac 12T_2)\text{,}\) then

\begin{equation*} d(y, x_1)\geq d(x_2, x)-d(x, x_1)-\frac 12 T_2>17\lambda^2\sigma'-d(x,x_1)>\lambda(d(x,x_1)+2\sigma')\text{.} \end{equation*}
