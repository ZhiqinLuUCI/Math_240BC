Section 8.1 Semigroup and its generator

In this section, we define abstract semigroup and prove some of the basic properties.

Definition 8.1.1.

A one-parameter semigroup on a complex Banach space \(\mathcal B\) is a family \(T_t\) of bounded linear operators, where \(T_t: \mathcal B\to\mathcal B\) parameterized by real numbers \(t\geq 0\) and satisfies the following relations:

  1. \(T_0=1\text{;}\)

  2. If \(0\leq s,t\lt \infty\text{,}\) then

    \begin{equation*} T_sT_t=T_{s+t}\text{.} \end{equation*}

  3. The map

    \begin{equation*} (t,f)\to T_tf \end{equation*}

    from \([0,\infty)\times\mathcal B\) to \(\mathcal B\) is jointly continuous.

The (infinitesimal) generator \(Z\) of a one-parmeter semigroup \(T_t\) is defined by

\begin{equation*} Zf=\limt t^{-1}(T_tf-f)\text{.} \end{equation*}

The domain \({\mathcal Dom}(Z)\) of \(Z\) being the set of \(f\) for which the limit exists. It is evident that \({\mathcal Dom}(Z)\) is a linear space. Moreover, we have

Lemma 8.1.2.

The subspace \({\mathcal Dom}(Z)\) is dense in \(\mathcal B\text{,}\) and is invariant under \(T_t\) in the sense that

\begin{equation*} T_t( {\mathcal Dom}(Z) )\subset {\mathcal Dom}(Z) \end{equation*}

for all \(t\geq 0\text{.}\) Moreover

\begin{equation*} T_tZf=ZT_t f \end{equation*}

for all \(f\in {\mathcal Dom}(Z)\) and \(t\geq 0\text{.}\)

Proof.

If \(f\in\mathcal B\text{,}\) we define

\begin{equation*} f_t=\int_0^t T_xf\, dx\text{.} \end{equation*}

The above integration exists in the following sense: since \(T_xf\) is a continuous function of \(x\text{,}\) we can define the integration as the limit of the corresponding Riemann sums. In a Banach space, absolute convergence implies conditional convergence. Thus in order to prove the convergence of the Riemann sums, we only need to verify that

\begin{equation*} \int_0^t||T_xf||\, dx \end{equation*}

is convergent. But this follows easily from the joint continuity in the definition of the semigroup: \(||T_xf||\) must be uniformly bounded for small \(x\text{.}\)

We compute

\begin{align*} \amp \limh h^{-1} (T_hf_t-f_t)\\ \amp =\limh\left\{ h^{-1}\int_h^{t+h}T_xf dx-h^{-1}\int_0^t T_x f dx\right\}\\ \amp =\limh \left\{ h^{-1}\int_t^{t+h}T_x f dx-h^{-1}\int_0^h T_x f dx\right\}\\ \amp =T_t f-f\text{.} \end{align*}

Therefore, \(f_t\in{\mathcal Dom}(Z)\) and

\begin{equation*} Z(f_t)=T_t f-f\text{.} \end{equation*}

Since \(t^{-1} f_t\to f\) in norm as \(t\to 0^+\text{,}\) we see that \({\mathcal Dom}(Z)\) is dense in \(\mathcal B\text{.}\)

The generator \(Z\text{,}\) in general, is not a bounded operator. However, we can prove the following

Lemma 8.1.3.

The generator \(Z\) is a closed operator.

Proof.

We first observe that

\begin{equation*} T_tf-f=\int_0^t T_x Zf dx \end{equation*}

for \(f\in{\mathcal Dom}(Z)\text{.}\) To see this, we consider the function \(r(t)=T_tf-f-\int_0^t T_x Zf dx\text{.}\) Obviously we have \(r(0)=0\) and \(r'(t)\equiv 0\text{.}\) Thus \(r(t)\equiv 0\text{.}\)

Let \(\{f_n\}\) be a sequence in the domain of \(Z\) such that \(f_n\to f\) and \(Zf_n\to g\text{.}\) Using the above formula, we have

\begin{equation*} T_t f-f=\lim_{n\to \infty} (T_t f_n -f_n) =\lim_{n\to \infty}\int_0^t T_x Zf_n dx\text{.} \end{equation*}

By the Lebesgue dominated convergence theorem, the above limit is equal to

\begin{equation*} \int_0^t T_x g dx\text{.} \end{equation*}

Thus we have

\begin{equation*} \limt t^{-1}(T_t f-f)=g\text{,} \end{equation*}

and therefore, \(f\in{\mathcal Dom}(Z), Zf=g\text{.}\)

Remark 8.1.4.

If \(Z\) is not a bounded operator on \({\mathcal Dom}(Z)\text{,}\) it is not possible to extend \(Z\) to the whole banach space \(\mathcal B\) because otherwise since \(Z\) is closed, \(Z\) has to be bounded by the closed graph theorem.

Theorem 8.1.5.

Let \(\mathfrak B\) be a Hilbert space. If the operators \(T_t\) are self-adjoint operators, then \(Z\) is a densely defined self-adjoint operator.
